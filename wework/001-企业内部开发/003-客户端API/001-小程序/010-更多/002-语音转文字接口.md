最后更新：2019/05/16

>仅在企业微信手机端2.7.5及以后版本支持

**参数**
**Object object**

| 属性 | 类型 | 默认值 | 是否必填 | 说明 | 
| -------- | ------ | ---- | -------------|----- | 
| filePath | String | | 否 | 需要识别的语音文件的文件路径，由录音相关接口获得，仅支持由wx.startRecord录制的语音文件 | 
| isShowProgressTips| number| | 否 | 默认为1，显示进度提示 | 
| success | function | | 否 | 接口调用成功的回调函数 | 
| fail | function | | 否 | 接口调用失败的回调函数 | 
| complete | function | | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） | |

**object.success 回调函数**
**参数**
**Object res**

| 属性 | 类型 | 说明 | 
| ----------------- | ------ | -------------- |
| translateResult | String | 语音识别的结果 | |

**示例代码**
```
wx.qy.translateVoice({
 filePath: '', // 需要播放的语音文件的文件路径，由录音相关接口获得
 isShowProgressTips: 1, // 默认为1，显示进度提示
 success: (res) => {
 var result = res.translateResult
 },
 fail: (res) => {
 console.log(JSON.stringify(res))
 }
})
```
