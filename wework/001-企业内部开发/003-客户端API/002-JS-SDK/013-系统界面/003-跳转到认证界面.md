## 旧版jweixin调用
### 示例代码
```javascript
 wx.invoke('enterpriseVerify', 
 {    // 此api无需额外参数，空对象即可
 },
 function(res) {
    if(res.err_msg == "enterpriseVerify:ok") {
   // 正常
   } else {
   // 错误处理
   }
 });

```
> 备注：此接口要求企业微信版本为为2.8.7及以上
