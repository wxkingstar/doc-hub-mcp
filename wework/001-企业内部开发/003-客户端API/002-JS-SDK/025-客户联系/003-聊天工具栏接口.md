## 概述

企业可将应用页面配置到聊天工具栏，方便成员在与客户的聊天中查看和使用，提高服务效率。从聊天工具栏打开的页面，除了一般的JSAPI之外，可以使用有会话属性的JSAPI：
- [获取当前客户ID](#17407)
- [获取当前客户群的群ID](#19330)
- [分享消息到当前会话](#29574)

## 支持的入口环境
开发者首先需要调用[ww.getContext](#21950)接口获取当前入口`entry`值，在聊天工具栏里返回的`entry`值是`single_chat_tools`或`group_chat_tools`，具体如下：
| 值 | 说明 | 支持的最低版本 |
| -------- | ---- | ----- |
| **single_chat_tools** | **从单聊会话的工具栏进入** |3.0.24 |
| **group_chat_tools** | **从群聊会话的工具栏进入** | 3.0.24 |
