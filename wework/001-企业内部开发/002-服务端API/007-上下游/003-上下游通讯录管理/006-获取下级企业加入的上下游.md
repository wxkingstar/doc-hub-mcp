上级企业自建应用/代开发应用通过本接口查询下级企业所在上下游

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/corpgroup/get_corp_shared_chain_list?access_token=ACCESS_TOKEN

**请求示例**

```json
{
 "corpid":"xxxxx"
}
```
**参数说明 ：**

| 参数 | 必须 | 说明 |
| ------------ | ---- | -------------------------------------- |
| access_token | 是 | 调用接口凭证。上游企业应用access_token |
| corpid | 否 | 已加入企业id |

**权限说明：**
* 自建应用、代开发应用和「[上下游- 可调用接口的应用](#42599/配置可调用接口的应用)」可调用，仅可指定应用可见范围内的企业

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "chains": [
 {
 "chain_id": "xxx",
 "chain_name": "xxx"
 }
 ]
}

```
**参数说明：**

| 参数 | 说明 |
| --------------- | -------------------------------------------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| chains | 上下游列表 |
| chains.chain_id | 上下游id |
| chains.chain_name | 上下游名称 |
