上级企业自建应用/代开发应用通过本接口查询成员自定义 id

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/corpgroup/corp/get_chain_user_custom_id?access_token=ACCESS_TOKEN

**请求示例**

```json
{
 "chain_id":"Chxxxxxx",
 "corpid":"xxxxx",
 "userid":"xxxxx"
}
```
**参数说明 ：**

| 参数 | 必须 | 说明 |
| ------------ | ---- | -------------------------------------- |
| access_token | 是 | 调用接口凭证。上游企业应用access_token |
| chain_id | 是 | 上下游id |
| corpid | 是 | 已加入企业id |
| userid | 是 | 企业内的成员 |

**权限说明：**
* 自建应用、代开发应用和「[上下游- 可调用接口的应用](#42599/配置可调用接口的应用)」可调用，仅可指定应用可见范围内的企业

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "user_custom_id":"1234"
}

```
**参数说明：**

| 参数 | 说明 |
| --------------- | -------------------------------------------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| user_custom_id | 成员自定义 id |
