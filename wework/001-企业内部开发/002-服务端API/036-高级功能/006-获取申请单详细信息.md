自建应用可使用此接口，获取企业成员高级功能申请单的详细信息。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/advanced_feature/get_approval_info?access_token=ACCESS_TOKEN 

**请求示例：**

``` json
{
 "apply_id": "APPLY_ID"
}
```

**参数说明：**
|参数 |类型 |是否必须 |说明 |
|:-|
|apply_id |string |是 |申请id |

**权限说明**:
调用的应用需要满足如下的权限：

|应用类型 |权限要求 |
|:-|
|自建应用 |配置到「!!#ff0000 高级功能-成员申请管理-通过api设置申请流程!!」中 |
|代开发应用 |暂不支持|
|第三方应用 |暂不支持|

**返回示例：**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "approval_info":{
 "applicant":"USERID",
 "create_time": 1715681559,
 "business_type":1,
 "approval_id":"APPROVAL_ID",
 "apply_id":"ApplyId",
 "approval_url":"APPROVAL_URL",
 "approval_status":0,
 "approval_type":1,
 "request_reason":"我有很多客户资料要管理，需要更多微盘空间"
 }
}
```
**参数说明：**

|参数 |说明 |
|:-|
|errcode |错误码 |
|errmsg |错误码说明 |
|approval_info.applicant |申请人userid |
|approval_info.create_time |创建时间戳 |
|approval_info.business_type |申请的高级账号类型<br>1-邮件 2-文档 3-微盘 4-会议 |
|approval_info.apply_id |请求id |
|approval_info.approval_id |自建应用中的审批id |
|approval_info.approval_url |自建应用中的审批单跳转链接 |
|approval_info.approval_status |审批单状态：1-审批人审批中 2-审批人驳回 3-审批人已同意 4-管理员驳回 5-管理员已分配 0-历史的申请记录，无流程信息 | 
|approval_info.approval_type |0-默认申请 1-自定义审批 2-企业openapi | 
|approval_info.request_reason |申请原因 |
