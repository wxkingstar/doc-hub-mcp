该接口可以修改通过[智能表格自动化创建的群聊](https://open.work.weixin.qq.com/help2/pc/19684?person_id=1&searchData=#4%E3%80%81%E8%87%AA%E5%8A%A8%E5%8C%96%E6%B5%81%E7%A8%8B%E5%88%9B%E5%BB%BA%E7%BE%A4%E8%81%8A)的群成员。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/groupchat/update?access_token=ACCESS_TOKEN

**请求包体**

```json
{
  "docid":"DOCID",
  "chat_id":"CHATID",
  "owner":"userid",
  "add_user_list" : ["userid1", "userid2", "userid3"],
  "del_user_list" : ["userid3", "userid4"]
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- |------------------------------------------------------------------------------------------------------------------|
| access_token | 是 | 调用接口凭证|
|docid|是|智能表格ID。需为当前应用创建的智能表格|
|chat_id|是|群聊ID，需为对应智能表自动规则创建群聊的ID|
|owner|否|新群主的id。若不需更新，请忽略此参数。del_user_list包含群主时本字段必填|
| add_user_list | 否 | 添加成员的id列表，一次最多传入500人|
| del_user_list | 否 | 踢出成员的id列表，一次最多传入500人|

**权限说明**

- 仅自建应用可调用，需配置到文档“[可调用应用](#43883)”列表中的应用
- 可见范围需包含根部门

**限制说明：**
- 群主需为本企业成员
- 操作的成员需为对应的智能表格中的成员
- 群成员人数不可超过2000人。
- 每企业变更群的次数不可超过1000次/小时。
- 同一群聊的修改需要串行执行

**并发限制：10** 

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok"
}
```
**参数说明：**

| 参数 | 说明 |
| ------- | ---------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
