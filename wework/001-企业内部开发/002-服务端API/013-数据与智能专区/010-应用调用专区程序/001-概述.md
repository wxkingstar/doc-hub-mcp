开发者若要触发专区程序执行指定任务时，可通过[应用同步调用专区程序](#53295)或者[应用异步调用专区程序](#53296)发起程序调用。专区程序需要监听8080端口接收企微后台调用专区的请求。具体参考[企微后台调用专区](#54006/企微后台调用专区)

>该场景call_type为1。

应用调用专区程序前，必须为程序添加能力，并且定义能力对应的输入协议以及输出协议，协议格式为json。配置页面如下图所示：
- ![enter image description here](https://wework.qpic.cn/wwpic3az/787910_6pF5FPVuTLiQt3O_1717486350/0)

**输入协议说明：**
协议格式为json，应用调用专区程序接口时，request_data字段必须匹配当前能力设置的输入协议。否则报错**790016**。

**输出协议说明：**
协议格式为json，应用调用专区程序接口时，专区程序返回的结果会填到response_data字段，程序返回的结果必须匹配当前能力设置的输出协议。否则报错**790014**。

>json协议匹配是指必须时配置的json示例的子集，即输入或者输出的json的key必须出现在当前能力配置的json示例中，且key对应的value类型必须匹配。
