> 第三方应用暂不支持商户号管理

##查询商户号详情
通过该接口可以查询**已绑定**商户号信息以及商户号使用范围

**请求方式：**POST(**HTTPS**)
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/externalpay/getmerchant?access_token=ACCESS_TOKEN

**请求示例：**
```
{
 "mch_id":"12334"
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| :----------- | :--- | :----------------- |
| mch_id   | 是 | 微信支付商户号,不超过32字节 |

**权限说明：**
调用的应用需要满足如下的权限

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「[对外收款- 可调用接口的应用](#24952/配置可调用接口的应用)」中 |
|代开发应用|暂不支持|
|第三方应用|暂不支持|
 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)

**返回结果：**
```
{
 "errcode":0,
 "errmsg":"ok",
 "bind_status":0,
 "mch_id":"12334",
 "merchant_name":"test",
 "allow_use_scope": {
 "user": ["zhangsan","lisi"],
 "partyid": [1],
   "tagid": [1,2,3]
 }
}
```

**参数说明：**

| 参数 | 说明 |
| :----------- | :----------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| mch_id   | 微信支付商户号,不超过32字节 |
| merchant_name | 微信支付商户号全称，不超过50字节|
| bind_status | 商户号绑定状态。 1:申请中 2:已绑定 3:已撤销|
| allow_use_scope | 该商户号使用范围，仅当商户号已绑定时才返回|
| allow_use_scope.user | 使用范围内的企业用户id列表|
| allow_use_scope.partyid | 使用范围内的部门id列表|
| allow_use_scope.tagid | 使用范围内的标签id列表|

##设置商户号使用范围
通过该接口可以设置**已绑定**商户号的使用范围

**请求方式：**POST(**HTTPS**)
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/externalpay/set_mch_use_scope?access_token=ACCESS_TOKEN

**请求示例：**
```
{
 "mch_id":"12334",
 "allow_use_scope": {
 "user": ["zhangsan","lisi"],
 "partyid": [1],
   "tagid": [1,2,3]
 }
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| :----------- | :--- | :----------------- |
| mch_id   | 是 | 微信支付商户号,不超过32字节
| allow_use_scope | 是 | 该商户号使用范围，支持按成员、部门或标签设置。|
| allow_use_scope.user | 否 | 使用范围内的企业用户id列表|
| allow_use_scope.partyid | 否 | 使用范围内的部门id列表|
| allow_use_scope.tagid | 否 | 使用范围内的标签id列表|

**权限说明：**
| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「[对外收款- 可调用接口的应用](#24952/配置可调用接口的应用)」中 |
|代开发应用|暂不支持|
|第三方应用|暂不支持|
 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)

**返回结果：**
```
{
 "errcode":0,
 "errmsg":"ok"
}
```

**参数说明：**

| 参数 | 说明 |
| :----------- | :----------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
