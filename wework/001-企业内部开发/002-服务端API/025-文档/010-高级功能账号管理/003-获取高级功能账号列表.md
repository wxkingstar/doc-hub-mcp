该接口用于查询企业已分配高级功能且在应用可见范围的账号列表。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/wedoc/vip/list?access_token=ACCESS_TOKEN

**请求包体**

```json
{
  "cursor":"CURSOR",
  "limit":2
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- |------------------------------------------------------------------------------------------------------------------|
| access_token | 是 | 调用接口凭证|
|cursor|否|用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填|
|limit|否|用于分页查询，每次请求返回的数据上限。默认100，最大200<br>!!#ff0000 注意：不保证每次返回的数据刚好为指定limit!!，必须用返回的`has_more`判断是否继续请求|

**权限说明：**
| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「!!#ff0000 协作 - 文档 - 可调用接口的应用!!」中 |
|代开发应用|暂不支持| 
|第三方应用|暂不支持|

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "has_more":true,
 "next_cursor":"GNIJIGEO",
 "userid_list":["zhangsan","lisi"]
}
```
**参数说明：**

| 参数 | 说明 |
| ------- | ---------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|has_more|是否还有更多数据未获取|
|next_cursor|下一次请求的cursor值|
|userid_list|符合条件的企业成员userid列表|
