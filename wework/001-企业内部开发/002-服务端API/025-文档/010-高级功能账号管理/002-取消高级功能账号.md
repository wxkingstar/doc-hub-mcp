该接口用于撤销分配应用可见范围企业成员的高级功能。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/wedoc/vip/batch_del?access_token=ACCESS_TOKEN

**请求包体**

```json
{
 "userid_list": ["zhangsan","lisi","wangwu"]
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- |------------------------------------------------------------------------------------------------------------------|
| access_token | 是 | 调用接口凭证|
|userid_list|是|要撤销分配高级功能的企业成员userid列表，单次操作最多限制100个|

**权限说明：**
| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「!!#ff0000 协作 - 文档 - 可调用接口的应用!!」中 |
|代开发应用|暂不支持| 
|第三方应用|暂不支持|

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "succ_userid_list":["zhangsan","lisi"],
 "fail_userid_list":["wangwu"]
}
```
**参数说明：**

| 参数 | 说明 |
| ------- | ---------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|succ_userid_list|撤销分配成功的userid列表|
|fail_userid_list|撤销分配失败的userid列表|
