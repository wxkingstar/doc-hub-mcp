该接口用于上传图片

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/wedoc/image_upload?access_token=ACCESS_TOKEN 

**请求包体**

```json
{
  "docid": "docid",
  "base64_content":"Content"
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|------------|--------|------|----------------------------|
| docid | string | 是 | 文档ID，通过[新建文档接口](#43939)创建后获得 |
| base64_content | string | 是 | base64之后的图片内容 |

**权限说明**
- 自建应用需配置到“[可调用应用](#43883)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限

**返回示例**

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "url": "URL",
  "height": 234,
  "width": 234,
  "size": 81259
}
```
**参数说明**

| 参数 |类型| 说明 |
|---------|-------|-------|
| errcode |int32| 错误码 |
| errmsg |string| 错误码说明 |
| url |string| 图片的url |
| height |string| 图片的高 |
| width |string| 图片的宽 |
| size |string| 图片的大小 |
