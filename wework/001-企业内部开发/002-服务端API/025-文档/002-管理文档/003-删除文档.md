该接口用于删除指定文档、表格、智能表格及收集表进行重命名。

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/wedoc/del_doc?access_token=ACCESS_TOKEN 

**请求包体**

```json
{
  "docid": "DOCID",
  "formid": "FORMID"
}
```

**参数说明**

| 参数 | 类型| 是否必须 | 说明 |
|------------| -----|------|----------------------------|
| docid | string | 否 | 文档docid（docid、formid只能填其中一个），仅可删除应用自己创建的文档 |
| formid | string | 否 | 收集表id（docid、formid只能填其中一个），仅可删除应用自己创建的收集表 |

**权限说明**
- 自建应用需配置到“[可调用应用](#43883)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限

**返回示例**

```json
{
  "errcode": 0,
  "errmsg": "ok"
}
```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
