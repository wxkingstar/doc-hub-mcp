添加客服账号，并可设置客服名称和头像。目前一家企业最多可添加**5000个**客服账号。

**请求方式**: POST(**HTTPS**)

**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/kf/account/add?access_token=ACCESS_TOKEN

**请求示例**

```json
{
 "name": "新建的客服账号",
 "media_id": "294DpAog3YA5b9rTK4PjjfRfYLO0L5qpDHAJIzhhQ2jAEWjb9i661Q4lk8oFnPtmj"
}
```

**参数说明**:

|参数|必须|类型|说明|
|:-----------|:---|:-------|:------------|
| access_token | 是 | string | 调用接口凭证 |
|name|是|string|客服名称</br>不多于16个字符|
|media_id|是|string|客服头像临时素材。可以调用[上传临时素材](#10112)接口获取。</br>不多于128个字节|

**权限说明**:
调用的应用需要满足如下的权限
| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「 [微信客服- 可调用接口的应用](#31106/如何开启API)」中，且在管理后台「通过API管理会话消息」-「企业内部开发」对应的自建应用的「可管理的客服账号」处，配置至少一个客服账号 |
|第三方应用|具有!!#ff0000 “微信客服->管理账号、分配会话和收发消息”!!权限|
|代开发自建应用|具有!!#ff0000 “微信客服->管理账号、分配会话和收发消息”!!权限|
 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)
 
!!#ff0000 通过接口创建的客服账号，将自动拥有该客服账号的管理权限。!!企业可在管理后台“微信客服-通过API管理微信客服账号”处设置对应的客服账号通过API来管理。
<!--自建应用需要曾设置过通过 API 管理某个微信客服账号才可调用此接口-->

**返回结果**:

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "open_kfid": "wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyJTw"
}
```

**参数说明**:

|参数|类型|说明|
|:-----------|:-------------|:----------|
|errcode|int32|返回码|
|errmsg|string|错误码描述|
|open_kfid|string|新创建的客服账号ID|
