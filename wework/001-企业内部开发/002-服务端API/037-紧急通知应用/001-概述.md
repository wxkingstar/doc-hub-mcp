医疗行业的企业，可通过语音电话提醒员工查看应用推送的重要消息，以便尽快处理紧急事件。

## 开始开发
1. 管理员进入企业微信“管理后台” — “应用管理”—“紧急通知”，启用应用。（注：目前仅医疗行业的企业可开启该应用）
- ![开启紧急通知应用](https://wework.qpic.cn/wwpic/845526_hvZS0V6ET0COA3x_1608104826/0)
  
2. 充值购买电话分钟数。
- ![充值购买分钟数](https://wework.qpic.cn/wwpic/260024_gGELpoRRTNapuXa_1608104875/0)

3. 设置可调用紧急通知的自建应用。
- ![关联应用](https://wework.qpic.cn/wwpic/864885_nxiaXUsjRym6fua_1608104927/0)

4. 进入自建应用详情，查看并妥善保存调用紧急通知接口所需的重要参数（secret），然后通过此应用的secret调用[获取access_token](#15074)得到接口调用凭证。
- ![enter image description here](https://wework.qpic.cn/wwpic3az/640875_BdWkOh6SRcSmADl_1701262205/0)

5. 调用**[发起语音电话](#16654)**接口，提醒员工查看应用推送的重要消息。

6. 语音电话模板：“XX（员工姓名），您好。XX（应用名称）给您发来一条重要消息，请打开XX（企业简称）的企业微信查看。”

7. 调用**[获取接听状态](#16655)**接口，了解员工是否已接听语音电话。
