**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/school/get_payment_result?access_token=ACCESS_TOKEN

**请求包体：**

```json
{
 "payment_id":"xxxx"
}

```

**参数说明：**

| 参数| 必须 | 说明|
| ---- | ---- | ---- |
| access_token | 是 | 调用接口凭证|
| payment_id | 是 | 收款项目id，由jssdk的[发起班级收款接口](#30233) 或者小程序的[发起班级收款接口](#30232)返回 |

**权限说明：**

- 只允许获取由应用本身创建的收款项目详情

**返回结果：**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "project_name": "学费",
  "amount": 998,
  "payment_result": [{
    "student_userid": "xxxx",
    "trade_state": 1,
    "trade_no": "xxxxx",
    "payer_parent_userid": "zhangshan"
  }]
}
```

**参数说明：**

| 参数| 说明|
| ---- | ----|
| errcode| 返回码 |
| errmsg| 对返回码的文本描述内容|
|payment_result|学生付款信息列表，详情见PayInfo说明|

**学生付款详情PayInfo说明**

| 参数 | 说明 |
|----|----|
|student_userid|学生账号|
|trade_state|付款状态。0表示未付款；1表示已付款|
|trade_no|订单号|
|payer_parent_userid|付款家长账号|
