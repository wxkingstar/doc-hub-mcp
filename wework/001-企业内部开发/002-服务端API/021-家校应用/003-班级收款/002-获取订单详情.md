**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/school/get_trade?access_token=ACCESS_TOKEN

**请求包体：**

```json
{
 "payment_id":"xxxx",
 "trade_no":"xxxx" 
}

```

**参数说明：**

| 参数| 必须 | 说明|
| ---- | ---- | ---- |
| access_token | 是 | 调用接口凭证|
| payment_id | 是 | 收款项目id，由[发起班级收款接口](#30233)返回 |
|trade_no|是|订单号，由[获取学生付款结果](#30127)返回|

**权限说明：**

- 只允许获取由应用本身创建的收款项目的订单详情

**返回结果：**

```
{
  "errcode":0,
  "errmsg":"ok",
  "transaction_id":"xxxxx",    
  "pay_time":12345
}
```

**参数说明：**

| 参数| 说明|
| ---- | ----|
| errcode| 返回码 |
| errmsg| 对返回码的文本描述内容|
|transaction_id|微信交易单号|
|pay_time|交易时间|
