该接口用于删除已有的公共邮箱。

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/delete?access_token=ACCESS_TOKEN

**请求包体**
```json
{
 "id": 1
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|--------------|---------|------|-------------------------------------|
| id | uint32 | 是 | 公共邮箱ID |

**权限说明**

- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 应用调用接口只能访问自身创建的公共邮箱

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok"
}
```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
