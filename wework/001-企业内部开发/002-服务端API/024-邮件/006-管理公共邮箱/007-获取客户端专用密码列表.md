该接口用于获取公共邮箱创建的客户端专用密码列表，包括客户端专用密码ID、创建时间、最后使用时间、备注。需要注意的是，该接口不返回客户端专用密码，客户端专用密码仅会在创建时候返回，参考[创建公共邮箱](#37504)、[更新公共邮箱](#44710)

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/get_auth_code_list?access_token=ACCESS_TOKEN

**请求包体**
```json
{
 "id" : 1
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|-----------------|--------|------|-------------------------------------------------------------------------------|
| id | uint32 | 是 | 公共邮箱ID |

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "auth_code_list":[
 {
 "auth_code_id": 1,
 "create_time": 1724743240,
 "last_use_time": 1724743241,
 "remark": "客户端专用密码备注"
 },
 {
 "auth_code_id": 2,
 "create_time": 1724743240,
 "last_use_time": 1724743241,
 "remark": "客户端专用密码备注"
 }
 ]
}
```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| auth_code_list.auth_code_id| uint32| 客户端专用密码ID|
| auth_code_list.create_time| uint32| 客户端专用密码创建时间戳 |
|auth_code_list.last_use_time| uint32 | 客户端专用密码最后使用时间，如果未使用过则返回0 |
|auth_code_list.remark| string | 客户端专用密码备注 |
