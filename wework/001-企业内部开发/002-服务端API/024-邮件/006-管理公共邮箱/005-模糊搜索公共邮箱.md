该接口用于模糊搜索公共邮箱，可匹配邮箱名也可匹配邮箱地址。

**请求方式**：GET（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/search?access_token=ACCESS_TOKEN

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|--------------|---------|------|-------------------------------------|
| fuzzy | uint32 | 是 | 1开启模糊搜索，0获取全部公共邮箱 |
| email | string | 否 | 公共邮箱名称或邮箱地址 |

**权限说明**

- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 应用调用接口只能访问自身创建的公共邮箱

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "list": [
 {
 "id": 11,
 "email": "apitest3@gzdev.com",
 "name": "apitest3"
 },
 {
 "id": 12,
 "email": "apitest4@gzdev.com",
 "name": "apitest4"
 }
 ]
}
```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| id | uint32 | 公共邮箱ID |
| email | string | 公共邮箱地址 |
| name | string | 公共邮箱名称 |
