获取指定成员邮箱当前未读邮件数量
**权限说明**

- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/exmail/mail/get_newcount?access_token=ACCESS_TOKEN

**请求包体**
```json
{
 "userid": "member"
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ------------ | ------------ |
| access_token | 是 | 调用接口凭证 |
| userid | 是 | 成员[UserID](https://developer.work.weixin.qq.com/document/path/90196) |

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "count": 1
}
```
**参数说明：**

| 参数 | 说明 |
| ------------ | ------------ |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| count | 成员邮箱中邮件未读数 |
