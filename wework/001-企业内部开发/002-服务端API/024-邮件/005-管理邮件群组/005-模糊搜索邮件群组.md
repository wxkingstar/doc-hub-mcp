该接口用于通过群组ID模糊搜索邮件群组。

**请求方式**：GET（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/exmail/group/search?access_token=ACCESS_TOKEN

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|--------------|---------|------|-------------------------------------|
| fuzzy | uint32 | 是 | 1开启模糊搜索，0获取全部邮件群组 |
| groupid | string | 否 | 邮件群组ID，邮箱格式 |

**权限说明**

- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 应用调用接口只能访问自身创建的邮件群组

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "count": 2,
 "groups": [
 {
 "groupid": "g_all@gzdev.com",
 "groupname": "任何人"
 },
 {
 "groupid": "g_inner@gzdev.com",
 "groupname": "组内成员"
 }
 ]
}

```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| count | string | 返回条数 |
| groupid | string | 邮件群组ID |
| groupname | string | 邮件群组名称 |
