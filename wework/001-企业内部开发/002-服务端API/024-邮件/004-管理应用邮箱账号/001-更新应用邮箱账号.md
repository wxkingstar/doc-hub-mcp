应用调用此接口，可以更新应用邮箱账号，原有的应用邮箱账号将会作为别名邮箱，具有收信能力

**请求方式：** POST（**HTTPS**）

**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/exmail/app/update_email_alias?access_token=ACCESS_TOKEN

 
**请求包体**
```json
{
 "new_email": "youxiangceshi@ceshi.wecom.work"
}
```
 

**参数说明**
| 参数 | 必须 | 类型 | 说明 |
| ------------ | ------------ | ------------ | ------------ |
| access_token | 是 | string | 调用接口凭证 |
| new_email | 是 | string |修改后的应用邮箱账号 |

**权限说明**
- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）

**返回示例**
 
```json
{
 "errcode":0,
 "errmsg":"ok"
}
```

**参数说明**

| 参数 | 类型| 说明 |
| ------------ | ------------ |------------ |
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
