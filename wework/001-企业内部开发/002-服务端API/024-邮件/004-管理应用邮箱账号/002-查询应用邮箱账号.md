应用调用此接口，可以查询自己的应用邮箱账号及别名邮箱

**请求方式：** POST（**HTTPS**）

**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/exmail/app/get_email_alias?access_token=ACCESS_TOKEN

**请求包体**

**不需要请求包体**

**参数说明**
| 参数 | 必须 | 类型 | 说明 |
| ------------ | ------------ | ------------ | ------------ |
| access_token | 是 | string | 调用接口凭证 |

**权限说明**
- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "email": "youxiangceshi@ceshi.wecom.work",
 "alias_list": [
 "youxiang1@ceshi.wecom.work",
 "youxiang2@ceshi.wecom.work"
 ]
}
```

**参数说明**
| 参数 | 类型| 说明 |
| ------------ | ------------ |------------ |
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| email | string | 当前发信账号的主邮箱地址，发邮件将会以此邮箱地址为发件人 |
| alias_list | string[] | 别名邮箱地址列表，能作为收件人收邮件 |
