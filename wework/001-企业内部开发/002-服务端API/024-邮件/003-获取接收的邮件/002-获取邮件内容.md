支持应用获取邮件内容。指定单个邮件id，获取邮件eml数据。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/exmail/app/read_mail?access_token=ACCESS_TOKEN

**请求包体**
```json
{
 "mail_id": "MAIL_ID"
}
```

**参数说明**

| 参数 | 必须 | 类型 | 说明 |
| ------------ | ------------ | ------------ | ------------ |
| access_token | 是 | string | 调用接口凭证 |
| mail_id | 是 | string |邮件id|

**权限说明**
- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有“邮件”权限
- 代开发自建应用需具有“邮件”权限

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "mail_data": "MAIL_DATA"
}
```
**参数说明**

| 参数 | 类型| 说明 |
| ------------ | ------------ |------------ |
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| mail_data | string | 邮件eml内容 |
