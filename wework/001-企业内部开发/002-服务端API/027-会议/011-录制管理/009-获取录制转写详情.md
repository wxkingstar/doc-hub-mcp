获取云录制转写的详情，包含时间戳、文本等内容。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record/transcript/get_detail?access_token=ACCESS_TOKEN

**请求包体：**
```
{
  "record_file_id": "FILEID",
  "meetingid": "MEETINGID",
  "pid": "1111",
  "limit": 10
}
```

**参数说明：**
| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|record_file_id |是 |string|录制文件 ID。 |
|meetingid |是 |string|会议ID |
|pid |否|string|查询的起始段落 ID。获取 pid 后（含）的段落，默认从0开始。示例："1111" |
|limit |否|int32|查询的段落数，默认查询全量数据。 |

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据
- 仅返回在应用可见范围内的成员

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "has_more": true,
  "transcripts": {
    "paragraphs": [{
      "pid": "11",
      "start_time": 11111,
      "end_time": 222222,
      "speaker_info": {
        "userid": "USERID"
      },
      "sentences": [{
        "sid": "1111",
        "start_time": 111,
        "end_time": 222,
        "words": [{
          "wid": "111",
          "start_time": 11,
          "end_time": 22,
          "text": "word"
        }]
      }]
    }],
    "keywords": ["nice", "hard"],
    "audio_detect": 1
  }
}
```

**参数说明：**

|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
|has_more|bool|如果传入 pid 或 limit 字段，且没有一次完全返回，则为 true；否则为 false。 |
|transcripts|object|录制转写详情，详见Detail|

**Detail说明**
|参数 |类型|说明 |
|:--|
|paragraphs|object[]|段落对象列表，详见Paragraph |
|keywords |string[]|关键词列表。 |
|audio_detect |int32|声纹识别状态：<br>0：未完成<br>1：已完成<br>**说明**：声纹识别是针对一台设备多人讲话场景，自动区分为多个发言人的能力。声纹识别与录制转写生成的过程独立。无需声纹识别或声纹识别已完成时，该值为1。|

**Paragraph说明**
|参数 |类型|说明 |
|:--|
|pid |string|段落 ID。 |
|start_time|int64|录制文件中的段落开始时间（毫秒）。 |
|end_time|int64|录制文件中的段落结束时间（毫秒）。 |
|speaker_info|object|发言人信息，详见Speaker |
|sentences |object[]|句子信息，详见Sentence |

**Speaker说明**
|参数 |类型|说明 |
|:--|
|userid |string|同企业返回企业成员 userid，非同企业不返回该字段。 |

**Sentence说明**
|参数 |类型|说明 |
|:--|
|sid |string|句子 ID。 |
|start_time |int64|录制文件中的句子开始时间（毫秒）。 |
|end_time |int64|录制文件中的句子结束时间（毫秒）。 |
|words |object[]|词信息列表，详见Word |

**Word说明**
|参数 |类型|说明 |
|:--|
|wid |string|词 ID。 |
|start_time |int64|录制文件中的词开始时间（毫秒）。 |
|end_time |int64|录制文件中的词结束时间（毫秒）。 |
|text |string|文本内容。 |
