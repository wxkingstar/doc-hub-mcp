获取会议录制 ID 对应的访问数据，按照天维度返回。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record/get_statistics?access_token=ACCESS_TOKEN
**请求包体：**

```
{
  "meeting_record_id": "record",
  "meetingid":"MEETINGID",
  "start_time": 16666666,
  "end_time": 1788888888
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meeting_record_id |是 |string|会议录制ID |
|start_time |否 |int32|查询起始时间戳，UNIX 时间戳（单位秒）。说明：默认展示最近31天的数据，时间区间不允许超过31天。 |
|end_time |否 |int32|查询结束时间戳，UNIX 时间戳（单位秒）。说明：默认展示最近31天的数据，时间区间不允许超过31天。 |

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据

**返回结果：**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "summaries": [{
    "date": "2033-01-01",
    "view_count": 12,
    "download_count": 12
  }]
}
```

**参数说明：**

|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
|summaries|object|统计结果列表，详见Summary|

**Summary说明**
|参数 |类型|说明 |
|:--|
|date |string|统计时间，格式：yyyy-MM-dd。 |
|view_count |int32|观看次数（当天数据），默认0。 |
|download_count |int32|下载次数（当天数据），默认0。 |
