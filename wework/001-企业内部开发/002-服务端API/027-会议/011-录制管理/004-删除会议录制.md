删除会议的所有录制文件，该接口会删除会议录制 ID 里对应的所有云录制文件。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record/delete?access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "meeting_record_id": "record",
  "meetingid":"MEETINGID"
}
```

**参数说明：**
| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meeting_record_id |是 |string|会议录制ID |
|meetingid|是|string|会议ID|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许删除该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok"
}
```
 
**参数说明：**
|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
