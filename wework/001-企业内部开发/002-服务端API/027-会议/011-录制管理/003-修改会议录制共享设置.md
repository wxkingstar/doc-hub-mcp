根据会议录制 ID 修改共享等配置，支持修改共享权限、共享密码、共享有效期等信息

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record/update_sharing_config?access_token=ACCESS_TOKEN

**请求包体：**
```
{
  "meeting_record_id": "record",
  "meetingid": "MEETINGID",
  "sharing_config": {
    "enable_sharing": true,
    "sharing_auth_type": 0,
    "enable_password": true,
    "password": "123352",
    "enable_sharing_expire": true,
    "sharing_expire": 1999999999,
    "allow_download": true
  }
}
```

**参数说明：**
| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meeting_record_id |是 |string|会议录制ID |
|meetingid |是 |string|会议ID |
|sharing_config|否|object|共享配置，详见SharingConfig|

**SharingConfig说明**
|参数 |必须 |类型|说明 |
|:--|
|enable_sharing |否 |bool|共享开关，是否开启共享，默认为 true。<br>true：开启<br>false：未开启<br>说明：<br>- 未开启时不允许设置以下参数。<br>- 修改为 false 关闭共享后，之前设置的共享设置将不保存。 |
|sharing_auth_type |否 |int32|共享权限类型。<br>0：仅允许登录成员查看<br>1：仅企业内成员可查看<br>2：仅参会成员可查看<br>3：全部成员可查看<br>4：通过权限审批的成员可查看（Rooms 不支持）<br>5：获取微信特邀链接的成员可查看（Rooms 不支持） |
|enable_password |否 |bool|是否开启密码，默认为 true。<br>true：开启 <br>false：不开启|
|password |否 |string|共享密码，默认随机生成。<br>**说明：**当 enable_password = true 时，必传；当 enable_password = false 时，不可传。 |
|enable_sharing_expire |否 |bool|是否开启共享链接有效期，默认为 false。<br>true：开启<br>false：不开启 |
|sharing_expire |否 |int64|共享链接有效期，unix 时间戳（单位毫秒），默认为空。<br>**说明：**当 enable_sharing_expire = true 时，必传；当 enable_sharing_expire = false 时，不可传。 |
|allow_download |否 |bool|是否允许下载，默认为 false。<br>true：允许下载<br>false：不允许下载 |

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许修改该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：**
|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
