修改网络研讨会的报名配置和报名问题，需要会议已开启报名。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/webinar/enroll/set_config?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "MEETINGID1",
  "approve_type": 1,
  "is_collect_question": 1,
  "no_registration_needed_for_staff": true,
  "question_list": [{
 "is_required":1,
 "question_type":1,
 "question_title":"acd",
 "special_type":1,
 "option_list":[{
 "content":"asr"
 }]
 }]
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meetingid |是 |string |网络研讨会 ID |
|approve_type |否 |int32 |审批类型：<br>1：自动审批，默认自动审批<br>2：手动审批|
|is_collect_question|否|int32 |是否收集问题：<br>1：不收集，默认值为不收集<br>2：收集|
|no_registration_needed_for_staff |否|bool |本企业成员无需报名：<br>true：本企业成员无需报名<br>false：默认配置，本企业成员及企业外成员需要报名 |
|question_list |否|object[] |报名问题列表，详见Question。 |

**Question说明**
|参数 |必须 |类型 |说明 |
|:--|
|is_required|是|uint32|是否必填：<br>1：否<br>2：是|
|question_title|否|string|问题标题，限制40个字符。<br>**说明：**special_type 为特殊问题时，该字段无效。|
|option_list|否|object[]|问题选项列表，按传入的顺序排序，仅单选和多选时有效，最多8个选项，每个选项限40个汉字。详见Option。|
|question_type|否|uint32|问题类型：<br>1：单选<br>2：多选<br>3：简答<br>**说明：**special_type 为特殊问题时，该字段无效。|
|special_type|否|uint32|特殊问题类型：<br>1：无<br>2：手机号<br>3：邮箱<br>4：姓名<br>5：公司名称<br>6：组织规模<br>**说明：**目前特殊问题均为简答题。|

**Option说明**
|参数 |必须 |类型 |说明 |
|:--|
|content|否|string|问题选项。每个选项限40个汉字。|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许修改该应用创建的网络研讨会的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "question_count": 1
}
```

**参数说明**

|参数 |类型 |说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
|question_count |int32 |报名问题数量，不收集问题时该字段返回0 |
