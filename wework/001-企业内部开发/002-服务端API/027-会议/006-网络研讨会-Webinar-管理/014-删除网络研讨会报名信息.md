删除指定网络研讨会（Webinar）的报名信息，支持删除成员手动报名的信息和导入的报名信息。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/webinar/enroll/delete?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "meetingid11234",
  "enroll_id_list": [{
    "enroll_id": "123456"
  }]
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
| meetingid | 是 | string|网络研讨会ID|
|enroll_id_list|是|object[]|报名ID列表。详见EnrollID。|

**EnrollID说明**
|参数|必须|类型|说明|
|:--|:--|
|enroll_id|是|string|报名ID|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许修改该应用创建的网络研讨会的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "total_count": 1
}
```

**参数说明**

| 参数 |类型| 说明|
| :--|
| errcode |int32| 返回码|
| errmsg |string| 对返回码的文本描述内容|
|total_count|uint32|成功删除的报名信息数量|
