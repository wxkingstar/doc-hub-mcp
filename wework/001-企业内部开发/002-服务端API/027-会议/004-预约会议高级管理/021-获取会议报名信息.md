获取已报名成员数量和报名成员答题详情。
>说明
>- 使用获取参会成员列表 接口获得成员的 tmp_openid 后，成员的报名 ID 可以通过 [获取会议成员报名 ID](#1234123)接口，或 [获取网络研讨会成员报名 ID](#12345) 接口来确认和匹配对应的报名信息。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/list?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "meetingid11234",
  "status": 0,
  "cursor": "CURSOR",
  "limit": 10
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
| meetingid | 是 | string|会议ID|
|status|否|int32|审批状态筛选字段，默认返回全部。审批状态：<br>0：全部。<br>1：待审批。<br>2：已拒绝。<br>3：已批准。|
|cursor|否|string|分页查询用，将上一个请求返回的next_cursor字段传入。第一次查询时可不传值|
|limit|否|int32|分页大小，最大50条。默认值为50|

>注意：
limit参数必须与首次调用获得cursor时传入的limit一致。

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "has_more": false,
  "next_cursor": "CURSOR",
  "enroll_list":[
    {
      "enroll_id": "mexxxxxx",
      "enroll_time": "2023/03/23 14:00",
      "enroll_source_type": 1,
      "nick_name": "NICKNAME",
      "status": 1,
      "userid": "USERID",
      "tmp_openid": "TMP_OPENID",
      "enroll_code": "CODE",
      "answer_list": [{
        "answer_content": [
          "CONTENT1",
          "CONTENT2"
        ],
        "is_required": 1,
        "question_num": 10,
        "question_title": "TITLE",
        "question_type": 1,
        "special_type": 1
      }]
    }
  ]
}
```

**参数说明**

| 参数|类型| 说明|
| :--|
| errcode |int32| 返回码|
| errmsg |string| 对返回码的文本描述内容|
|has_more |bool |是否还有待拉取的成员列表 |
|next_cursor |string |分页用，下一次拉取列表将该字段填入cursor字段 |
|enroll_list |object[] |当前页的报名列表。详见**Enroll**|

**Enroll 说明**
|参数 |类型 |说明|
|:--|
|enroll_id |string |报名 ID |
|enroll_time |string |报名时间（utc +8，非时间戳） |
|enroll_source_type |int32 |报名来源：<br>1：成员手动报名<br>2：批量导入报名 |
|nick_name |string |昵称。若通过手机号导入报名且未设置昵称，则该字段显示手机号|
|status |int32 |报名状态：<br>1：待审批<br>2：已拒绝<br>3：已批准|
|userid|string|为企业内成员时返回userid|
|tmp_openid |string |当场会议的成员临时 ID|
|enroll_code |string |报名码，用于电话入会的凭证|
|answer_list |object[] |答题对象列表，详见**Answer** |

**Answer 说明**
|参数 |类型 |说明|
|:--|
|answer_content| string[]| 回答内容：单选/简答只有一个元素，多选会有多个|
|is_required |int32 |是否必填：<br>1：否<br>2：是 |
|question_num |int32 |问题编号，1,2,3...等形式 |
|question_title |string |问题标题|
|question_type |int32 |问题类型：<br>1：单选<br>2：多选<br>3：简答|
|special_type |int32 |特殊问题类型：<br>1：无<br>2：手机号<br>3：邮箱<br>4：姓名<br>5：公司名称<br>**说明：**目前特殊问题均为简答题 |
