该接口用于取消一个指定的预约会议。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/cancel?access_token=ACCESS_TOKEN

**请求包体：**

```json
{
 "meetingid": "XXXXXXXXX",
 "sub_meetingid":"sub_meetingid1111"
}

```

**参数说明：**

| 参数 | 必须 |类型| 说明 |
|:--|
|access_token|是|string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
| meetingid | 是 |string| 会议id，仅允许取消预约状态下的会议|
|sub_meetingid|否|string|周期性子会议 ID。如果取消周期性会议且该字段不传，则会取消该系列的周期性会议。|

**权限说明：**

- 仅允许取消当前应用创建的会议。
- 自建应用需要配置在“可调用接口的应用”列表

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok"
}
```

**参数说明：**

| 参数 | 类型|说明 |
| ---------- | ---------------------- |
| errcode |int32| 返回码 |
| errmsg |string| 对返回码的文本描述内容 |
