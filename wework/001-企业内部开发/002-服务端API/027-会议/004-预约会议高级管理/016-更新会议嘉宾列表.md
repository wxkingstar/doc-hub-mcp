通过会议ID更新会议嘉宾列表。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/set_guests?access_token=ACCESS_TOKEN

**请求包体**
```
{
  "meetingid": "ID",
  "guests": [{
    "area": "86",
    "phone_number": "xxxxxxxxx",
    "guest_name": "xxxx"
  }]
}
```

**参数说明：**
| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meetingid|是|string|会议ID。|
|guests|是|object[]|会议嘉宾列表，详见Guest。|

**Guest说明**
|参数 |必须 |类型 |说明 |
|:--|
|area |是 |string |国家/地区代码（例如：中国传86，不是+86，也不是0086）。 |
|phone_number |是 |string |手机号。 |
|guest_name |否 |string |嘉宾名称。最大长度限制16个字符 |

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许修改该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：**
| 参数 | 类型|说明 |
| ---------- | ---|------------------- |
| errcode | int32|返回码 |
| errmsg |string| 对返回码的文本描述内容 |
