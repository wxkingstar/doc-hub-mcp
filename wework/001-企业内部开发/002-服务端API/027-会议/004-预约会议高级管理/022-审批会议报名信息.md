批量审批会议的报名信息。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/approve?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "meetingid11234",
  "action": 1,
  "enroll_id_list":[
    "mexxxxxxx"
  ]
}
```

**参数说明：**

| 参数 | 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
| meetingid | 是 | string|会议ID|
|action|是|int32|审批动作：<br>1：取消批准<br>2：拒绝<br>3：批准<br>**说明：**取消批准后状态将变成待审批。|
|enroll_id_list|是|string[]|报名ID列表|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许修改该应用创建的会议

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "handled_count": 10
}
```

**参数说明**

| 参数| 类型|说明|
|:--|
| errcode | int32|返回码|
| errmsg | string|对返回码的文本描述内容|
|handled_count|int32|成功处理的数量。
