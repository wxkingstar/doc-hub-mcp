此接口用于获取电话入会的成员ID，以tmp_openid返回。
- tmp_openid 是 PSTN 的识别标识，成员拨号进入后，在实时参会列表等 API及回调事件中可以获取。
- 由于可能出现一个座机号同时拨入的情况，所以同一个座机号可能对应多个 tmp_openid。
- 仅当 PSTN 接通后，该接口会返回对应的 tmp_openid。
- 支持查询会议下 PSTN 的历史 tmp_openid 列表。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/phone/get_tmp_openid?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "MEETINGID",
  "phone_numbers": [
    {
      "area": 86,
      "phone": "13333333333",
      "extension_number": "12345678"
    }
  ]
}
```

**参数说明：**

| 参数 | 必须 |类型| 说明|
| ------------ | ---- | --------------------
| access_token | 是 |string | 调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meetingid|是|string|会议ID|
|phone_numbers|是|object[]|外呼的电话号码对象数组，上限20个。详见PhoneNumber。|

**PhoneNumber说明**
|参数|必须|类型|说明|
|:--|:--|
|area|是|uint32|国家/地区代码。（例如：中国是86）|
|phone|是|string|电话号码或固定电话总机号|
|extension_number|否|string|固定电话分机号|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "tmp_openid_list": [
    {
      "area": 86,
      "phone": "13333333333",
      "extension_number": "12345678",
      "tmp_openid": "TMP_OPENID"
    }
  ]
}
```

**参数说明**

| 参数 | 类型|说明|
| -------------------- | -------------------------------------------------------------------------------------------------------------------- |
| errcode | int32|返回码|
| errmsg | string|对返回码的文本描述内容|
|tmp_openid_list|object[]|返回的 tmp_openid 对象数组。详见TmpOpenId。|

**TmpOpenId**
|参数|类型|说明|
|:--|
|area|uint32|国家/地区代码。（例如：中国是86）|
|phone|string|电话号码或固定电话总机号|
|extension_number|string|固定电话分机号|
|tmp_openid|string|当场会议的成员临时 ID，可用于会控操作，适用于所有成员。|
