此接口用于获取外呼电话入会的数据。
- 无论是从客户端、API 还是 Web 端发起的外呼，在该接口都可以获取到，不针对呼叫人做数据的隔离。
- 需要分页，分页最大条数100。
- Webinar 暂不支持外呼。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/phone/get_callout_status?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "MEETINGID",
  "cursor": "CURSOR",
  "limit": 10
}
```

**参数说明：**

| 参数 | 必须 |类型| 说明|
| ------------ | ---- | ----| --------------------
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meetingid|是|string|会议ID|
|cursor|否|string|分页查询用，将上一个请求返回的next_cursor字段传入。第一次查询时可不传值。|
|limit|否|uint32|分页大小，默认20，最大100。|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "phone_numbers": [
    {
      "area": 86,
      "phone": "13333333333",
      "extension_number": "12345678",
      "status": "STATUS",
      "tmp_openid": "TMP_OPENID"
    }
  ],
  "next_cursor": "NEXT_CURSOR",
  "has_more": false
}
```

**参数说明**

| 参数 | 类型| 说明|
| -------------------- | -------------------------------------------------------------------------------------------------------------------- |
| errcode | int32| 返回码|
| errmsg | string|对返回码的文本描述内容|
|phone_numbers|object[]|电话号码对象数组。详见PhoneNumber。|
|next_cursor|string|分页用，下一次拉取列表将该字段填入cursor字段|
|has_more|bool|是否还有待拉取的列表|

**PhoneNumber说明**
|参数|类型|说明|
|:--|
|area|uint32|国家/地区代码。（例如：中国是86）|
|phone|string|电话号码或固定电话总机号|
|extension_number|string|固定电话分机号|
|status|string|外呼状态|
|tmp_openid|string|当场会议的成员临时 ID，可用于会控操作，适用于所有成员。|
