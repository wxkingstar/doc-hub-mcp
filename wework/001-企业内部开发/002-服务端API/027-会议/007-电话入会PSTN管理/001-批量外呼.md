电话入会支持成员通过手机或座机拨打对应号码进入会议，也可在会中输入电话号码呼叫其他成员加入，通话质量更稳定。此接口用于创建批量电话入会呼叫。
- 在拨打后，立刻返回呼叫状态，可通过查询接口和回调事件获取。
- 支持在会议未开始、会中外呼。
- 每次调用支持批量外呼50路。
- 支持境外电话号及分机号的外呼。
- Webinar 暂不支持外呼。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/phone/callout?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "MEETINGID",
  "phone_numbers": [
    {
      "area": 86,
      "phone": "13333333333",
      "extension_number": "12345678"
    }
  ]
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meetingid|是|string|会议ID|
|phone_numbers|是|object|外呼的电话号码对象数组。详见PhoneNumber。|

**PhoneNumber说明**
|参数|必须|类型|说明|
|:--|:--|
|area|是|int32|国家/地区代码。（例如：中国是86）|
|phone|是|string|电话号码或固定电话总机号|
|extension_number|否|string|固定电话分机号|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许操作该应用创建的会议

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "phone_numbers": [
    {
      "area": 86,
      "phone": "13333333333",
      "extension_number": "12345678",
      "status": "STATUS"
    }
  ],
  "invalid_phone_numbers": [
    {
      "area": 86,
      "phone": "13333333333",
      "extension_number": "12345678"
    }
  ]
}
```

**参数说明**

| 参数 |类型 | 说明|
| -------------------- | -------------------------------------------------------------------------------------------------------------------- |
| errcode |int32| 返回码|
| errmsg |string| 对返回码的文本描述内容|
|phone_numbers|object[]|成功外呼的电话号码对象数组。详见SuccessPhoneNumber。|
|invalid_phone_numbers|object[]|不合法的外呼电话号码对象数组。详见PhoneNumber。|

**SuccessPhoneNumber说明**
|参数|类型|说明|
|:--|
|area|int32|国家/地区代码。（例如：中国是86）|
|phone|string|电话号码或固定电话总机号|
|extension_number|string|固定电话分机号|
|status|string|外呼状态|
