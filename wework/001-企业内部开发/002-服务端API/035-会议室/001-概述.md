企业通过会议室预定相关接口，可以方便的将企业已有系统的会议室预定管理，同步到企业微信的会议室应用。

##名词说明

● 会议室：即对应一个线下具体的会议室。会议室有一些基本的属性，如会议室名称、所在位置、可容纳人数、设备信息。
- ![](https://wework.qpic.cn/wwpic/719791_q_WoRHkfRoOzZve_1594800467/0)

● 会议室管理：包括新增、删除会议室、获取当前企业的会议室列表。
- ![](https://wework.qpic.cn/wwpic/219767_JafbV8eoQ3e-yEb_1594800553/0)

● 会议室预定：可以指定某一会议室在某个时间段由某位成员预定使用，其他成员则不能预定该会议室的当前时间段，除非原有预定者取消了会议室预定。
- ![](https://wework.qpic.cn/wwpic/634494_3YWNez2USGeCI3R_1594800569/0)

##开发流程

1. 进入管理后台-应用与小程序-会议室。

2. 将自建应用配置到“可调用接口的应用”里，然后通过相应的secret及corpid [获取access_token](#15074)。
- ![enter image description here](https://wework.qpic.cn/wwpic3az/67806_Q-9ly7PqRHKddTc_1701257538/0)

3. 通过获得的access_token调用会议室相关接口。
4. 会议室secret可以调用以下日程相关的接口：
  [创建日程](#26903)
  [更新日程](#44567)
  [更新重复日程](#41941)
  [新增日程参与者](#44568)
  [删除日程参与者](#44569)
  [获取日程详情](#44571)
  [取消日程](#44573)
