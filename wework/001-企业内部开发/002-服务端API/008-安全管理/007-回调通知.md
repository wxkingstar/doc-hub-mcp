##企业微信域名IP变更事件

企业微信的域名或IP发生变更时，回调此事件给企业。

权限要求：

| 应用类型 | 权限要求 | 接收事件开关|
| -------- | ---- | 
|自建应用|配置到「!!#ff0000 我的企业 - 设置 - 域名IP - 可调用API的应用!!」中|无|
|代开发应用|暂不支持| 无|
|第三方应用|暂不支持|无|

事件示例：

```
<xml>
  <ToUserName><
- ![CDATA[toUser]]></ToUserName>
  <FromUserName><
- ![CDATA[sys]]></FromUserName> 
  <CreateTime>1403610513</CreateTime>
  <MsgType><
- ![CDATA[event]]></MsgType>
  <Event><
- ![CDATA[security]]></Event>
  <ChangeType><
- ![CDATA[change_domain_ip]]></ChangeType>
</xml>
```

参数说明：

 | 参数 | 说明 |
 |----|---- |
 |ToUserName|企业微信CorpID |
 |FromUserName|此事件该值固定为sys，表示该消息由系统生成 |
 |CreateTime|消息创建时间 （整型） |
 |MsgType|消息的类型，此时固定为event |
 |Event|事件的类型，此时固定为security |
 |ChangeType|此时固定为change_domain_ip |
