上传打印机扫描文件后，在企业微信接收到文件推送。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/device/push_scan_file?device_access_token=DEVICE_ACCESS_TOKEN

**请求包体：**
```
{
  "open_userid": "zhangsan",
  "filename": "abc.pdf",
  "media_id":"MEDIAID",
  "download_url":"URL",
  "auth_code":"xxxxxxx"
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------------ | ---- | ------------- | ---------------------------------------- |
| device_access_token | 是 | string | device_access_token ,获取方式见[获取设备调用凭证](#40436) |
|open_userid|是|string|用户id|
|filename|是|string|文件名|
|media_id|否|string|临时素材id，获取方法见[上传临时素材](#10112)|
|download_url |否|string | cdn下载链接，由厂商侧提供（如有），没有cdn能力的厂商可以使用mediaid，当传入download_url时，接口会忽略mediaid字段|
|auth_code|是|string|上传扫描文件auth_code 参见 [用户扫描指令](#43754) auth_code字段|
 
**权限说明：**
只有勾选扫描能力的打印机才可以调用该接口。

**返回结果：** 

```
{
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| --------- | ------------------- |
| errcode | 错误码 |
| errmsg | 错误码描述 |
