最后更新：2022/07/28

可在model_access_token对应型号下录入单个设备sn。
>SN限制规则：
>1.SN全局唯一；
>2.SN不为空，且长度不超过128个字符；
>3.SN仅支持数字和大小写字母，即SN应由0-9 A-Z a-z构成，后续接口大小写不敏感。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/model/register_sn?model_access_token=MODEL_ACCESS_TOKEN
**请求包体：**

```
{
 "device_sn":"SN1"
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| model_access_token | 是 | string | model_access_token ,获取方式见[设备型号调用凭证](#40195) |
| device_sn | 是 | string | 设备SN（全局唯一） |

**返回结果：** 

```
{
  "errcode": 0,
  "errmsg": "ok",
  "qr_code":"https://open.work.weixin.qq.com/connect/hardware?hw_code=lS5***********"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| errcode | 错误码 |
| errmsg | 错误码描述 | 
| qr_code    | 设备静态二维码 |
