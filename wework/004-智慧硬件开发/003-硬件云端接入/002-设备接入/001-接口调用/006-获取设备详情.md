可获取model_access_token对应型号下sn设备的详情。

>SN应为对应型号下登记的设备SN。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/device/get_device_detail?model_access_token=MODEL_ACCESS_TOKEN
**请求包体：**

```
{
  "device_sn":"SN1"
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| model_access_token | 是 | string | model_access_token ,获取方式见[设备型号调用凭证](#40195) |
| device_sn | 是 | string | 设备SN |

**返回结果：** 

```
{
  "errcode": 0,
  "errmsg": "ok",
  "device_detail":{
    "model_name":"device_model",
    "default_name":"默认名",
    "qr_code":"https://open.work.weixin.qq.com/connect/hardware?hw_code=lS5***********",
    "bind_status":1,
    "auth_corp_info":
    {
      "open_corpid":"wpTD_dsdfsdfsdfsdfs1w",
      "corp_name":"测试企业"
    },
    "remark_name":"小区大门"
  }
}
```
**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| errcode | 错误码 |
| errmsg | 错误码描述 |
| device_detail.model_name | 设备型号名 |
| device_detail.default_name | 同型号设备出厂名 |
| device_detail.qr_code | 设备静态二维码 |
| device_detail.bind_status | 设备绑定状态：0-未绑定；1-已绑定，绑定情况下才会返回下述参数 |
| device_detail.bind_corpinfo.open_corpid | 绑定企业corpid |
| device_detail.bind_corpinfo.corp_name | 绑定企业corp_name |
| device_detail.remark_name | 设备备注名，若用户未设置则返回空字符串 |
