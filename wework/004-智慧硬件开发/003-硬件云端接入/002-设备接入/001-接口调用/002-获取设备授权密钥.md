企业绑定设备时，企业微信回调了授权code，通过该授权code，厂商可获取到该设备的授权密钥。当需要使用设备的身份调用接口时，需要使用设备授权密钥获取设备调用凭证。

::: warning 注意
① 设备授权密钥是获取设备调用凭证的关键，请妥善保管
② 设备授权密钥在企业解绑设备后失效
③ 设备授权code仅支持消费一次
:::

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/get_device_secret?model_access_token=MODEL_ACCESS_TOKEN

**请求包体：**

```
{
  "auth_code": "auth_code_value"
}
```

**参数说明：** 
 
 | 参数 | 是否必须 | 说明 |
 |----|---- | ----- |
 | auth_code | 是 | 企业绑定设备时，企业微信回调的授权code，长度为64至512个字节。 仅支持消费一次。|

**返回结果：** 
```
{
  "errcode":0,
  "errmsg":"ok",
  "device_secret": "ddddddd",
  "device_access_token": "xxxxxx", 
  "expires_in": 7200
}
 ```
**参数说明：** 

 | 参数 | 说明 |
 |----|---- |
 |device_secret|设备授权密钥，最长为512字节 |
 |device_access_token|设备access_token,最长为512字节。|
 |expires_in|设备access_token超时时间（秒）。 |
