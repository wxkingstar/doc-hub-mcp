该API用于获取设备型号调用接口的凭证（model_access_token）。
> 获取model_access_token时，需要model_ticket参数。model_ticket由企业微信后台定时推送给“指令回调URL”，每十分钟更新一次.
model_ticket实际有效期为30分钟，可以容错连续两次获取model_ticket失败的情况，但是请永远使用最新接收到的model_ticket。
> 通过本接口获取的model_access_token有效期为2小时，开发者需要进行缓存，不可频繁获取。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/get_model_token

**请求包体：**

```
{
  "model_id":"mdddddccc7775555aaa" ,
  "model_secret": "ldAE_H9anCRN21GKXVfdAAAAAAAAAAAAAAAAAA", 
  "model_ticket": "Cfp0_givEagXcYJIztF6sfbdmIZCmpaR8ZBsvJEFFNBrWmnD5-CGYJ3_NhYexMyw" 
}

 ```
**参数说明：**
 
 | 参数 | 是否必须 | 说明 |
 |----|---- | ----- |
 | model_id | 是 | 设备型号ID，在服务商管理端创建设备型号之后，可在页面获取该参数 |
 | model_secret | 是 | 设备型号secret，每个型号有一个唯一的secret |
 | model_ticket | 是 | 企业微信后台推送的ticket |

**返回结果：** 

```
{
  "errcode":0,
  "errmsg":"ok",
  "model_access_token":"61W3mEpU66027wgNZ_MhGHNQDHnFATkDa9-2llMBjUwxRSNPbVsMmyD-yq8wZETSoE5NQgecigDrSHkPtIYA",
  "expires_in":7200
}
```

**参数说明：** 

 | 参数 | 说明 |
 |----|---- |
 | model_access_token | 设备型号access_token,最长为512字节 |
 | expires_in | 有效期（秒） |
 
**注意**：因历史原因，该接口在调用失败时才返回errcode。没返回errcode视为调用成功
