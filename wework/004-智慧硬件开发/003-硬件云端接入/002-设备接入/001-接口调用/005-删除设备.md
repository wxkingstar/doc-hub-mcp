最后更新：2022/07/28

可在device_access_token对应的型号下删除尚未绑定的设备sn。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/model/unregister_sn?model_access_token=MODEL_ACCESS_TOKEN
**请求包体：**

```
{
 "device_sn":"SN1"
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| model_access_token | 是 | string | model_access_token ,获取方式见[设备型号调用凭证](#40195) |
| device_sn | 是 | string | 设备SN |

**返回结果：** 

```
{
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| errcode | 错误码 |
| errmsg | 错误码描述 |
