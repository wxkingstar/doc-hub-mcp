当设备连上云端、更新固件导致设备属性等变化时，上报、通知云端。

**请求包体：**

```
{
 "cmd":"report_device_feature",
 "headers":
 {
 "req_id":"xxxxx"
 },
 "body":
 {
 "device_feature": "{\"printer_type\":\"CM12838-W2\"}"
 }
}

```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |
| device_feature|是|string|终端设备反馈的自定义标识串（可以由厂商自由扩展）|

**返回结果：** 

```
{
  "headers": 
  {
   "req_id" : "xxxx"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
