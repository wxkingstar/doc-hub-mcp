推送用户删除任务的事件。

**指令包体：**

```
{
  "cmd": "push_del_printer_job",
  "headers": {
     "req_id": "xxxx"
   },
  "body":
  {
    "jobid_list": ["aa","bb"]
  }
}
```

**参数说明：**

| 参数名 | 类型 | 描述 |
| --------- | ------ | -------------------------------- |
| req_id | String | 透传请求的req_id |
| cmd | String | 指令号，此处固定为push_del_printer_job |
| jobid_list | Array(string) | 任务id列表 |
