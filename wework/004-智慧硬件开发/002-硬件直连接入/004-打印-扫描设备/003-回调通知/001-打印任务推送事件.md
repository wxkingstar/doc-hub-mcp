企业成员在app上提交打印任务，此时企业微信云端向设备推送任务。

**指令包体：**

```
{
  "cmd": "push_printer_job",
  "headers": {
     "req_id": "xxxx"
   },
  "body":
  {
    "jobid_list": ["aa","bb"]
  }
}
```

**参数说明：**

| 参数名 | 类型 | 描述 |
| --------- | ------ | -------------------------------- |
| req_id | String | 透传请求的req_id |
| cmd | String | 指令号，此处固定为push_printer_job |
| jobid_list | Array(string) | 任务id列表 |
