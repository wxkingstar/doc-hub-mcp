获取通讯录当前的版本号

**请求包体：**

```
{
  "cmd":"get_perm_version",
  "headers":
  {
    "req_id":"xxxxx"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |

**返回结果：** 

```
{
  "headers": 
  {
   "req_id" : "xxxx"
  },
  "body":
  {
    "perm_version": 5
  }
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
| perm_version | 当前云端通讯录版本号 |
