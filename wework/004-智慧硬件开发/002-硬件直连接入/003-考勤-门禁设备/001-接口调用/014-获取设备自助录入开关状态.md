获取当前设备开启自助录入开关状态

**请求包体：**

```
{
 "cmd":"get_open_self_input",
 "headers":
  {
    "req_id":"xxxxx"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |

**返回结果：** 

```
{
  "headers": 
  {
   "req_id" : "xxxx"
  },
  "body":
  {
    "open_self_input_fp":true,
    "open_self_input_fa":true
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ----------- | ----------- |
| req_id | 透传请求的req_id |
| open_self_input_fp | 指纹自助录入开关状态 |
| open_self_input_fa | 人脸自助录入开关状态 |
| errcode | 错误码 |
| errmsg | 错误码描述 |
