云端下发 门禁设备远程开门 的指令, 接收到该指令后设备应进行门禁开门操作。

**指令包体：**

```
{
  "cmd" : "push_remote_open_door",
  "headers": 
  {
     "req_id" : "xxxx"
  },
  "body":
  {
     "userid": "xxxx",
     "oper_id": "xxxx"
  }
}
```

**参数说明：**

| 参数名 | 类型 | 描述 |
| --------- | ------ | -------------------------------- |
| req_id | string | 透传请求的req_id |
| userid | string | 发起门禁远程开门指令的用户id |
| oper_id | string | 本条指令的唯一标识id |
| cmd | string | 指令号，此处固定为push_remote_open_door |

本指令从云端下发后，门禁设备应立即通过 [门禁远程开门结果上报](#34458) 上报本次指令下发后执行的结果。

指令执行结果上报完成后，需将人员通行记录通过 [打卡/通过门禁](#15362) 接口上报到云端，通行上报没有实时性要求。
