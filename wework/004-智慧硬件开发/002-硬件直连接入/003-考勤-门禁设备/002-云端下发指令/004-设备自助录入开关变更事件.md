企业设置设备是否允许成员自助录入指纹或者人脸时推送该事件

**指令包体：**

```
{
 "cmd": "push_open_self_input",
 "headers": {
 "req_id": "xxxx"
 },
 "body":
 {
 "open_self_input_fp":true，
 "open_self_input_fa":false
 }
}
```

**参数说明：**

| 参数名 | 类型 | 描述 |
| --------- | ------ | -------------------------------- |
| req_id | String | 透传请求的req_id |
| cmd | String | 指令号，此处固定为push_open_self_input |
| open_self_input_fp | bool | 是否支持自助录入指纹|
| open_self_input_fa | bool | 是否支持自助录入人脸|
