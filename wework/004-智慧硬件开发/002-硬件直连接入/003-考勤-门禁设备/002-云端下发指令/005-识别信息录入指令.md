控制考勤机进入录入信息的页面（或流程）

**指令包体：**

```
{
  "cmd" : "push_enter_page",
  "headers": 
  {
    "req_id" : "xxxx"
  },
  "body": 
  {
    "page_type": "fa",
    "oper_id": "xxxxx",
    "userid":"xx",
    "user_type":0
   }
}
```

**参数说明：**

| 参数名 | 类型 | 描述 |
| --------- | ------ | -------------------------------- |
| req_id | string | 透传请求的req_id |
| cmd | string | 指令号，此处固定为push_enter_page |
| page_type | string | 页面类型： ”fa”:录入脸部识别信息， “fp”：录入指纹信息 |
| userid | string | 成员id（字符串） |
| user_type |Uint32 | 成员的类型： 0：企业员工 1：访客 2：学生 不填则默认值为0 |
| oper_id | string | 本次操作的id |
