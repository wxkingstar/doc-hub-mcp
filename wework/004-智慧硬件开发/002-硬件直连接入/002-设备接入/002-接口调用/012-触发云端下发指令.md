为了方便联调，设备可以自己触发企业微信云端下发通知

**请求包体：**

```
{
  "cmd":"selfhelp_notify",
  "headers":
  {
    "req_id":"xxxxx"
  },
  "body":
  {
    "data": "helloworld"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string |请求的id，自行保证不会重复即可 |
| data | 是 | string|服务器会透传该值给硬件，如包含json语法，务必进行转义 |

**返回结果：** 

```
{
  "headers": 
  {
   "req_id" : "xxxx"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ----------- | ----------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
