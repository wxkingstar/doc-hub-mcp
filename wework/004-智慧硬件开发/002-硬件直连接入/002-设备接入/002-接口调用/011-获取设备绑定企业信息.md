获取当前设备绑定企业信息

**请求包体：**

```
{
 "cmd":"get_corp_info",
 "headers":
  {
    "req_id":"xxxxx"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |

**返回结果：** 

```
{
  "headers": 
  {
   "req_id" : "xxxx"
  },
  "body":
  {
    "corp_name":"xxxx"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ----------- | ----------- |
| req_id | 透传请求的req_id |
| corp_name | 企业名称 |
| errcode | 错误码 |
| errmsg | 错误码描述 |
