获取服务器时间戳，此接口不需要校验登录态，机器开机后应主动对齐一下服务器时间。

**请求包体：**

```
{
   "cmd":"get_timestamp",
   "headers":
  {
    "req_id":"xxxxx"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |

**返回结果：** 

```
{
  "headers": 
  {
    "req_id" : "xxxx"
  },
  "body":
  {
    "timestamp":123456
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| --------- | ----------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
| timestamp | 服务器时间戳(s) |
