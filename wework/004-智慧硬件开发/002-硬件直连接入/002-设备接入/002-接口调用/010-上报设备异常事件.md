上报设备异常

**请求包体：**

```
{
 "cmd":"error_report",
 "headers":
  {
    "req_id":"xxxxx"
  },
  "body":
  {
     "errcode": -1,
     "errmsg": "connect timeout"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 描述 |
| ------- | ---- | ------------------ |
| req_id | 是 | 请求的id，自行保证不会重复即可 |
| errcode | 是 | 错误码 |
| errmsg | 是 | 错误描述，如网络错误，设备人数超限等 |

**返回结果：** 

```
{
  "headers": {
     "req_id" : "xxxx"
   },
   "errcode": 0,
   "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------- | ----------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
