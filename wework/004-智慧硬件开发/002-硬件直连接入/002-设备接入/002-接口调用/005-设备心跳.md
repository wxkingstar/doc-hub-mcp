最后更新：2018/12/06

设备登录成功之后，需要每20~40秒调用一次

**请求包体：**

```
{
 "cmd":"ping",
 "headers":
  {
    "req_id":"xxxxx"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |

**返回结果：** 

```
{
  "headers": 
  {
     "req_id" : "xxxx"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ----------- | ----------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
