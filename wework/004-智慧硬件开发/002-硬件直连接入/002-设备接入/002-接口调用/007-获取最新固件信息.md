获取最新版本固件信息。

**请求包体：**

```
{
  "cmd":"get_firmware_info",
  "headers":
  {
    "req_id":"xxxxx"
  }
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------------- | ---- | ---------- | ---------------- |
| req_id | 是 | string | 请求的id，自行保证不会重复即可 |

**返回结果：** 

```
{
  "headers": 
  {
   "req_id" : "xxxx"
  },
  "body":
  {
    "firmware_version": "1.0.1001",
    "md5sum": "7ed1f027745e3e05fe7489b8f27def35",
    "size": 12345,
    "download_url":"http://DOWNLOAD_URL/PATH"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ---------------- | --------------------------- |
| req_id | 透传请求的req_id |
| errcode | 错误码 |
| errmsg | 错误码描述 |
| firmware_version | 当前固件版本号（string，比较函数由硬件自己实现） |
| md5sum | 固件md5值 |
| size | 固件大小（Bytes） |
|download_url |下载地址|
