查询指定企业下的平台能力服务账号列表。
>若为上下游场景，corpid指定的为上游企业，仅返回上游企业激活的账号；若corpid指定为下游企业，若激活码为上游企业分享过来的且已绑定，也会返回。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/license/list_actived_account?provider_access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "corpid" : "CORPID",
  "limit":500,
  "cursor":"xxxx"
}
```

**参数说明：** 

| 参数 | 是否必须 | 说明 |
| ------- | ---- | ---- |
| provider_access_token | 是 | 应用服务商的接口调用凭证，获取方法参见[服务商的凭证](#15143 ) |
| corpid | 是 | 企业corpid 。若为上下游场景，为上游企业corpid |
|limit|否|返回的最大记录数，整型，最大值1000，默认值500|
|cursor|否|用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填|

**返回结果：** 
```
{
  "errcode": 0,
  "errmsg": "ok",
  "next_cursor":"xxx",
  "has_more":1,
  "account_list":[
    {
      "userid": "user1",
      "type": 1,
      "expire_time":1500000000,
      "active_time":1500000000
    },
    {
      "userid": "user2",
      "type": 1,
      "expire_time":1500000000,
      "active_time":1500000000
    }
  ]
}
```

**参数说明：** 

| 参数 | 说明 |
| ------- | ---- |
| errcode |错误码 |
|errmsg|错误码说明|
|next_cursor|分页游标，再下次请求时填写以获取之后分页的记录，如果已经没有更多的数据则返回空|
|has_more |是否结束 |
|account_list|已激活成员列表，已激活过期的也会返回|
|account_list.userid|企业的成员userid。返回加密的userid|
|account_list.type|激活码账号类型：1:基础账号，2:互通账号|
|account_list.expire_time|过期时间|
|account_list.active_time|激活时间|
