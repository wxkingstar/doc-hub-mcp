**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/service/get_order?suite_access_token=SUITE_ACCESS_TOKEN

**请求包体：**

```
{
  "orderid" : "2018091822ks1sd3s"
}
```

**参数说明：** 

| 参数 | 是否必须 | 说明 |
| ------- | ---- | ---- |
| orderid | 是 | 订单号 |

**返回结果：** 

```
{ 
 "errcode" : 0,
 "errmsg" : "ok",
 "orderid" : "2018091822ks1sd3s",
 "order_status" : 1,
 "order_type" : 1,
 "paid_corpid" : "wwfedd7e5291d63aaa",
  "operator_id" : "zhangsan",
 "suiteid" : "wx67cce113441ccaaa",
 "appid" : 1,
 "edition_id" : "RLS65535",
 "edition_name" : "协同版",
 "price" : 100,
 "user_count" : 1000,
 "order_period": 365,
 "order_time" : 1533702999,
 "paid_time" : 1533702910,
  "begin_time" : 1533702910,
  "end_time" : 1553515904,
  "order_from" : 1, 
  "operator_corpid" : "wwfedd7e5292d63aaa",
  "service_share_amount" : 60,
  "platform_share_amount" : 10,
  "dealer_share_amount" : 30,
  "dealer_corp_info": 
  {
    "corpid": "xxxx",
    "corp_name": "name"
  }
}
```

**参数说明：**

| 参数 | 说明 |
| ------------- | ---------------------------------------------------- |
| errcode | 错误码 |
| errmsg | 错误码说明 |
| orderid | 订单号 |
| order_status | 订单状态。0-待支付，1-已支付，2-已取消， 3-支付过期， 4-申请退款中， 5-退款成功， 6-退款被拒绝|
| order_type | 订单类型。0-新购应用，1-扩容应用人数，2-续期应用时间，3-变更版本 |
| paid_corpid | 客户企业的corpid |
| operator_id | 下单操作人员userid。部分情况没有该字段（如服务商代下单；服务商下的免支付订单等） |
| suiteid | 应用id |
| appid | 套件应用id（仅旧套件有该字段）|
| edition_id | 购买版本ID |
| edition_name | 购买版本名字 |
| price | 应付价格，单位分 |
| user_count | 购买的人数 |
| order_period | 购买的时长，单位为天 |
| order_time | 下单时间（UNIX时间戳） |
| paid_time | 付款时间（UNIX时间戳） |
| begin_time | 购买生效期的开始时间（UNIX时间戳） |
| end_time | 购买生效期的结束时间（UNIX时间戳） |
| order_from | 下单来源。0-企业下单；1-服务商代下单；2-代理商代下单; 3-服务商收银台下单 |
| operator_corpid | 下单方corpid |
| service_share_amount | 服务商分成金额，单位分 |
| platform_share_amount | 平台分成金额，单位分 |
| dealer_share_amount | 代理商分成金额，单位分 |
| dealer_corp_info | 渠道商信息（仅当有渠道商报备后才会有此字段） |
| corpid | 代理商corpid |
| corp_name | 代理商的企业简称 |
