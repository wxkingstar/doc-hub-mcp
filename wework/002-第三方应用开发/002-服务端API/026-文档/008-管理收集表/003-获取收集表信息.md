该接口用于读取收集表的信息

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/wedoc/get_form_info?access_token=ACCESS_TOKEN 

**请求包体**

```json
{
 "formid":"FORMID"
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
| -------------------- | ------ | -------- | ------------------------------------------ |
| formid | string | 是 | 操作的收集表ID |

**权限说明**
- 自建应用需配置到“[可调用应用](#43883)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限
- 只能操作该应用创建的文档

| form_setting | object | 收集表的设置 |
| repeated_id | string[] | 收集表的周期id，用于获取答案列表和具体的回答 |
