当企业在管理端后台点击同意授权第三方应用的“组织架构”权限时，会触发该事件的回调。该事件会推送到第三方应用的**指令回调URL**上。

**请求包体：**
```
  <xml>
    <SuiteId><
- ![CDATA[ww4asffe99e54c0fxxxx]]></SuiteId>
    <AuthCorpId><
- ![CDATA[wxf8b4f85f3a79xxxx]]></AuthCorpId>
    <InfoType><
- ![CDATA[corp_arch_auth]]></InfoType>
    <TimeStamp>1403610513</TimeStamp>
  </xml>
```

**参数说明：**

 | 参数 | 说明 |
 |---- | ----- |
 |SuiteId | 第三方应用ID|
 |AuthCorpId|授权企业的CorpID|
 |InfoType | 事件类型，此时固定为corp_arch_auth |
 |TimeStamp | 时间戳 |
