第三方服务商可以用此接口获取授权企业中某个第三方应用或者代开发应用的管理员列表(不包括外部管理员)，以便服务商在用户进入应用主页之后根据是否管理员身份做权限的区分。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/agent/get_admin_list?access_token=ACCESS_TOKEN

**参数说明：** 

 | 参数 | 是否必须 | 说明 |
 |----|---- | ----- |
 | access_token | 是 | 代开发自建应用access_token，获取方法参见[获取access_token](#15074)<br>第三方应用access_token，获取方法参见[获取企业凭证](#14944)|

**返回结果：** 
```
{
 "errcode": 0,
 "errmsg": "ok",
 "admin":[
 {"userid":"woxxxx","auth_type":1},
 {"userid":"woxxxx","auth_type":0}
 ]
}
```

**参数说明：** 

 | 参数 | 说明 |
 |----|---- |
 |errcode|错误码，0表示正常返回，可读取admin的管理员列表|
 |errmsg|错误说明|
 |admin|应用的管理员列表（不包括外部管理员）。[成员授权模式](#30245)下，不返回管理员列表|
 |userid|管理员的userid|
 |auth_type|该管理员对应用的权限：0=发消息权限，1=管理权限|
 
 注：旧的使用suite_access_token[获取管理员](#14945)的方式已不再维护，建议使用当前新的接口
