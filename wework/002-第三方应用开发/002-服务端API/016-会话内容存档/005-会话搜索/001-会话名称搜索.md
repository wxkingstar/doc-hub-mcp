服务商可调用接口对企业的群会话名称进行搜索
**权限说明**
> 对于搜索会话名称，**仅支持有命名的群聊**，对于单聊和未命名的群聊不支持搜索
> 搜索会话名称时，基于当前最新的会话名称搜索

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|暂不支持 |
|代开发应用|暂不支持|
|第三方应用|暂不支持|
|会话存档接口授权|需具备「!!#ff0000 会话存档接口权限- 使用展示组件!!」权限|

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/chatdata/search_chat?access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "query_word": "VIP",
  "limit": 50,
  "cursor": "CURSOR"
}
```

**参数说明：** 

| 参数 | 是否必须 | 说明 |
| ------- | ---- | ---- |
|access_token|是|调用接口凭证|
|query_word|是|搜索的文本，要求大于等于2个UTF8字符。|
|limit|否|用于分页查询，每次请求返回的数据上限。最大为100，不填默认为50|
|cursor|否|用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填|

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "has_more": 1,
  "next_cursor": "CURSOR",
  "chat_list":[
    {
      "chatid": "CHATID1"
    },
    {
      "chatid": "CHATID2"
    }
  ]
}
```
**参数说明：**

| 参数 | 说明 |
| ------- | ---- |
| errcode |错误码 |
|errmsg|错误码说明|
|has_more|是否还有更多数据未获取，不填默认为0<br>0: 否<br>1: 是|
|next_cursor|下一次请求的cursor值|
|chat_list|搜索结果的群列表|
|chatid|群聊会话id|
