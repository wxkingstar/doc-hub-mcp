当会话存档接口授权范围内成员的会话命中关键词规则时，企业微信服务器会向「会话存档接口授权」配置的“指令回调URL”推送相应的事件消息。消息接收方式参见[回调接口](#14946/#概述)。

**事件示例：**
```
<xml>
<SuiteId><
- ![CDATA[xxxx]]></SuiteId>
<AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
<InfoType><
- ![CDATA[hit_keyword]]></InfoType>
<Token><
- ![CDATA[zzzzzzz]]></Token>
<TimeStamp>1403610513</TimeStamp>
</xml>
```

**参数说明：**
|参数 |说明 |
|:--|
|SuiteId |「会话存档接口授权」的应用ID |
|AuthCorpId |授权企业的CorpID |
|InfoType |事件类型，此时固定为hit_keyword |
|Token|10分钟内有效，用于调用[获取命中关键词规则的会话记录](#42728)，不超过128字节|
|TimeStamp |时间戳 |
