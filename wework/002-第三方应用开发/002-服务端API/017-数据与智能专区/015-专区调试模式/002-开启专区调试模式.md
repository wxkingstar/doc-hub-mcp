## 应用开启调试模式

应用通过接口将指定的程序开启调试模式

**权限说明**

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|需具备「!!#ff0000 数据与智能专区权限!!」|
|代开发应用|需具备「!!#ff0000 数据与智能专区权限!!」|
|第三方应用|需具备「!!#ff0000 数据与智能专区权限!!」|

> 对于第三方应用，仅通过测试安装的企业可开启调试模式
> 对于代开发应用，企业必须为服务商的测试企业

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/chatdata/open_debug_mode?access_token=ACCESS_TOKEN

**请求包体：**

```
{
   "program_id": "xxx",
  "debug_token": "xxx"
}
```

**参数说明：** 

| 参数 | 是否必须 | 说明 |
| ------- | ---- | ---- |
|program_id|是|应用关联的程序id|
|debug_token|是|程序的调试凭证|
 
**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok"
}
```
**参数说明：**

| 参数 | 说明 |
| ------- | ---- |
| errcode |错误码 |
|errmsg|错误码说明|

## SDK开启调试模式
| 依赖的参数 | 说明 |
| ---- | ---- |
|access_token|应用的access_token，注意!!#ff0000 必须确保在有效时间内，一般是2小时!!|
|debug_token|程序的调试凭证|

### Python
``` python
from wwspecapisdk import spec_open_debug_mode

spec_open_debug_mode(debug_token, access_token)
```
### C++
``` c++
#include "cpp_sdk.h"

bool is_succ = Tencent::WxWork::SpecOpenDebugMode(debug_token, access_token);
```
### Java
``` java
import com.tencent.wework.SpecUtil;

boolean isSuccess = SpecUtil.OpenDebugMode(debugToken, accessToken);
```
