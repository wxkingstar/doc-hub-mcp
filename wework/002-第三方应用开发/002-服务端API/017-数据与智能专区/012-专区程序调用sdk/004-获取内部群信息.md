可通过此接口，获取本企业的内部群信息，包括群主id、群创建时间以及所有群成员的id与加入时间。

**权限说明**
> 内部群中至少存在一个员工在授权范围内

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|需具备「!!#ff0000 数据与智能专区权限!!」 |
|代开发应用|需具备「!!#ff0000 数据与智能专区权限!!」|
|第三方应用|需具备「!!#ff0000 数据与智能专区权限!!」|

**请求方法**
通过SDK调用，具体方式参考[专区程序使用指引](#54006)。

**请求接口名：**
get_group_chat

**请求示例：**
```
{
  "chatid":"wrNplhCgAAIVZohLe57zKnvIV7xBKrig"
}

```

**参数说明：**

| 参数 | 必须 | 说明 |
| :----------- | :--- | :----------------- |
| chatid | 是 | 待查询的群id |

此接口可以查询chatid对应的群信息，chatid可以从[获取会话记录](#53341)中获取到的chatid填充。只支持内部群和企业互联群。

**返回结果**
```
{
 "errcode": 0,
 "errmsg": "ok",
 "creator": "woAAAAAAA",
 "room_create_time": 1592361604,
 "members": [
 {
 "memberid": "woAAAAAAAA",
 "jointime": 1592361605,
      "type": 1
 },
 {
 "memberid": "woBBBBBBBB",
 "jointime": 1592377076,
      "type": 2
 }
 ]
}
```

**返回字段说明：**

| 字段名 | 说明 |
| ------------ | ------------ |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| creator| chatid对应的群创建者，userid|
| room_create_time| chatid对应的群创建时间|
| members| chatid对应的群成员列表|
| memberid| chatid群成员的id，userid、external_userid、机器人id|
| jointime| chatid群成员的入群时间|
|type|chatid群成员的类型<br>1：员工<br>2：外部联系人<br>3：机器人|

**错误说明：**

| 返回码 | 说明 |
| ------------ | ------------ |
| 301052 | 会话存档已过期 |
