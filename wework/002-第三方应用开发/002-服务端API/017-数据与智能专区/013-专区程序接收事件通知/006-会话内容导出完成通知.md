## 会话内容导出任务完成回调

当会话内容导出任务完成时，企业微信专区后台会向有企业会话内容权限的应用关联的程序推送相应的事件消息。

**事件示例：**
```
{
 "event_type": "chat_archive_export_finished",
 "timestamp": 1408091189,
 "chat_archive_export_finished": {
 "jobid": "xxx"
 }
}
```

**参数说明：**
|参数 |说明 |
|:--|
|event_type |事件类型，此时固定为chat_archive_export_finished |
|jobid|24小时内有效，用于调用[获取会话内容导出任务结果](#53355/获取会话内容导出任务结果)，不超过64个字符|
|timestamp |时间戳 |
