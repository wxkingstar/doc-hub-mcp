本事件是接口在使用异步导出接口时，用于接收任务执行完毕的结果通知，事件回调到提交任务的第三方套件配置的回调地址。

请求示例：

```
<xml>
  <SuiteId><
- ![CDATA[ww543f52cb62809e5a]]></SuiteId>
  <InfoType><
- ![CDATA[batch_job_result]]></InfoType>
  <TimeStamp>1628653600</TimeStamp>
  <BatchJob>
    <JobId><
- ![CDATA[jobid_FTR983N1FNGMsoZKXogNzmLwS8cwu150GTufJB]]></JobId>
    <JobType><
- ![CDATA[export_user]]></JobType>
    <ErrCode>0</ErrCode>
    <ErrMsg><
- ![CDATA[ok]]></ErrMsg>
  </BatchJob>
</xml>
```

参数说明：

 | 参数 | 说明 |
 |----- | ----- |
 | SuiteId | 套件ID |
 | CreateTime | 消息创建时间（整型）|
 | InfoType | 事件类型：batch_job_result |
 | JobId | 异步任务id，最大长度为64字符 |
 | JobType | 操作类型，字符串，在异步导出的场景下分别有：export_user(导出成员详情)、 export_simple_user(导出成员）、export_department(导出部门）、export_taguser(导出标签成员)|
 | ErrCode | 返回码 |
 | ErrMsg | 对返回码的文本描述内容 |
