**参数说明：**

 | 参数 | 说明 |

 |ServiceCorpId|服务商corpid |
 |InfoType|事件类型，batch_job_result，通讯录导出完成回调 |
 |TimeStamp|回调事件的时间戳 |
 |AuthCorpId|企业corpid|
 |JobId|任务id,最大长度为64字符|
 |JobType|任务类型，目前有contact_id_translate|
