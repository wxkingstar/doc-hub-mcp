当企业当前授权模式为[成员授权](#30245)时，可调用该接口获取成员授权列表。

**请求方式：**POST（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/user/list_member_auth?access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "cursor": "NEXT_KEY",
  "limit": 20
}
```
**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ------------ | ------------ |
| access_token | 是 | 企业调用接口凭证 |
|cursor|否|上一次调用时返回的next_cursor，第一次拉取可以不填|
|limit|  否|  每次拉取的数据量，默认值和最大值都为1000|

**权限说明：**

仅企业为[成员授权模式](#30245)下可调用

**返回结果：**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "next_cursor":"next_cursor",
  "member_auth_list":[
    {
      "open_userid":"xxx"
    }
  ]
}

```
**参数说明：**

| 参数 | 说明 |
| ------------ | ------------ |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|next_cursor|当前数据最后一个key值，如果下次调用带上该值则从该key值往后拉，用于实现分页拉取，返回空字符串代表已经是最后一页|
|member_auth_list|成员授权列表|
|open_userid|当用户为某个企业内部成员时返回。全局唯一，对于同一个服务商不同的应用open_userid是相同的；同一用户，对于不同服务商open_userid是不同的|
