最后更新：2021/09/07

当企业当前授权模式为[成员授权](#30245)时，可调用该接口查询成员用户是否已授权

**请求方式：**POST（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/user/check_member_auth?access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "open_userid": "xxx"
}
```
**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ------------ | ------------ |
| access_token | 是 | 企业调用接口凭证 |
|open_userid|企业成员的open_userid。全局唯一，对于同一个服务商不同的应用open_userid是相同的；同一用户，对于不同服务商open_userid是不同的|

**权限说明：**

仅企业为[成员授权模式](#30245)下可调用

**返回结果：**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "is_member_auth": true
}

```
**参数说明：**

| 参数 | 说明 |
| ------------ | ------------ |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|is_member_auth|[成员授权模式](#30245)下，该成员已授权|
