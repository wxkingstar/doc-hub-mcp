该接口用于取消指定的日程。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/del?access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "schedule_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "op_mode": 1,
  "op_start_time": 1571274600
}
```

**参数说明：** 

| 参数 | 是否必须 |类型| 说明 |
| ----------- | -------- | ---------| --------------------------------- |
|schedule_id| 是 | string|日程ID |
|op_mode|否|uint32|操作模式。是重复日程时有效。</br>0-默认删除所有日程；</br>1-仅删除此日程；</br>2-删除本次及后续日程</br>详见[重复日程的不同操作模式](#41941)|
|op_start_time|否|uint32|操作起始时间。仅当操作模式是1或2时有效。该时间必须是重复日程的某一次开始时间|

**返回结果：** 

```
{ 
  "errcode": 0,
  "errmsg" : "ok"
}
```

**参数说明：**

| 参数 | 说明 |
| ------------ | ----------------------------------------- |
| errcode | 错误码|
| errmsg | 错误码说明|
