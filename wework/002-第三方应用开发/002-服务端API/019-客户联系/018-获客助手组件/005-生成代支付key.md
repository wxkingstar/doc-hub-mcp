服务商可通过此接口生成用于对应链接代支付的key。key使用一次之后失效。生成key之后需要在14天内使用。

**请求方式：**POST(**HTTPS**)
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/create_once_key?access_token=ACCESS_TOKEN

**请求示例：**
```
{
 "link_id":"xxxxxx",
 "key_num":1,
}
```

**参数说明**:

|参数|必须|说明|
|:-----------|:---|:-----------------|
|access_token | 是 | [调用接口凭证](#14944) |
| link_id | 是 | 获客链接id |
| key_num | 否 | 生成的代支付key数量，默认100，最大可填写1000 |

**权限说明：**
- 获客助手组件可调用，需企业授权时为服务商代支付模式才可调用

**返回结果：**
```
{
 "errcode": 0,
 "errmsg": "ok",
 "keys":[
      "xxxxxx"
 ]
}
```
 
将生成的key拼在带组件参数的获客链接上，添加好友时即扣除的是服务商的使用量

**参数说明：**

| 参数 | 说明 |
| :-|
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| keys|生成的代支付key的列表|
