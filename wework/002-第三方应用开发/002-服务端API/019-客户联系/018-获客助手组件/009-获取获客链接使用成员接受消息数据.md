##获取获客链接使用成员接受消息数据
获客组件可通过此接口获取成员收消息情况，如次数、客户来源等信息。

**请求方式**: **POST(HTTP)**

**请求地址:**https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get_chat_info?access_token=ACCESS_TOKEN

**请求示例**

```
{
   "chat_key":"CHAT_KEY"
}
```

**参数说明**:

|参数|必须|说明|
|:-----------|:---|:-----------------|
|access_token | 是 | 调用接口凭证 |
|chat_key|是|[成员多次收消息事件](#42924/成员多次收消息事件)中回调的会话信息凭据ChatKey，回调后30分钟内有效|

**返回结果**:

```
{
  "errcode": 0,
  "errmsg": "ok",
  "chat_info":
  {
      "recv_msg_cnt":3,
      "link_id":"caXXXXXXX",
      "state":"STATE"
  }
}
```

**参数说明**:

|参数|说明|
|:-----------|:-----------------|
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|chat_info.recv_msg_cnt|成员收到的某个客户的消息次数|
|chat_info.link_id|成员添加客户的获客链接id|
|chat_info.state|成员添加客户的state|
