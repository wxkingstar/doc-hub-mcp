# 提升广告有效率的几种方式

在广告中使用获客助手时，由于链路短、转化高，可降低获客成本，但同时也因为链路短，缺少了筛客环节，会添加一些低意向客户。以下方式可帮你过滤低质线索，提升广告有效率。

##通过优化广告落地页提升有效率
可在落地页进行表单收集，筛选客户意向，再引导填写完成的客户添加成员。
- ![1.png](https://wework.qpic.cn/wwpic3az/19676_dC8kxfTbTFeuone_1702966282/0)

##通过广告回传提升有效率
除了用[「客户添加」](#15260/添加企业客户事件)这种浅层动作进行广告回传以外，还可用获客助手提供的[「客户首次发消息」](#47918/微信客户发起会话事件)等后端行为进行广告回传。
- ![2.png](https://wework.qpic.cn/wwpic3az/605350_REu8gYNaQESrNSE_1702966394/0)

##通过微信客服互动问答提升有效率
可引导客户从广告进入微信客服会话，通过互动问答进一步了解产品详情，明确客户需求后[发送获客链接](#31152/获客链接消息)，引导客户添加。
- ![3.png](https://wework.qpic.cn/wwpic3az/757751_Xpqvx17tSKO10fg_1702966453/0)
