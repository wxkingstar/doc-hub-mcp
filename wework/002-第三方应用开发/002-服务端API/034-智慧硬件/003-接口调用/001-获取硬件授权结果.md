此接口用于第三方应用获取企业授权的硬件设备列表，后续第三方应用可获取这批设备产生的数据，例如考勤记录、门禁通行记录、测温记录等。

> 注意：
> 1.第三方应用须具有“设备接口信息权限”；
> 2.企业须同时安装硬件设备和第三方应用；
> 3.企业须向第三方应用**二次授权** 具体硬件设备的原始数据。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/devicedata/get_auth_info?access_token=ACCESS_TOKEN

**请求包体：**
```
{
  "agentid":1
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| access_token | 是 | string | 调用接口凭证。见[获取access_token](#14944) |
|agentid|否|uint32|应用id，仅套件应用需要填写该字段|

**返回结果：** 

```
{
  "errcode": 0,
  "errmsg": "ok",
  "device_list": {
    "item":[
      {
        "device_sn":"SN",
        "remark_name":"设备备注名",
        "default_name":"设备出厂名",
        "model_name":"PRODUCT_MODEL",
        "device_type":1,
        "device_ability":1
      }
    ]
  }
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| errcode | 错误码 |
| errmsg | 错误码描述 |
| device_list | 授权读取数据的设备列表 |
| device_list.item.SN |设备SN       |
| device_list.item.remark_name | 企业设置的备注名称 |
| device_list.item.default_name | 设备出厂时的名称 |
| device_list.item.model_name | 设备出厂时的型号 |
| device_list.item.device_type | 设备类型：1-考勤/门禁设备 |
| device_list.item.device_ability | 设备能力：<br>1-具备考勤能力；<br>2-具备门禁能力；<br>4-具备测温能力；<br>设备能力字段可叠加，例如：返回5表示设备具备考勤和测温能力|
