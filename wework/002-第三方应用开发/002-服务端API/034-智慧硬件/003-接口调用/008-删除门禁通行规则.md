第三方应用向企业微信删除企业已经存在的的门禁规则。

>注意：
1.第三方应用须具有“设备接口信息权限-创建门禁规则”授权；
2.企业须同时安装硬件设备和第三方应用；
3.第三方应用只能删除自己创建的门禁规则

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/devicedata/del_accesscontrol_rule?access_token=ACCESS_TOKEN
**请求包体：**

```
{
 "rule_id":"xxx"
}
```

**参数说明：**
 
| 参数名 | 是否必须 | 类型 | 描述 |
| ------ | ---- | ------ | ---------------- |
| access_token | 是 | string | 应用调用接口权限 |
| rule_id | 是| string | 企业微信返回的rule_id，门禁规则的标识id |

**返回结果：** 

```
{
  "errcode": 0,
  "errmsg": "ok"
}
```

**参数说明：** 

| 参数名 | 描述 |
| ------------ | ----------- |
| errcode | 错误码 |
| errmsg | 错误码描述 |
