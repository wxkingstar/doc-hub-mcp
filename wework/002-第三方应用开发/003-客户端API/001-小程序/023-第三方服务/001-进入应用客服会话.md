**接口说明**：第三方应用可以使用该接口提供按钮，让用户可快速打开应用客服的会话。

**调用前提：**
> 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态。
仅第三方应用可调用。
第三方应用需要提前配置客服，否则调用返回fail。
当前用户需要有添加外部联系人权限，否则调用返回fail。

### wx.qy.openThirdAppServiceChat(Object object)
#### 参数

##### Object object

| 参数名 | 类型 | 必填 | 说明 | 企业微信版本要求 |
| -------- | -------- | ---- | -------------------------------- | --- |
| success | Function | 否 | 接口调用成功的回调函数 | |
| fail | Function | 否 | 接口调用失败的回调函数 | |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） | &nbsp;|

#### 示例代码：

```js
wx.qy.openThirdAppServiceChat({
    success: function(res) {
 // 回调
 }
});

```

**平台版本说明**

|平台|企业微信版本要求|
|--|--|
|ios|≥3.1.18|
|android|≥3.1.18|
|win|≥4.1.6|
|mac|≥4.1.6|

**应用权限说明**

|应用类型|可调用的应用|
|--|--|
|自建应用|不支持|
|第三方应用|提前配置了客服|
