政府单位可在自己的小程序配置「居民上报」小程序跳转入口，居民点击后进入「居民上报」小程序反馈问题。在微信小程序里可按以下参数调用小程序接口<a href="https://developers.weixin.qq.com/miniprogram/dev/api/navigate/wx.navigateToMiniProgram.html" target=_blank>wx.navigateToMiniProgram</a>，唤起「居民上报」小程序。

**请求参数**

| 参数 | 必须 | 说明 |
| ------------ | ---- | ---------------------------------------------------------------------------------------------------- |
| appId | 是 | 固定填企业微信居民上报小程序appid: wx85c9a2814145dbeb |
| path | 是 | 跳转到小程序的路径，路径格式固定为：pages/feedback/feedback?strcorpid=${strcorpid}&userid=${userid}|
| strcorpid | 是 | 企业corpid |
| userid | 是 | 企业成员的userid|

**调用示例**
```
wx.navigateToMiniProgram({
 appId: 'wx85c9a2814145dbeb',
 path: 'pages/feedback/feedback?strcorpid=${strcorpid}&userid=${userid}'
 success(res) {
 // 打开成功
 }
})
```
