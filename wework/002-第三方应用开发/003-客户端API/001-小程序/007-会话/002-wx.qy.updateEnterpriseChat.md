wx.qy.updateEnterpriseChat(Object object)
>此接口仅在企业微信3.0.36及以后版本支持。
>调用前提：必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态。
>当前成员必须在应用的可见范围，且必须在该群内。
>仅支持往群里添加企业内部成员。
>目前仅支持客户群调用，若不是客户群，将报错 unsupported chat.

变更群成员接口

**参数**
**Object object**

| 属性 | 类型 | 默认值 | 是否必填 | 说明 | 
| -------- | ------ | ---- | -------------|----- | 
| chatId | String | | 否 | 指定的群会话id | 
| userIdsToAdd | String | | 否 | 参与会话的企业成员列表，格式为userid1;userid2;...，用分号隔开 | 
| success | function | | 否 | 接口调用成功的回调函数 | 
| fail | function | | 否 | 接口调用失败的回调函数 | 
| complete | function | | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） | |

**示例代码**
```
wx.qy.updateEnterpriseChat({
 chatId: "CHATID", //通过企业微信创建群聊接口返回的chatId
 userIdsToAdd: 'zhangshan;lisi;wangwu', //参与会话的企业成员列表，格式为userid1;userid2;...，用分号隔开。
 success: (res) => {
 console.log(JSON.stringify(res))
 },
 fail: (res) => {
 console.log(JSON.stringify(res))
 }
})
```
