## 旧版jweixin调用
### 示例代码

```js
wx.invoke('openAppPurchase', {}, function(res){
 if(res.err_msg == "openAppPurchase:ok") {
 // 调用成功 
 }
 if(res.err_msg == "openAppPurchase:fail:no permission") {
 // 调用人身份不符合 
 }
 if(res.err_msg == "openAppPurchase:fail") {
 // 其它错误 
 } 
})
```
