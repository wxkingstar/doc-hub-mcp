## 旧版jweixin调用
### 示例代码
```javascript
wx.invoke('wedriveSelectFileForDownload', {
 selectedFileNum: 1
}, function (res) {
 if (res.err_msg == "wedriveSelectFileForDownload:ok") {
 var selectedTickets = res.result.selectedTickets;
 }
});
```
