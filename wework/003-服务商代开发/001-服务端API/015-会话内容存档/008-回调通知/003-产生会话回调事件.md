当企业下新产生消息时，企业微信服务器会向『会话存档接口授权』配置的“指令回调URL”推送相应的事件消息。消息接收方式参见[回调接口](#14946/#概述)。
>会话需在企业授权的范围内（企业可授权内部会话和外部会话）
>会话中至少有一个成员在企业授权范围内

**事件示例：**
```
<xml>
<SuiteId><
- ![CDATA[xxxx]]></SuiteId>
<AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
<InfoType><
- ![CDATA[conversation_new_message]]></InfoType>
<Token><
- ![CDATA[zzzzzzzzzzz]]></Token>
<TimeStamp>1403610513</TimeStamp>
</xml>
```

**参数说明：**
 | 参数 | 说明 |
 |---- | ----- |
|SuiteId |『会话存档接口授权』的应用ID |
|AuthCorpId |授权企业的CorpID |
|InfoType |事件类型，此时固定为conversation_new_message |
|Token|10分钟内有效，用于调用[获取会话记录](#42660)，不超过128字节|
|TimeStamp |时间戳 |
