服务商可以使用该接口查询某个客户或某段时间内申请的应用订单发票列表

__请求方式__:POST(__HTTPS__)

__请求地址:__ https://qyapi.weixin.qq.com/cgi-bin/paytool/get_invoice_list?provider_access_token=PROVIDER_ACCESS_TOKEN

__请求示例__

```json
{
  "start_time": 1680000000,
  "end_time": 1680003600,
  "cursor": "CURSOR",
  "limit": 50
}
```

__参数说明__:

|参数|必须|类型|说明|
|:-----------|:---|:-------|:------------|
| provider_access_token | 是 | string | [服务商调用接口凭证](#15143) |
| start_time | 否 |uint32|开始时间, 申请时间。可不填。但是不能单独指定该字段，start_time跟end_time必须同时指定。|
| end_time | 否 |uint32| 结束时间, 申请时间。可不填。但是不能单独指定该字段，start_time跟end_time必须同时指定。|
| cursor | 否 | string| 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填|
| limit | 否 | uint32 | 返回的最大记录数，整型，最大值100，默认值50 |

__权限说明__:
- 服务商需有在收银台完成商户号注册

__参数说明__:

|参数|类型|说明|
|:-----------|:-------------|:----------|
|errcode|int32|返回码|
|errmsg|string|错误码描述|
|has_more|uint32|是否有更多。 0-没有；1-有|
|next_cursor|string|分页游标，再下次请求时填写以获取之后分页的记录|
|invoice_list|obj[]|发票列表|
|invoice_list.order_id|string|申请开票的订单号|
|invoice_list.custom_corpid|string|订单对应的客户企业corpid|
|invoice_list.apply_time|uint64|客户申请开票的时间（UNIX时间戳）|
|invoice_list.invoice_type|uint32|企业客户申请的发票类型<br>0-普通发票<br>1-增值税专用发票|
|invoice_list.paid_price|uint64|实付金额，单位分|
|invoice_list.invoice_status|uint32|开票状态<br>0-开票中<br>1-已寄出<br>2-已发送<br>3-已取消|
|invoice_list.invoice_title|string|发票抬头|
|invoice_list.tax_number|string|纳税人识别号|
|invoice_list.send_way|uint32|发票收取方式<br>0-待定<br>1-快递<br>2-电子邮箱|
|invoice_list.contact_name|string|联系人姓名|
|invoice_list.contact_tel|string|联系电话|
|invoice_list.contact_addr|string|收件地址|
|invoice_list.contact_postcode|string|邮政编码|
|invoice_list.receive_email|string|电子邮箱|
|invoice_list.company_addr|string|公司地址|
|invoice_list.company_tel|string|公司电话|
|invoice_list.bank_name|string|开户行|
|invoice_list.bank_account_number|string|银行账号|
|invoice_list.invoice_note|string|备注信息|
