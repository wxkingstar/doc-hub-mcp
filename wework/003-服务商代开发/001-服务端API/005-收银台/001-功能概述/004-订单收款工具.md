# 一、概述
为帮助服务商更便捷的面向客户企业进行应用的销售和收款，企业微信为服务商提供订单收款工具。服务商可以提前为企业创建订单支付二维码/链接，客户企业打开后可立即购买支付。
# 二、创建订单
## 1. 创建入口
前往 “服务商助手 - 工具 - 收银台 “ 创建订单，创建成功后可获取订单支付二维码/链接。
- ![image (14).png](https://wework.qpic.cn/wwpic/803527_rH-6cZNoTMyLBNX_1669809366/0)
## 2. 选择应用
支持选择已经上线发布的第三方应用、代开发模板和行业方案，选择后即可完成创建订单。
- ![image (6).png](https://wework.qpic.cn/wwpic/863155_0Re65XKkT_yxGis_1669809373/0)
## 3. 更多配置
为满足更为个性化的订单创建需求，服务商可以开启“更多配置”，开启后可配置详细的订单参数，包括订单类型、购买人数、购买时长、额外优惠和指定企业等。详细配置说明如下：

| 配置项 | 说明 |
| ------ | ------ |
| 订单类型 | 新购、续期、扩容。其中续期和扩容面向已经购买的历史客户企业，新客户需选择“新购”类型 |
| 应用版本 | 第三方应用和行业方案需要选择版本，代开发应用无需配置版本 |
| 购买人数 | 按照客户企业实际购买人数填写 |
| 购买时长 | 按照客户企业实际购买时长填写，最长不可超过5年 |
| 生效方式 | 当订单类型为新购时，支持配置购买后生效的时间，包括“支付后立即生效”和“指定日期生效” |
| 额外优惠 | 对于第三方应用和代开发应用，服务商创建订单时可以在“原应付金额”的基础上额外配置优惠。最多减免金额为“原应付金额”的30%，最大折扣为“原应付金额”的7折 |
| 指定企业 | 可指定订单生效的客户企业ID，仅所指定企业的成员和管理员可以扫描该二维码或打开链接 |
- ![image (7).png](https://wework.qpic.cn/wwpic/616134_NvbWQzZYT5qJ2fv_1669809393/0)
## 4. 创建完成
订单创建完成后，服务商可获取订单的支付链接或二维码，转发给客户企业的管理员和成员打开后，即可购买应用并支付订单。
1) 支付链接和二维码的有效期为10天，过期后企业无法通过此二维码/链接进行支付。
2) 一个订单链接/二维码只能被一个企业扫码并下单。如有其他企业需要下单，服务商需重新创建订单并将支付二维码/链接提供给企业进行支付。
- ![image (8).png](https://wework.qpic.cn/wwpic/569752_F6njAvfWQQaUlem_1669809416/0)
## 5. 企业支付
客户企业的成员和管理员均可通过二维码/链接进行下单支付。其中，移动端支持“微信支付”、电脑端支持“微信支付和网银支付“。
根据服务商所配置的订单信息的差异，客户企业的支付流程表现如下：
1) **没有开启更多配置**
任意企业扫码后，需要自主配置应用版本、购买人数和购买时长后，下单支付。
- ![image (9).png](https://wework.qpic.cn/wwpic/279887_Fh_slKyYTOWIzYB_1669809437/0)
2) **开启更多配置，但未指定企业**
任意企业扫码后，均可查看服务商配置的订单信息，无需自主配置。如企业满足购买条件，可立即下单支付。
- ![image (10).png](https://wework.qpic.cn/wwpic/148556_L5wRsFBIQwKxs-I_1669809452/0)
3) **开启更多配置，且指定了企业**
仅所指定的企业扫码后，可查看服务商配置的订单信息，可立即下单支付。

注：支付链接和二维码的有效期为10天；若1个企业扫码并下单后，不支持其他企业再次打开该订单页，服务商需重新创建二维码。

4) **企业可通过微信支付/网银进行订单支付。**
- ![image (11).png](https://wework.qpic.cn/wwpic/513605_c6zAaft_StiwsCj_1669809499/0)<br>
## 6. 订单管理
企业成员可对订单进行开取发票和退款操作，其中订单管理的入口和范围区分管理员和普通成员：
- 管理员：订单管理入口为企业微信移动端-工作台-企业管理-应用管理、企业微信移动端-工作台-应用市场-订单中心、企业管理后台-应用管理，管理员可管理企业的所有订单；
- 普通成员：订单管理入口为企业微信移动端-工作台-应用市场-订单中心，普通成员仅可管理自己购买的订单。
