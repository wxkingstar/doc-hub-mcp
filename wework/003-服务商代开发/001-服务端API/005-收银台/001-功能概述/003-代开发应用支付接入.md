## 一、概述
企业微信向企业客户提供了线上的代开发应用购买流程，希望为企业提供安全、便捷、有保障的应用购买体验。
为此，服务商为企业提供代开发应用时，需要进行代开发应用定价策略的配置，便于企业客户通过企业微信购买应用。
本指引将介绍代开发应用接入支付，服务商配置和管理、企业购买和订单管理、应用到期拦截等内容。
## 二、代开发应用定价策略配置
- 服务商需要先完成商户号开通。
- 企业授权代开发应用后，才可进行企业代开发应用定价策略配置。
- 服务商前往 「桌面端服务商助手- 应用开发 - 应用代开发」中的代开发应用详情页，配置定价策略，配置后企业侧可按照定价策略购买应用。
- 企业授权后，服务商需为代开发应用配置**定价策略**后再提交上线，配置信息详细说明如下：
| 配置项 | 说明 |
| ------ | ------ |
| 试用时间 | 基础版本需设置应用的试用时间。客户的试用期从代开发应用上线时开始计算，试用过期后需购买代开发应用，否则使用应用时将被拦截 |
| 额外试用 | 可开启。当客户试用到期后，企业管理员可以点击额外试用 |
| 定价周期 | 可选年（365天） 或 月（31天 |
| 定价规则 | 两种类型可选<br>1. 固定单价（按人）：按照单人定价<br>2. 阶梯定价：每个人数阶梯内可单独设置“固定价格”或“固定单价” |
| 购买优惠 | 可开启。根据购买时长配置优惠策略，两种类型可选<br>1. 价格折扣：不可低于7折扣<br>2. 赠送时长：不可高于购买天数的35% |
| 人数限制 | 可开启。可设置“起售人数”和“最多购买”，当企业购买人数不符合人数限制的区间，不支持下单 |
| 总价限制 | 可开启。可设置应用的“最高价格”，当按照定价规则和优惠计算的价格高于最高价格，企业按照最高价格支付 |

## 三、客户企业购买
### 1. 企业购买 
**1) 代开应用授权和上线**
企业管理员授权代开发应用，且服务商完成代开发应用开发和定价策略配置，并提交代开发应用上线成功后，企业管理员才可进行代开发应用购买支付。
**2) 购买代开发应用**
企业管理员可以在 「企业管理后台 - 应用管理 - 应用详情页“」或「手机端工作台 - 管理企业 - 应用管理 -应用详情页」，配置 **购买人数** 和 **购买时长** 后，下单完成支付。
- ![image (1).png](https://wework.qpic.cn/wwpic/687570_c87N_hJKS7OhEOJ_1669796810/0)
**3) 变更付费方案**
购买代开发应用后，企业管理员可在 「企业管理后台 - 应用管理 - 应用详情页“」或「手机端工作台 - 管理企业 - 应用管理 -应用详情页」，变更付费方案，包括扩容可用人数、续期使用时间。
- ![image (2).png](https://wework.qpic.cn/wwpic/613187_grsxoFD3T_CTmq5_1669796860/0)
说明：扩容可用人数时，系统将自动对企业当前版本按剩余可用时间等比例进行价值折算，以抵扣部分付款金额。（例如：企业当前版本价格1000元，包含365天有效时间，当前剩余有效时间100天，则最高抵扣金额为：1000元x（100天/365天））
### 2. 企业支付
1) 移动端：支持微信支付
2) 电脑端：企业管理后台，支持微信支付+商企付（网银对公转账）
- ![image (18).png](https://wework.qpic.cn/wwpic/357620_gq56P0sOTAmni73_1669810855/0)
### 3. 企业订单管理
企业管理员可前往 「企业管理后台 - 应用管理 - 查看购买订单」 或 「移动端工作台 - 添加应用 - 右上角订单中心」对代开发应用订单进行管理。对于已经支付的订单，支持订单服务：
1) 开发票：可向服务商申请开具普通发票或增值税发票，发票由服务商开具。
2) 退款：支付后10天内且为开发票的订单可以申请退款，单个应用仅支持从该应用的最后一个订单开始申请退款。
3) 投诉服务商：若对服务商不满意，可向企业微信投诉服务商
- ![image (4).png](https://wework.qpic.cn/wwpic/215003_TEWVzezWRqi-Lvq_1669797130/0)
### 4. 应用到期拦截
- **试用到期**
企业授权的代开发应用上线后，可按照服务商设置的试用时间进行试用。应用试用到期后，企业需购买付费购买后方可继续使用。（试用期从代开发应用上线开始计算）
1) 试用版本到期前。企业微信将提前7天向企业管理员发送付费购买通知。
2) 试用版本到期后，当企业访问应用时将会被拦截，购买付费版本后可继续使用。
- ![image (12).png](https://wework.qpic.cn/wwpic/112308_VJap4LinTimt7xk_1669810867/0)
- **服务到期**
企业购买付费购买代开发应用后后，将按照企业所购买的时长生效服务。应用服务到期后，企业需续费后方可继续使用。
1) 服务到期前，企业微信将提前7天向企业管理员发送续费通知。
2) 服务到期后，当企业访问应用时将会被拦截，企业续费后可继续使用。
- ![image (13).png](https://wework.qpic.cn/wwpic/699448_jjbsA68ORBei1FU_1669810879/0)
