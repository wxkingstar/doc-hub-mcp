# 概述
服务商需要在企业微信平台，分别开通微信支付、商企付两个商户号，实现企业客户购买应用的收款。资金T+12（包括10天企业退款冻结期）即可提现到服务商银行账户。
收银台功能涉及到的支付服务，由财付通支付科技有限公司提供。
本文将介绍商户号开通和管理、提现、交易手续费相关内容。

2. **资料审核**
- ![image (4).png](https://wework.qpic.cn/wwpic/542646_BSvomvaoTZORGxP_1669804502/0)
- 提交开户资料申请后，微信支付平台将对开户资料进行审核，预计1-3个工作日。
- 若审核通过，将会通过 **「企业微信服务商助手」** 进行消息通知，可进入【桌面端服务商助手-收银台】，进行下一步“扫码签约”操作。

## 商户号管理
完成商户开通后，进入【桌面端服务商助手-工具-收银台-商户号管理】，可查看已开通的微信支付商户和商企付信息。
- ![image (14).png](https://wework.qpic.cn/wwpic/21641_XXZFYqFFTQ2ot-l_1669805689/0)
- 微信支付商户，可点击“修改银行账户”，进行银行账户修改
  - 修改信息并提交后，微信支付平台会进行校验，校验成功后银行账户信息会更新为最新信息；
  - 银行账户信息更新后，微信支付会自动汇款0.01元至新结算账户，验证是否可正常使用；
  - 系统汇款成功，该账户可正常发起提现；
  - 系统汇款失败，该账户无法发起提现，请检查修改；
- 商企付商户，暂不支持线上修改银行账户
  - 可点击“联系客服修改银行账户”，联系客服同学进行银行账户信息修改。
## 提现管理
- ![image.jpg](https://wework.qpic.cn/wwpic/785410_SPSCP_PZQ22P4iD_1669805748/0)
- 进入【桌面端服务商助手-收银台-提现】可查看商户号金额、发起提现、查看提现记录。
- 从客户支付完成到可提现
  - 企业客户订单支付完成后，次日将进入10天的冻结期。10天内企业客户可发起退款，冻结金额不可提现；
  - 经过10天冻结后，第11天企业微信会发起解冻，次日解冻金额进入服务商商户号。即企业客户支付完成到可提现周期为T+12；
  - 解冻金额为订单实付金额减去平台交易手续费后的金额，解冻金额可全部提现；
- 发起提现
  - 需要选择微信支付商户和商企付商户分别发起提现，资金可提现到对应的商户号绑定的对公银行账户；
  - 提现到账时间基本为T+1，具体以每个银行到账时间为准。
## 交易手续费
- 交易手续费为微信支付平台、商企付平台，针对商户号产生的收款交易收取的手续费。非企业微信收取。
- 商企付手续费
  - 商企付手续费暂未开始收取；
  - 后续收取将通过「企业微信服务商助手」另行通知；
- 微信支付手续费
  - 根据订单支付金额进行收取；
  - 交易手续费费率通常为：0.6%（千分之六），例如订单支付金额为1000元，则微信支付平台收取交易手续费6元，具体的费率服务商可登录微信支付商户平台查看 [费率查看帮助](https://kf.qq.com/faq/210817bUFRb2210817FfA3Yz.html) ；
  - 若产生订单退款，则微信支付交易手续费会原路退回；
  - 微信支付交易手续费发票由微信支付平台提供。微信支付商户的超级管理员（即企业法人）可登录[微信支付商户平台](http://pay.weixin.qq.com) ，按照指引开取手续费发票：[微信支付手续费发票指引](https://kf.qq.com/faq/170606N3YZ36170606IbUvui.html)。
## 常见问题
- **Q：开通商户号后，订单支付资金就会进入商户号吗？**
  A：开通商户号后，第三方应用、代开发应用配置了定价策略，企业通过企业微信进行应用购买支付或者服务商通过创建收款订单企业支付后，对应的订单支付资金会进入对应商户号，冻结期结束可提现。
（应用服务付费和代理商代下单付费，暂时不支持商户号收款，还是原流程不变）
- **Q：一个服务商可以申请几个商户号？**
  A：一个服务商在企业微信服务商助手，只能申请一个微信支付商户号、一个商企付商户号。
- **Q：开通商户、商户管理和提现，只能超管操作吗？**
  A：开通商户只能由超管进行操作。完成商户开通后，商户管理和提现的操作，默认只有超管可以操作，若超管给自定义分组设置了「收银台-商户管理」、「收银台-提现」的权限，则该自定义组成员也可以进行商户管理和提现操作。
- **Q：发起退款后交易手续费会退回吗？**
  A：订单发起退款后，交易手续费会合并至订单金额原路退回。
- **Q：商户号收款有限额吗？**
  A：服务商商户号收款没有限额。订单付款限额，微信支付根据不同银行支付有所不同，详见[微信支付银行卡交易限额](https://kf.qq.com/touch/faq/151210NZzmuY151210ZRj2y2.html)。商企付支付根据企业对公转账银行自身的支付限额。
