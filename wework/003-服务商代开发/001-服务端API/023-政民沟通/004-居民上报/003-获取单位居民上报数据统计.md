**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/resident/get_corp_status?access_token=ACCESS_TOKEN

**请求包体：**

```json
{
 "grid_id": "grid_id"
}

```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- | ---------------------------------------------------------------------------------------- |
| access_token | 是 | 调用接口凭证。“[获取access_token](https://work.weixin.qq.com/api/doc/90001/90143/90605)” |
| grid_id | 否 | 网格id，不传的话获取整个企业的概况 |

**权限说明：**

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「[居民上报 - 可调用接口的应用](#25813/配置可调用接口的应用)」中 |
|代开发应用|暂不支持| 
|第三方应用|暂不支持|
 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)

**返回结果：**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "processing": 1,
 "added_today": 1,
 "solved_today": 1,
 "pending": 10
 "total_case": 1,
 "total_accepted": 1,
 "total_solved": 1,
}
```

**参数说明：**

| 参数 | 说明 |
| -------------- | ---------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| processing | 办理中 |
| added_today | 今日上报 |
| solved_today | 今日办结 |
| pending | 待受理 |
| total_case | 累计上报 |
| total_accepted | 累计受理 |
| total_solved | 累计办结 |
