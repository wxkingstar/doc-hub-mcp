**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/patrol/get_order_info?access_token=ACCESS_TOKEN

**请求包体：**

```json
{
 "order_id": "order_id"
}

```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- | ---------------------------------------------------------------------------------------------------- |
| access_token | 是 | 调用接口凭证。获取方法查看“[获取access_token](https://work.weixin.qq.com/api/doc/90001/90143/90605)” |
| order_id | 是 | 工单id，支持通过工单id直接查询某工单，此参数不为空的话，其他参数无效 |

**权限说明：**

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「[巡查上报 - 可调用接口的应用](#25814/配置可调用接口的应用)」中 |
|代开发应用|暂不支持| 
|第三方应用|暂不支持|
 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)

**参数说明：**

| 参数 | 说明 |
| ------------------------------------ | ------------------------------------------------------------ |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| order_info.order_id | 工单id |
| order_info.desc | 事件描述 |
| order_info.urge_type | 紧急程度，1:一般，2:重要，3:紧急 |
| order_info.case_name | 事件类别名称 |
| order_info.grid_name | 所属网格名称 |
| order_info.grid_id | 所属网格id |
| order_info.create_time | 创建时间 |
| order_info.image_urls | 工单图片列表 |
| order_info.video_media_ids | 工单视频列表 |
| order_info.location | 发生地点 |
| order_info.location.name | 位置名 |
| order_info.location.address | 地址详情说明 |
| order_info.location.latitude | 纬度，浮点数，范围为90 ~ -90 |
| order_info.location.longitude | 经度，浮点数，范围为180 ~ -180 |
| order_info.processor_userids | 当前流程处理人列表 |
| order_info.process_list | 历史流程列表 |
| order_info.process_list.process_type | 流程类型,1:创建 2:受理 3:分配 4:转交 5:办结 6:拒绝 7:办理中 |
| order_info.process_list.solve_userid | 该流程的办结人 |
| order_info.process_list.process_desc | 处理流程详细描述 |
| order_info.process_list.status | 当前流程状态, 0:处理中，1:已处理 |
| order_info.process_list.solved_time | 处理时间 |
| order_info.process_list.image_urls | 流程图片列表 |
| order_info.process_list.video_media_ids | 流程视频列表 |
