可以查询目标网格的父网格及全部子网格ID列表

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/grid/list?access_token=ACCESS_TOKEN

**请求包体：**

```json
{
 "grid_id": "grid_id"
}

```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- | ---------------------------------------------------------------------------------------- |
| access_token | 是 | 调用接口凭证。“[获取access_token](https://work.weixin.qq.com/api/doc/90001/90143/90605)” |
| grid_id | 否 |网格id，不填则表示拉取根节点及其儿子节点|

**权限说明：**

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「[居民联系 - 可调用接口的应用](#30633/配置可调用接口的应用)」中 |
|代开发应用|暂不支持| 
|第三方应用|暂不支持|
 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)

**返回结果：**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "grid_list":[
 {
 "grid_id": "grid_id",
 "grid_name": 1,
 "grid_parent_id" : "grid_id",
     "grid_admin": ["zhangsan"],
     "grid_member": ["lisi", "11111"]
 }
 ]
}
```

**参数说明：**

| 参数 | 说明 |
| ------------------------ | ---------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| grid_list.grid_id | 网格id |
| grid_list.grid_name | 网格名称 |
| grid_list.grid_parent_id | 网格父节点的网格id |
| grid_list.grid_admin | 该节点的管理员列表 |
| grid_list.grid_member | 该节点的成员列表 |
