当企业下新产生消息时，企业微信专区后台会向具有企业会话内容权限的应用关联的程序推送相应的事件消息。
>会话需在企业会话内容授权的范围内（企业可授权内部会话和外部会话）
>会话中至少有一个成员在企业会话内容授权范围内

**事件示例：**
```
{
 "event_type": "conversation_new_message",
 "timestamp": 1408091189,
 "conversation_new_message": {
 "token": "zzzzzzzzzzz"
 }
}
```

**参数说明：**
 | 参数 | 说明 |
 |---- | ----- |
|event_type |事件类型，此时固定为conversation_new_message |
|timestamp |时间戳 |
|token|10分钟内有效，用于调用[获取会话记录](#53341)，不超过128字节|
