当企业授权知识集、取消授权知识集或授权的知识集发生变动时，企业微信服务器会向该知识集授权应用关联的程序推送消息。

##授权知识集
**事件示例：**
```
{
 "event_type": "auth_knowledge_base",
 "timestamp": 1408091189,
 "auth_knowledge_base": {
 "knowledge_base_id": "xxxxxxx",
 "knowledge_base_name": "zzzzz"
 }
}
```

**参数说明：**
 | 参数 | 说明 |
 |:--|
| event_type | 事件类型，此时固定为：```auth_knowledge_base``` |
|knowledge_base_id|知识集ID|
|knowledge_base_name|知识集名称|
|timestamp|授权时间时间戳|

## 內容学习完成(每个內容学习完成都会回调一次)
```
{
  "event_type": "knowledge_base_learn_done",
  "timestamp": 1408091189,
  "knowledge_base_learn_done": {
    "knowledge_base_id": "xxxxxxx",
    "knowledge_base_name": "zzzzz",
    "doc_id" : 1,
    "learn_status" : 0
  }
}
```

**参数说明：**
 | 参数 | 说明 |
 |:--|
| event_type | 事件类型，此时固定为：```knowledge_base_learn_done``` |
|timestamp|删除知识集时间戳|
|knowledge_base_id|知识集ID|
|knowledge_base_name|知识集名称|
|doc_id|內容ID|
|learn_status|学习状态。0-学习成功；1-学习失败|
