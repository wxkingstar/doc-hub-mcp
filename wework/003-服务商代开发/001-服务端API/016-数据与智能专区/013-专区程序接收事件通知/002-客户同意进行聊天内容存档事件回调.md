[配置了客户联系功能](#13473/开始开发)的成员添加外部联系人同意进行聊天内容存档时，回调该事件。
>回调的json包序列化为字符串传给注册的回调函数data字段

请求示例：

```
{
 "event_type": "chat_archive_audit_approved_single",
 "timestamp": 1408091189,
 "chat_archive_audit_approved": {
 "userid": "xxxxxxx",
 "external_userid": "zzzzz",
        "chatid":"xxx"
 }
}
```
参数说明：

 | 参数 | 说明 |
 |----|---- |
  |event_type |事件类型<br>当客户在单聊中同意存档时，该字段值为chat_archive_audit_approved_single<br>当客户在群聊中同意存档时，该字段值为chat_archive_audit_approved_room |
  |timestamp |时间戳 |
 |userid|单聊时返回，企业服务人员的UserID|
 |external_userid|外部联系人的userid，注意不是企业成员的账号 |
 |chatid|群聊时返回，所在群聊的ChatId|
