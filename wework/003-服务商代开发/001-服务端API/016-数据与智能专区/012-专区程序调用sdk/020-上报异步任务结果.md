可以通过这个接口上报异步任务的结果。

**请求方法**
通过SDK调用，具体方式参考[专区程序使用指引](#54006)。

**请求接口名：**
program_async_job_call_back

```
{
  "errcode":0,
  "result":"aaaabbbccc"
}
```
**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- | ---------------------------------------------------------------------------------------------------- |
|errcode| 是| 错误码，为0表示任务执行成功|
|result|否|任务结果。返回的结果必须符合模型或程序的输出协议。|

**返回结果**
```
{
 "errcode": 0,
 "errmsg": "ok"
}
```

**返回字段说明：**

| 字段名 | 说明 |
| ------------ | ------------ |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
