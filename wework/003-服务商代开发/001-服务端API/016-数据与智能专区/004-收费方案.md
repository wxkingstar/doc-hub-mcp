数据与智能专区会话数据调用接口为增值接口，面向服务商单独定价收费，服务商可前往「桌面端服务商助手-工具-通用配置-购买高级接口」或「服务商后台-应用管理-购买高级接口」进行购买和使用。

[[toc]]
### 1. 接口介绍

按照接口能力分为内外部会话、内外部会话及语音通话：

| 版本 | 说明 |
| :---: | :---: |
| **内外部会话** | 用于企业内外部服务沟通质量管理 |
| **内外部会话及语音通话** | 用于企业内外部服务沟通质量管理（含语音通话） |
- 服务商购买数据专区会话数据第三方接口时需指定企业，购买的接口仅可在指定的企业下使用。
- 一个接口账号可用于调用一个员工的数据专区会话数据第三方接口，并使用展示组件展示对应的会话内容。

[[toc]]
### 2. 接口试用
服务商应用获得企业授权数据与智能专区权限后，自授权日起，服务商应用可在该企业下免费使用接口 30 天，免费到期后需购买接口使用。

[[toc]]
### 3. 会话内容存储时间
企业开启数据专区会话内容，并授权服务商应用数据与智能专区权限后，服务商应用可调用[设置公钥接口](/document/path/100016)设置会话数据的加密公钥，设置后企业微信将对授权范围内成员的会话内容加密存储，服务商可使用展示组件展示会话内容。
- **企业可设置存储最近 90 天/180 天/1 年/2 年的会话内容**，企业设置后，在数据专区会话数据第三方接口付费有效期内，存储企业自授权日起所设置存储时长的会话内容，服务商可使用展示组件展示对应的会话内容；过期的会话内容不存储，展示组件不展示。
- 接口付费到期后，超过 30 天未续期，则之前付费期间存储的会话内容将不再存储，展示组件不展示。
- 试用到期后，超过 30 天未付费，则之前试用期间存储的会话内容将不再存储，展示组件不展示。

[[toc]]
### 4. 客户管理
服务商可前往「桌面端服务商助手-工具-通用配置-高级接口管理」或「服务商后台-应用管理-高级接口管理」查看和管理数据专区会话数据第三方接口的企业客户，企业客户包括：试用中、试用到期、付费中、付费到期的企业客户。

### 5. 其他
- 退款：数据专区会话数据第三方接口支付后十天内支持申请退款，申请通过后将原路退款，超过十天后不支持申请退款。
- 发票：数据专区会话数据第三方接口按月开取发票，服务商可在每月的十号开始申请上一个月对应订单的发票。
