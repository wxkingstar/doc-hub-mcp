当服务商购买高级接口账号并完成付款后，企业微信后台会推送“支付成功通知”到服务商的**系统事件接收URL**（应用管理 -通用开发参数-系统事件接收URL）。

**事件示例：**
```
<xml>
    <ServiceCorpId><
- ![CDATA[xxxx]]></ServiceCorpId>
    <InfoType><
- ![CDATA[advanced_api_pay_success]]></InfoType>
    <AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
    <AdvanceApiType><
- ![CDATA[1]]></AdvanceApiType>
    <OrderId><
- ![CDATA[zzzz]]></OrderId>
    <BuyerUserId><
- ![CDATA[USERID]]</BuyerUserId>
    <TimeStamp>1403610513</TimeStamp>
</xml>
```

**参数说明：**
 | 参数 | 说明 |
 |---- | ----- |
 | ServiceCorpId | 服务商CorpID|
 | InfoType | 事件类型，此时固定为advanced_api_pay_success|
 | AuthCorpId | 客户企业CorpID |
 |AdvanceApiType|高级接口类型，1-会话内容数据接口|
 | OrderId | 订单号 |
 | BuyerUserId | 服务商内下单用户UserID|
 | TimeStamp | 时间戳 |
