当服务商购买的高级接口账号过期时，企业微信后台会推送“接口购买到期通知”到服务商的**系统事件接收URL**（应用管理 -通用开发参数-系统事件接收URL）。

**事件示例：**
```
<xml>
    <ServiceCorpId><
- ![CDATA[xxxx]]></ServiceCorpId>
    <InfoType><
- ![CDATA[advanced_api_expired]]></InfoType>
    <AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
    <AdvanceApiType><
- ![CDATA[1]]></AdvanceApiType>
    <ChatArchiveApi>
      <Edition><
- ![CDATA[2]]></Edition>
    </ChatArchiveApi>
    <TimeStamp>1403610513</TimeStamp>
</xml>
```

**参数说明：**
 | 参数 | 说明 |
 |---- | ----- |
 | ServiceCorpId | 服务商CorpID|
 | InfoType | 事件类型，此时固定为advanced_api_expired|
 | AuthCorpId | 客户企业CorpID |
 |AdvanceApiType|高级接口类型，1-会话内容数据接口|
 |ChatArchiveApi|会话内容数据接口相关信息，详见`ChatArchiveApi`|
 | TimeStamp | 时间戳 |
 
 **ChatArchiveApi说明**
 |参数|说明|
 |-----|------|
 | Edition | 到期的账号版本。2-内外部会话，3-内外部会话及语音通话|
