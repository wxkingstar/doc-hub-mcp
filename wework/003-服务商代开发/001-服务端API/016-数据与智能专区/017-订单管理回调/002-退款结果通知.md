当服务商提交退款申请的订单发生状态变更时，企业微信后台会推送“退款结果通知”到服务商的**系统事件接收URL**（应用管理 -通用开发参数-系统事件接收URL）。

**事件示例：**
```
<xml>
    <ServiceCorpId><
- ![CDATA[xxxx]]></ServiceCorpId>
    <InfoType><
- ![CDATA[advanced_api_refund]]></InfoType>
    <AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
    <AdvanceApiType><
- ![CDATA[1]]></AdvanceApiType>
    <OrderId><
- ![CDATA[zzzz]]></OrderId>
    <OrderStatus>1</OrderStatus>
    <TimeStamp>1403610513</TimeStamp>
</xml>
```

**参数说明：**
 | 参数 | 说明 |
 |---- | ----- |
 | ServiceCorpId | 服务商CorpID|
 | InfoType | 事件类型，此时固定为advanced_api_refund|
 | AuthCorpId | 客户企业CorpID |
 |AdvanceApiType|高级接口类型，1-会话内容数据接口|
 | OrderId | 订单号 |
 | OrderStatus | 订单状态，1:退款成功，2:退款被拒绝。|
 | TimeStamp | 时间戳 |
