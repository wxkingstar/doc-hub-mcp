当服务商购买接口调用许可账号并完成付款后，企业微信后台会推送“支付成功通知”到服务商的**系统事件接收URL**（应用管理 -通用开发参数-系统事件接收URL）。
>支付完成后会生成账号码，账号码生成完毕后才会推送该通知。若购买较多数量账号码，则会导致该通知产生一定的延迟。

**事件示例：**
```
<xml>
    <ServiceCorpId><
- ![CDATA[xxxx]]></ServiceCorpId>
    <InfoType><
- ![CDATA[license_pay_success]]></InfoType>
    <AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
    <OrderId><
- ![CDATA[zzzz]]></OrderId>
    <BuyerUserId><
- ![CDATA[USERID]]</BuyerUserId>
    <TimeStamp>1403610513</TimeStamp>
</xml>
```

**参数说明：**
 | 参数 | 说明 |
 |---- | ----- |
 | ServiceCorpId | 服务商CorpID|
 | InfoType | 事件类型，此时固定为license_pay_success|
 | AuthCorpId | 客户企业CorpID |
 | OrderId | 订单号。如果为多企业新购订单，该值为子订单号 |
 | BuyerUserId | 服务商内下单用户UserID|
 | TimeStamp | 时间戳 |
