该接口用于获取空间信息。包括：空间成员及权限及安全设置。

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/wedrive/new_space_info?access_token=ACCESS_TOKEN 

**请求包体**
```json
{
 "spaceid": "SPACEID"
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
| ------- | ------ | -------- | ------------ |
| spaceid | string | 是 | 空间spaceid |

**权限说明**
- 自建应用需配置到“[可调用应用](#23784)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有“微盘”权限
- 代开发自建应用需具有“微盘”权限
