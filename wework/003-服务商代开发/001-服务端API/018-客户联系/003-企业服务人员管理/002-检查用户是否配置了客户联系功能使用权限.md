企业和第三方服务商可通过此接口检查部门或成员是否具备客户联系权限。

**请求方式：**POST（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/externalcontact/check_follow_user?access_token=ACCESS_TOKEN

**请求示例：**

```
{
 "userid":["zhangsan","lissi"],
 "partyid":[1]
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| :----------- | :--- | :----------------- |
| access_token | 是 | 调用接口凭证 |
| userid |否|用户的userid |
|partyid|否|部门id|

**权限说明：**
- 企业需要使用配置到“[可调用应用](#13473/开始开发)”列表中的自建应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）；
- 第三方应用需具有“企业客户权限->客户基础信息”权限

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "authorized":
 {
     "userid":["lisi"]
 },
 "unauthorized":
 {
    "userid":["zhangsan"],
    "partyid":[1]
 }
}
```
**参数说明：**

| 参数 | 说明 |
| :------- | :---------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|authorized.userid|有客户联系使用权限的用户id|
|authorized.partyid|有客户联系使用权限的部门id|
|unauthorized.userid|无客户联系使用权限的用户id|
|unauthorized.partyid|无客户联系使用权限的部门id|
