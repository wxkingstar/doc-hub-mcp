**接口说明**：报销方在获得用户选择的电子发票标识参数后，可以通过该接口查询电子发票的结构化信息，并获取发票PDF文件。

**请求方式：**POST（**HTTPS**）
**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/card/invoice/reimburse/getinvoiceinfo?access_token=ACCESS_TOKEN

**请求包体：**

```
{
 "card_id":"CARDID" ,
 "encrypt_code":"ENCRYPTCODE"
}

```
**参数说明：**

|参数|必须|说明|
|----|----|----|
|access_token|是|调用接口凭证|
|card_id|是|发票id|
|encrypt_code|是|加密code|

**权限说明：**

仅认证的企业微信账号有接口权限

**返回数据：**

```
{
 "errcode":0,
 "errmsg":"ok",
 "card_id":"CARDID",
 "begin_time":1469084420,
 "end_time":2100236420,
 "openid":"oxfdsfsafaf",
 "type":"增值税电子普通发票",
 "payee":"深圳市XX有限公司",
 "detail":"可在公司企业微信内报销使用",
 "user_info":{
 "fee":4000,
 "title":"XX有限公司",
 "billing_time":1478620800,
 "billing_no":"00000001",
 "billing_code":"000000000001",
 "info":[
 {
    "name":"NAME",
    "num":10,
    "unit":"吨",
    "fee":10,
    "price":4
   }
  ],
  "fee_without_tax":2345,
  "tax":123,
  "detail":"项目",
  "pdf_url":"pdf_url",
  "reimburse_status":"INVOICE_REIMBURSE_INIT",
  "check_code": "check_code"
  }
}

```
**参数说明：**

|参数|说明|是否必返回|
|----|----|----|
|card_id|发票id|是|
|begin_time|发票的有效期起始时间|是|
|end_time|发票的有效期截止时间|是|
|openid|用户标识|是|
|type|发票类型，如广东增值税普通发票|是|
|payee|发票的收款方|是|
|detail|发票详情|是|
|user_info|发票的用户信息，见user_info结构说明|是|
 user_info包含以下信息：
 
|参数|说明|是否必返回|
|----|----|----|
|fee|发票加税合计金额，以分为单位|是|
|title|发票的抬头|是|
|billing_time|开票时间，为十位时间戳|是|
|billing_no|发票代码|是，全电发票发票号码是该字段|
|billing_code|发票号码|是（全电发票发票号码为billing_no字段）|
|tax|税额,以分为单位|是|
|fee_without_tax|不含税金额，以分为单位|是|
|detail|发票详情，一般描述的是发票的使用说明|是|
|pdf_url|这张发票对应的PDF_URL|是|
|trip_pdf_url|其它消费凭证附件对应的URL，如行程单、水单等|否|
|check_code|校验码|是|
|buyer_number|购买方纳税人识别号|否|
|buyer_address_and_phone|购买方地址、电话|否|
|buyer_bank_account|购买方开户行及账号|否|
|seller_number|销售方纳税人识别号|否|
|seller_address_and_phone|销售方地址、电话|否|
|seller_bank_account|销售方开户行及账号|否|
|remarks|备注|否|
|cashier|收款人，发票左下角处|否|
|maker|开票人，发票右下角处|否|
|reimburse_status|发报销状态INVOICE_REIMBURSE_INIT：发票初始状态，未锁定；INVOICE_REIMBURSE_LOCK：发票已锁定；INVOICE_REIMBURSE_CLOSURE：发票已核销|是|
|info|商品信息结构，见下方说明|否|
 info包含以下信息：
 
|参数|说明|是否必返回|
|----|----|----|
|name|项目（商品）名称|是|
|num|项目数量|否|
|unit|项目单位|否|
|price|单价，以分为单位|是|
