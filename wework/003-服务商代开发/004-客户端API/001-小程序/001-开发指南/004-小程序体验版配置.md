**当[小程序关联到企业](#19704)后**，可在未发布小程序的情况下，配置体验版进行体验，以方便进行开发

**配置体验版流程**

1. 开发者在微信开发者工具生成二维码
- ![](https://wework.qpic.cn/wwpic/41046_IFMAo4LgRMOZbJA_1584955028/0)

2. 开发者使用企业微信扫描二维码打开小程序开发版

3. 点击进入右上角"..."--》点击配置体验版
- ![](http://p.qpic.cn/pic_wework/808606030/22247a3267c4a2e5421a5b3031890c5efcad5ec8d12ddb78/0)

4. 进入页面选择体验成员后，发送体验邀请（被邀请的成员需在小程序可见范围内，可在管理端-应用与小程序-小程序详情页进行配置）
- ![](http://p.qpic.cn/pic_wework/985556960/71772961c95d08559c12ebe6077c5256e476b16fd3a534c5/0)

5. 成员将收到小程序通知，通过点击小程序通知卡片即可使用该小程序体验版
- ![](http://p.qpic.cn/pic_wework/16809907/a6a0feb66b86c133b3652dfa37a1dcd977513c8619108b9a/0)
