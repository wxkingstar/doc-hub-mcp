**onUserOpStatistic(Object object)**
监听用户某些操作触发的事件，用于统计用途，目前仅支持转发事件。
>**调用前提：必须先调用过wx.qy.login，且session_key未过期**，开发者可调用checkSession 检查当前登录态
>**当前成员必须在应用的可见范围**，<font color='red'>如果成员不在应用的可见范围，转发时不会触发回调</font>

**OBJECT参数说明**

| 参数名 | 类型 | 说明 |
| ---------- | -------- | ---- | ---------------------------------------- |
| op | String | 用户操作类型，目前仅有share，表示转发分享。 |
| path | String | 转发的小程序路径 |

**示例代码：**
```
page{
  onUserOpStatistic: function(e) {
    if(e.op == 'share') {
      var path = e.path;
    }
  }
}
```
