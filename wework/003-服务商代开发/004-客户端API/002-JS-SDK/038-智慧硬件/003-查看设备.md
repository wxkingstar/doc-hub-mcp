## 旧版jweixin调用
### 示例代码
```
wx.invoke("openDeviceProfile", {
 "deviceSn": "QYWX001", // 设备SN
},function(res){
 if (res.err_msg == "openDeviceProfile:ok")
 {
 // 打开设备详情页成功
 }
});
```
