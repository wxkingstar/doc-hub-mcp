## 旧版jweixin调用
### 示例代码
```
wx.invoke('saveApprovalSelectedItems', {
 "key": 'key', // 字符串，从 URL 中获取到的 key
 "selectedData": '[{"key":"item-1","value":"选项1"},{"key":"item-2","value":"选项2"}]'// ！！！注意，旧版这里传入的是，JSON.stringify后的字符串
}, (res) => {
 if (res.err_msg === 'saveApprovalSelectedItems:ok') {
 // 保存成功
 }
});
```
