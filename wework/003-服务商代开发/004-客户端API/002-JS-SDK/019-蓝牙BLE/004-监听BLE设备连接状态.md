## 旧版jweixin调用
### 示例代码
```javascript
wx.onBLEConnectionStateChange(function(res) {
 // 该方法回调中可以用于处理连接意外断开等异常情况
 console.log(`device ${res.deviceId} state has changed, connected: ${res.connected}`)
})
```
