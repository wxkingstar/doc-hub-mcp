# UserInfo
用户信息

名称 | 数据类型 | 是否必填 | 描述
--- | --- | --- | ---
userInfo | object | 是 | /
∟ nickName | string | 是 | 用户昵称
∟ avatarUrl | string | 是 | 用户头像
∟ gender | string | 是 | 用户性别，可选值：  
- 未知 - ''   
- 男性 - male   
- 女性 - female
∟ country | string | 是 | 用户所在国家或地区
∟ city | string | 是 | 用户所在城市
∟ language | string | 是 | 所在地区所用的语言，可选值：  
- 英文 - en_US  
- 中文 - zh_CN
