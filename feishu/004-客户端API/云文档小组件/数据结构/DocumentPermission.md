# DocumentPermission
用户对文档的权限
| **名称**      | **数据类型** | **是否必填** | **描述** |
| ----------- | -------- | -------- | ------ |
| editable    | boolean  | 是        | 是否可编辑  |
| copyable    | boolean  | 是        | 是否可复制  |
| commentable | boolean  | 是        | 是否可评论
| manageable | boolean  | 是        | 是否有管理权限
