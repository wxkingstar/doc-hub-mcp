# ImageBlockData
图片 Block 的数据结构，继承于 [BlockData](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/docs-add-on/05-api-doc/BlockData/blockdata)

| **名称** | **数据类型** | **是否必填** | **描述**     |
| ------ | -------- | -------- | ---------- |
| token  | string   | 是        | 图片 Token   |
| width  | number   | 是        | 图片宽度，单位 px |
| height | number   | 是        | 图片高度，单位 px
