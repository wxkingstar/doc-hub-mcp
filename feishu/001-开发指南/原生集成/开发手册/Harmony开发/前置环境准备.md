# 前置环境准备

## 开发环境配置

### Command Line Tools

- 版本要求：Command Line Tools for HarmonyOS 5.0.5 Release（2025/08/19更新或最新版本）

### 环境变量配置

- 设置环境变量 ALCHEMY_HARMONY_SDK_HOME 为 鸿蒙SDK的目录路径
- 确保安装 Nodejs, 在终端可以调用 node -v 命令，正确输出结果。nodejs版本号 和 鸿蒙开发要求一致
- 确保安装**JDK11或 17最新版本**，在终端可以调用 java --version, 能正确输出结果
- 在Path中配置鸿蒙commandlineTool中的ohpm路径，确保在终端可以直接调用 ohpm -v 能输出结果
- 在Path中配置鸿蒙commandlineTool中的hdc路径，确保在终端可以直接调用 hdc -v 能输出结果，**确保 hdc版本为 3.1.0。**
- 在Path中配置鸿蒙 commandlineTool中的hvigorw路径，确保在终端可以直接调用hvigorw -v 能输出结果

## 开发规范
- 遵循Harmony对应的官方代码风格、命名格式
- 资源命名添加服务商、客户相关前缀标识，防止和飞书资源冲突
- 类包名使用服务商、客户相关域名，避免使用飞书 com.ss.android.lark包名
- 使用飞书标准日志能力，在相关接口入口、出口位置打日志标记
- 三方依赖库需要对齐飞书版本，开发过程通过集成工具的【冲突检测】功能提前进行检测
