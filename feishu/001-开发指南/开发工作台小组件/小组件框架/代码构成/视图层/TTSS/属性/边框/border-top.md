# border-top

## 介绍

用于设置上边框的样式，该属性可按顺序设置`border-top-width`、`border-top-style`、`border-top-color`。

## 语法

```css
/* 上边框是粗双线红边框 */
border-top: thick double red;
/* 上边框是细点线蓝边框 */
border-top: thin dotted blue;
/* 上边框是10px虚线橙边框 */
border-top: 10px dashed orange;
```

## 标准化语法

```css
border-top: <line-width> || <line-style> || <color>
```
