# transition-delay

## 介绍

用于设置过渡动画延迟执行的时间。

## 语法

```css
/* time */
transition-delay: 100ms;
```

### 取值

-   `<time>`

定义动画的延迟执行的时间，默认为`0`。

## 标准化语法

```css
transition-delay: <time>
```
