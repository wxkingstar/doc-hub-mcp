# 步骤五：调试小程序


在小程序开发完成后，你可以先将小程序发布到测试版应用中进行测试。
## 步骤一：真机调试
:::html
<md-td>
1. 打开移动端飞书，切换到测试企业。

2. 打开飞书开发者工具，然后在左上角单击**真机调试**，然后使用移动端飞书扫描弹出的二维码。

	手机需要与开发者工具处于同一网络才可以进行真机调试。

 	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/529b5ff758304cafa6cfd06b278355b6~tplv-goo7wpa0wc-image.image?lazyload=true&width=300&height=464" style="width:40%"/>

	如下图所示，移动端小程序中可以展示个人信息并设置个人签名。
 
 	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/5fa2cd53b6b14b79b549d84e40a1a813~tplv-goo7wpa0wc-image.image?lazyload=true&width=720&height=1556" style="width:40%"/>
</md-td>
:::
## 步骤二：发布测试版小程序
:::html
<md-td>
1. 打开小程序开发工具。

2. 单击右上角 **上传**，然后在弹出的页面中选择 **平台** 和 **版本号**，最后单击 **上传** 按钮完成上传。
 	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/0849868c7fe542fbaa09f6d14b623913~tplv-goo7wpa0wc-image.image?lazyload=true&width=2556&height=1960" style="width:70%"/>

3. 登录[开发者后台](https://open.feishu.cn/app)，进入测试版应用详情页，单击 **小程序** 进入小程序页面。

	如下图所示，小程序版本包已成功上传。

 	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/3007cf4082b247d9a02c42c78f29a9fb~tplv-goo7wpa0wc-image.image?lazyload=true&width=1283&height=729" style="width:70%"/>

4. 选择小程序 **版本号** 和 **最低兼容飞书版本** ，然后单击 **保存**。

	测试版应用无需发布即可直接在测试企业的工作台进行测试和验证。


 	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/ae54633b76264936b1611583034d860f~tplv-goo7wpa0wc-image.image?lazyload=true&width=1283&height=670" style="width:70%"/>
    
</md-td>
:::
