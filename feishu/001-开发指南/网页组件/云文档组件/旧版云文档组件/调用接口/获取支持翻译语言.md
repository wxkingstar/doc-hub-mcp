# 获取支持翻译语言
## 示例
```js
myComponent.invoke.getSupportedTranslateLanguages().then(function(response) {
  const { code, msg, data } = response;
  // ...
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|code|	Number |	0：成功 、 -1：失败 |
|msg|	String |	信息 |
|data|Array|支持翻译语言列表|
