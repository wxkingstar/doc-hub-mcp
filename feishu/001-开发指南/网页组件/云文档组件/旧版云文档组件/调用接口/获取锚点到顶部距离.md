# 获取锚点到顶部距离
## 示例
```js
myComponent.invoke.getAnchorTop('anchor-abc').then(function(response) {
  const { code, msg, data } = response;
  // ...
});
```

## 参数
|属性|	类型|	是否必须	|说明|
| ---|----- | -------|------ | 
|anchor|	String	|是|	锚点|

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|code|	Number |	0：成功 、 -1：失败 |
|msg|	String |	信息 |
|data|Number | 距离文档顶部距离 |
