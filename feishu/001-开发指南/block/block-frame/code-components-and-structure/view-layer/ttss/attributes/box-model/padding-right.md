# padding-right

## 介绍

指定一个元素内容与右边界的距离，该属性不能为负值。

## 语法

```css
/* <length> values */

padding-right: 0.5em;

padding-right: 0;

padding-right: 2px;

/* <percentage> value */

padding-right: 10%;
```

### 取值

-   `<length>`

以固定值为内边距。

-   `<percentage>`

相对于包含块的宽度，以百分比值为内边距。

### 标准化语法

```css
padding-right: <length> | <percentage>
```
