# animation-play-state

## 介绍

用于设置播放动画的状态。

## 语法

```css
animation-play-state: running;

animation-play-state: paused;
```

### 取值

-   `running`

**默认值。** 当前动画正在运行。

-   `paused`

当前动画已被停止。

### 标准化语法

```css
animation-play-state: running | paused
```
