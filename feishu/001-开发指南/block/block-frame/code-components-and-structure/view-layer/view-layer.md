# 视图层

框架的视图层由 TTML 与 TTSS 编写，由基础组件来进行展示。将逻辑层的数据反映成视图，同时将视图层的事件发送给逻辑层。
- TTML（TT Markup language）用于描述页面的结构。

- TTSS（TT Style Sheet）用于描述页面的样式。
