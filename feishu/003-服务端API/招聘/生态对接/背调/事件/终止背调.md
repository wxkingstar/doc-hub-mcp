# 终止背调

飞书招聘客户在招聘系统内终止背调后，系统会推送「终止背调」事件给对应的背调服务商，服务商可根据此事件获取背调 ID，完成服务商内部的订单取消等后续操作。

## 事件

基本 | 
--- | ---
事件类型 | hire.eco_background_check.canceled_v1
支持的应用类型 | 自建应用、商店应用
权限要求  
            **订阅该事件所需的权限，开启其中任意一项权限即可订阅** | 获取招聘背调信息(hire:background_check_order:readonly)
推送方式 | [Webhook](https://open.feishu.cn/document/ukTMukTMukTM/uUTNz4SN1MjL1UzM)

### 事件体

名称 | 类型 | 描述
--- | --- | ---
schema | string | 事件模式
header | event_header | 事件头
event_id | string | 事件 ID
event_type | string | 事件类型
create_time | string | 事件创建时间戳（单位：毫秒）
token | string | 事件 Token
app_id | string | 应用 ID
tenant_key | string | 租户 Key
event | \- | \-
background_check_id | string | 背调 ID，招聘系统内唯一，可通过[创建背调](https://open.feishu.cn/document/ukTMukTMukTM/uMzM1YjLzMTN24yMzUjN/hire-v1/eco_background_check/events/created)事件获取
termination_reason | string | 终止原因

```json
        "app_id": "cli_9f5343c580712544",

    },
    "event": {
        "background_check_id": "6931286400470354183",
        "termination_reason": "主动拒绝此次背调"
    }
}
```

import (
	"context"
	"fmt"

larkcore "github.com/larksuite/oapi-sdk-go/v3/core"
	larkevent "github.com/larksuite/oapi-sdk-go/v3/event"
	"github.com/larksuite/oapi-sdk-go/v3/event/dispatcher"

// SDK 使用说明 SDK user guide：https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/server-side-sdk/golang-sdk-guide/preparations
func main() {
	// 注册事件 Register event
	eventHandler := dispatcher.NewEventDispatcher("", "").
		OnP2EcoBackgroundCheckCanceledV1(func(ctx context.Context, event *larkhire.P2EcoBackgroundCheckCanceledV1) error {
			fmt.Printf("[ OnP2EcoBackgroundCheckCanceledV1 access ], data: %s\n", larkcore.Prettify(event))
			return nil
		})

// 构建 client Build client
	cli := larkws.NewClient("YOUR_APP_ID", "YOUR_APP_SECRET",
		larkws.WithEventHandler(eventHandler),
		larkws.WithLogLevel(larkcore.LogLevelDebug),
	)

// 建立长连接 Establish persistent connection
	err := cli.Start(context.Background())

if err != nil {
		panic(err)
	}
}
