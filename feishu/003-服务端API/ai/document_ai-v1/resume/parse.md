# 识别文件中的简历信息

简历信息解析接口，支持PDF/DOCX/PNG/JPG四种文件类型的一次性的识别。文件大小需要小于30M。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=document_ai&version=v1&resource=resume&method=parse)

<md-alert type="tip">
