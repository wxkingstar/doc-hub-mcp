# 识别图片中的文字

可识别图片中的文字，按图片中的区域划分，分段返回文本列表。文件大小需小于5M。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=optical_char_recognition&version=v1&resource=image&method=basic_recognize)

## 注意事项

- 单租户限流为 20QPS，即同租户下的应用共享本租户的 20 QPS 限流。
- 该接口不支持通过飞书个人版调试。
