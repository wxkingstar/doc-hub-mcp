# 列取附件

列取一个资源的所有附件。返回的附件列表支持分页，按照附件上传时间排序。

每个附件会返回一个可供下载的临时url，有效期为3分钟，最多可以支持3次下载。如果超过使用限制，需要通过本接口获取新的临时url。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=attachment&method=list)

:::html
<md-alert type="tip">
获取任务的附件列表，需要该任务的读取权限。
</md-alert>
