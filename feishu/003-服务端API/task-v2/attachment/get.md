# 获取附件

提供一个附件GUID，返回附件的详细信息，包括GUID，名称，大小，上传时间，临时可下载链接等。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=attachment&method=get)

:::html
<md-alert type="tip">
获取附件需要附件归属资源的可读取权限。
</md-alert>
:::
