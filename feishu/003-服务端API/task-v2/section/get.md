# 获取自定义分组详情

获取一个自定义分组详情，包括名称，创建人等信息。如果该自定义分组归属于一个清单，还会返回清单的摘要信息。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=section&method=get)

:::html
<md-alert type="tip">
需要清单归属资源的读取权限。
</md-alert>
:::
