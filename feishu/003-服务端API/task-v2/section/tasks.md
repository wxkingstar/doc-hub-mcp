# 列取自定义分组中的任务

列取一个自定义分组里的所有任务。支持分页。任务按照自定义排序的顺序返回。本接口支持简单的过滤。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=section&method=tasks)

:::html
<md-alert type="tip">
需要自定义分组所在资源的读取权限。
</md-alert>
:::
