# 获取自定义分组列表

获取一个资源下所有的自定义分组列表。支持分页。返回结果按照自定义分组在界面上的顺序排序。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=section&method=list)

:::html
<md-alert type="tip">
获取自定义分组列表需要资源的读取权限。
</md-alert>
:::
