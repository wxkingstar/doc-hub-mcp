# 将自定义字段加入资源

将自定义字段加入一个资源。目前资源类型支持清单tasklist。一个自定义字段可以加入多个清单中。加入后，该清单可以展示任务的该字段的值，同时基于该字段实现筛选，分组等功能。

如果自定义字段的设置被更新，字段加入的所有资源都能收到这个更新，并进行相应的展示。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=custom_field&method=add)

:::html
<md-alert type="tip">
将自定义字段加入一个资源需要该字段和资源的可编辑权限。
</md-alert>
