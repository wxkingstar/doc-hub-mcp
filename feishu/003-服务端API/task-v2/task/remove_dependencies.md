# 移除依赖

从一个任务移除一个或者多个依赖。移除时只需要输入要移除的`task_guid`即可。

注意，如果要移除的依赖非当前任务的依赖，会被自动忽略。接口会返回成功。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=task&method=remove_dependencies)

:::html
<md-alert type="tip">
移除任务依赖时，需要当前任务的编辑权限。
</md-alert>
:::
