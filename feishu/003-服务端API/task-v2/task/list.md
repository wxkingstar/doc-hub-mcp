# 列取任务列表

基于调用身份，列出特定类型的所有任务。支持分页。

目前只支持列取任务界面上“我负责的”任务。返回的任务数据按照任务在”我负责的“界面中”自定义拖拽“的顺序排序。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=task&method=list)
