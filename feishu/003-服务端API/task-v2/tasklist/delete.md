# 删除清单

删除一个清单。

删除清单后，不可对该清单做任何操作，也无法再访问到清单。清单被删除后不可恢复。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=tasklist&method=delete)

:::html
<md-alert type="tip">
删除清单需要清单管理权限。详情见[清单功能概述](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/task-v2/tasklist/overview)中的“清单是如何鉴权的？“章节。
</md-alert>
