# 创建评论

为一个任务创建评论，或者回复该任务的某个评论。

若要创建一个回复评论，需要在创建时设置`reply_to_comment_id`字段。被回复的评论和新建的评论必须属于同一个任务。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=comment&method=create)

:::html
<md-alert type="tip">
对任务进行评论时需要评论的读取权限。详情见[清单功能概述](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/task-v2/tasklist/overview)中的“清单是如何鉴权的？“章节。
</md-alert>
