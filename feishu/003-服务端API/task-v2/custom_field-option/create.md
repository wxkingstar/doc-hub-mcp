# 创建自定义字段选项

为单选或多选字段添加一个自定义选项。一个单选/多选字段最大支持100个选项。

新添加的选项如果不隐藏，其名字不能和已存在的不隐藏选项的名字重复。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=task&version=v2&resource=custom_field.option&method=create)

:::html
<md-alert type="tip">
需要对自定义字段的编辑权限。
</md-alert>
