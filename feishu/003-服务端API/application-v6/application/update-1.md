# 更新应用协作者

某个应用（包括自建应用和商店应用）中添加/移除应用协作者，添加后协作者将会收到添加通知。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=application&version=v6&resource=application.collaborators&method=update)

:::html
<md-alert type="tip">
若用 user_access_token 代表某个终端用户操作API，则需确保该用户为应用的所有者或管理员，否则无法操作成功。
</md-alert>
