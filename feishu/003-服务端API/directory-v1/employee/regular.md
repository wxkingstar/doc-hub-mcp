# 更新待离职成员为在职

本接口用于为待离职员工取消离职，将其更新为「在职」状态。取消离职时会清空离职信息。
使用user_access_token时默认为管理员用户，仅可操作「人事管理模式」的管理员可操作。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=directory&version=v1&resource=employee&method=regular)
