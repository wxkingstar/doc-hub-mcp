# 删除评论

删除某审批实例下的一条评论或评论回复（不包含审批同意、拒绝、转交等附加的理由或意见），删除后在审批中心的审批实例内不再显示评论内容，而是显示 **评论已删除**。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=approval&version=v4&resource=instance.comment&method=delete)
