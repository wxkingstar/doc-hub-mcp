# 服务台机器人向工单绑定的群内发送消息

通过服务台机器人给指定用户的服务台专属群或私聊发送消息，支持文本、富文本、卡片、图片。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=helpdesk&version=v1&resource=bot.message&method=create)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">
- 暂不支持新版卡片（ JSON 2.0 版本）。
- 暂不支持商店应用使用。
- 只支持向没有进行中工单的专属群中发送消息。

</md-alert>
:::
