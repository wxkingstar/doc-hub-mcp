# 更新雇佣信息

更新人事工作信息下的字段，如：工号、工作邮箱、雇佣类型、自定义字段等{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v1&resource=employment&method=patch)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">

</md-alert>
:::

:::html
<md-alert type="tip">
- 非必填字段，不传时即不做变更
- 若开启工号自动编码规则则无需输入人员“工号”，系统将自动进行工号生成；若手动输入工号，则会按照手动输入工号内容进行人员档案建立
- 暂不支持更新时间轴分组对象
</md-alert>
:::
