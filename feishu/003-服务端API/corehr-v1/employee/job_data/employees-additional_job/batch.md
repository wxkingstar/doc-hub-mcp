# 批量查询兼职信息

批量查询兼职信息，包括开始日期、职务、序列、上级、薪资类型等信息。
支持全量遍历和筛选查询。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v2&resource=employees.additional_job&method=batch)

:::html
<md-alert type="tip">
- 本接口会按照「员工资源」权限范围返回数据，请确定在「开发者后台 - 权限管理 - 数据权限」中已申请此数据权限

</md-alert>
:::

:::html
<md-alert type="warn">
- 延迟说明：搜索同步延迟 10s 以内，即：直接创建、更新对象后10s内调用此接口可能查询不到最新数据。
</md-alert>
:::
