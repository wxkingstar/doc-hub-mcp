# 【事件】更新成本中心

飞书人事中「成本中心信息被更新」时将触发此事件。

**注意事项**：[【搜索成本中心信息】](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/corehr-v2/cost_center/search)接口数据查询存在一定延迟(2s内)，不建议收到事件后立即查询以上接口。
- 以下字段变更会收到事件：
  - ```cost_center(成本中心)相关字段: is_autogenerate(是否自动生成)、support_cost_center(支持的业务线)、cost_center_manager(成本中心负责人)、effective_time(生效日期)```
  - ```organization(组织)相关字段：name(名称)、code(编码)、description(描述)、superior_org(上级组织)、active(启用)```
- 以下字段变更不会收到事件：```wk_created_at(创建时间)、wk_updated_at(更新时间)、wk_deleted_at(删除时间)、wk_tid(时间轴记录ID)、wk_created_by(创建人)、wk_updated_by(更新人)、wk_id(ID)、expiration_time(失效时间)、wk_status(时间轴状态)```
**注意事项**：- **该接口只会推送当前生效成本中心的变更事件**。
- 未来生效的版本数据，会在生效日期当天凌晨推送事件。例如：今天为1月1日，修改成本中心名称1月10日生效，则1月10日凌晨发送该对象变更事件。

## 前提条件
你需要在应用中配置事件订阅，这样才可以在事件触发时接收到事件数据。了解事件订阅可参见[事件订阅概述](https://open.feishu.cn/document/ukTMukTMukTM/uUTNz4SN1MjL1UzM)。

## 事件

基本 | 
--- | ---
事件类型 | corehr.cost_center.updated_v2
支持的应用类型 | 自建应用、商店应用
权限要求  
            **订阅该事件所需的权限，开启其中任意一项权限即可订阅**  
            开启任一权限即可 | 查看成本中心信息(corehr:cost_center:read)  
            创建、更新、删除成本中心信息(corehr:cost_center:write)
推送方式 | [Webhook](https://open.feishu.cn/document/ukTMukTMukTM/uUTNz4SN1MjL1UzM)

### 事件体

名称 | 类型 | 描述
--- | --- | ---
schema | string | 事件模式
header | event_header | 事件头
event_id | string | 事件 ID
event_type | string | 事件类型
create_time | string | 事件创建时间戳（单位：毫秒）
token | string | 事件 Token
app_id | string | 应用 ID
tenant_key | string | 租户 Key
event | \- | \-
cost_center_id | string | 成本中心ID。  
- 调用[【搜索成本中心信息】](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/corehr-v2/cost_center/search)接口返回成本中心详细信息
field_changes | string\[\] | 发生变更的字段  
**数据校验规则**：  
- 长度范围：`0` ～ `200`

package main

import (
	"context"
	"fmt"

larkcore "github.com/larksuite/oapi-sdk-go/v3/core"
	larkevent "github.com/larksuite/oapi-sdk-go/v3/event"
	"github.com/larksuite/oapi-sdk-go/v3/event/dispatcher"
	"github.com/larksuite/oapi-sdk-go/v3/service/corehr/v2"

// SDK 使用说明 SDK user guide：https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/server-side-sdk/golang-sdk-guide/preparations
func main() {
	// 注册事件 Register event
	eventHandler := dispatcher.NewEventDispatcher("", "").
		OnP2CostCenterUpdatedV2(func(ctx context.Context, event *larkcorehr.P2CostCenterUpdatedV2) error {
			fmt.Printf("[ OnP2CostCenterUpdatedV2 access ], data: %s\n", larkcore.Prettify(event))
			return nil
		})

// 构建 client Build client
	cli := larkws.NewClient("YOUR_APP_ID", "YOUR_APP_SECRET",
		larkws.WithEventHandler(eventHandler),
		larkws.WithLogLevel(larkcore.LogLevelDebug),
	)

// 建立长连接 Establish persistent connection
	err := cli.Start(context.Background())

if err != nil {
		panic(err)
	}
}
