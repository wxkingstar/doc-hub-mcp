# 步骤四：体验效果

在本步骤，你将在飞书客户端查看和验证示例代码运行后生成的日历。

:::html
<md-td>
## 操作步骤
1. 打开桌面端飞书客户端。
2. 在客户端左侧单击 **日历** 进入日历页面，依次选择 **+** > **订阅日历** 。
    
    <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/fca9a862e1f642ee8151584d1189b1a6~tplv-goo7wpa0wc-image.image?lazyload=true&width=2380&height=1556" style="width:60%">
    
3. 选择 **公共日历** ，然后搜索 **项目版本迭代计划** ，单击 **订阅** 。
    
    <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/6bde8a7244d84263bd13b16c964fdeb2~tplv-goo7wpa0wc-image.image?lazyload=true&width=993&height=1157" style="width:40%">
    
4. 订阅成功后，打开日历界面，即可看到对应的项目进度日历。
    
    <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/8d1ef744d0734deca29976b88dd50b37~tplv-goo7wpa0wc-image.image?lazyload=true&width=1640&height=727" style="width:60%">
    
5. 打开多维表格，可以看到多维表格中的版本迭代计划与日历一致。

    <img src="//sf3-cn.feishucdn.com/obj/open-platform-opendoc/073d9142304d69478a3572bc07d1bdbc_82jyRMjK7z.png?lazyload=true&width=1640&height=603" style="width:60%">
