# 步骤二：添加机器人到群
在本步骤，你需要将已创建的机器人应用添加到用于接收周报通知的群组。
## 操作步骤
:::html
<md-td>
1. 使用测试企业账号登录飞书客户端。
  
	<img src="//sf3-cn.feishucdn.com/obj/open-platform-opendoc/fe000dbf4dd4043c5a73ff4a4dd5dff6_X3PUzeUqUe.png?lazyload=true&width=2202&height=1658" style="width:70%"/>
  
2. 创建一个用于测试的群组。
  
3. 在群组右上角依次选择 **···** > **设置**，在 **设置** 页面单击 **群机器人**。

	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/33564a9bb81b46fbb99622dfaff71e81~tplv-goo7wpa0wc-image.image?lazyload=true&width=1776&height=1468" style="width:70%"/>
  
4. 单击 **添加机器人**。在弹出的对话框中搜索测试应用名称，单击 **添加**。

	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/f7529490cfb64380a6059314d014ff0c~tplv-goo7wpa0wc-image.image?lazyload=true&width=1522&height=1146" style="width:70%"/>

  	你已成功在群组内添加该机器人。你可以在群机器人页面进行查看该机器人。

	<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/23ed3fdb832b4dffade81f0e3cdff74e~tplv-goo7wpa0wc-image.image?lazyload=true&width=1778&height=964" style="width:70%"/>

</md-td>
:::
