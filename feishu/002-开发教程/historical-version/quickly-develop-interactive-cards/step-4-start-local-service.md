# 步骤四：启动本地服务

在本步骤中，您将运行教程提供的示例代码。示例代码需要本地环境已安装 Node.js 环境。

## 操作步骤

1. 执行以下命令，进入 message-card 文件夹。
     ```
     cd  message-card
     ```

2. 执行以下命令，安装项目依赖。
    
    ```
    npm install
    ```
	
    ![](//sf3-cn.feishucdn.com/obj/open-platform-opendoc/9a515b50fc0f0428effab5f8d7339082_J0lvaKkCeO.png?height=548&lazyload=true&maxWidth=600&width=1596)

3. 依赖安装成功后，执行以下命令，启动本地服务。
    
    ```
    node index.js
    ```
