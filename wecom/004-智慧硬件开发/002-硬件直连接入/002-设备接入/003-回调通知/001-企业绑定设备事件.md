最后更新：2022/09/16

告知设备当前已经完成企业绑定，可以立刻来获取secret，优化响应速度

**指令包体：**

```
{
  "cmd" : "push_bind_corp",
  "headers": 
  {
     "req_id" : "xxxx"
  }
}
```

**参数说明：**

| 参数名 | 类型 | 描述 |
| --------- | ------ | -------------------------------- |
| req_id | string | 透传请求的req_id |
| cmd | string | 指令号，此处固定为push_bind_corp |
