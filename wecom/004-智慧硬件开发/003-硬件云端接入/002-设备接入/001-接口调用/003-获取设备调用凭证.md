通过[设备授权秘钥](#40202)接口获取秘钥后，硬件服务商可通过此接口获取到设备的access_token。而后使用device_access_token来调用获取设备详情、上报记录等接口。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/openhw/get_device_token?model_access_token=MODEL_ACCESS_TOKEN

**请求包体：**

```
{
  "device_sn":"SN1" ,
  "device_secret": "ldAE_H9anCRN21GKXVfdAAAAAAAAAAAAAAAAAA", 
}

 ```
**参数说明：**
 
 | 参数 | 是否必须 | 说明 |
 |----|---- | ----- |
 | device_sn | 是 | 设备SN |
 | device_secret | 是 | 设备授权秘钥，通过[获取设备秘钥](#40202)接口获取 |

**返回结果：** 

```
{
  "errcode":0,
  "errmsg":"ok",
  "device_access_token":"61W3mEpU66027wgNZ_MhGHNQDHnFATkDa9-2llMBjUwxRSNPbVsMmyD-yq8wZETSoE5NQgecigDrSHkPtIYA",
  "expires_in":7200
}
```

**参数说明：** 

 | 参数 | 说明 |
 |----|---- |
 | device_access_token | 设备access_token,最长为512字节 |
 | expires_in | 有效期（秒） |
 
**注意**：因历史原因，该接口在调用失败时才返回errcode。没返回errcode视为调用成功
