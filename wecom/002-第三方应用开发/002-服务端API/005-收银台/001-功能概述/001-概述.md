## 概述
企业微信在桌面端「服务商助手」及「服务商后台」中向服务商提供了「收银台」功能，帮助服务商更加便捷的进行应用的销售，及相关订单和收入的管理。
服务商可开通商户号，实现企业客户购买应用的收款。资金T+12（包括10天企业退款冻结期）即可提现到服务商银行账户。
## 商户号

- ![2.png](https://wework.qpic.cn/wwpic/32929__8aM7iNwSSW1Ldq_1669867825/0)
- 服务商需要在企业微信平台，分别开通微信支付、商企付两个商户号（微信支付和商企付商户号的支付服务均由财付通提供）。
- 其中微信支付商户号用于收款企业客户通过微信支付的资金，商企付商户号用于收款企业通过网银B2B转账的资金。两个商户号的开通、收款、退款、提现完全独立。
- 服务商在企业微信平台开通的商户号，是企业微信平台的子商户号。若服务商之前已在微信支付或商企付开通过其他商户号，与在企业微信平台开通的商户号并不冲突，是完全独立的，不支持之前开通的商户号与在企业微信平台开通的商户号合并。<br>
## 收银台
企业微信服务商助手-收银台，提供给服务商管理所有客户订单、进行收益提现等功能。包含如下几个功能模块：
- 收款：创建收款订单并管理所有待付款订单。
- 订单管理：查看并管理服务商的所有客户订单。包括第三方应用订单、代开发应用订单、行业方案订单。
- 退款管理：处理客户的退款申请。
- 发票管理：处理客户的开票申请。
- 商户号管理：开通并管理用于收款的商户号。
- 收入数据：查看商户号收入统计与明细。
- 提现：提现到银行账户。

### 1、收款
#### 创建收款订单
1. 服务商帮助客户选择所需要购买的第三方应用/代开发应用/行业方案。
2. 配置购买的人数和时长。（可选，非必须配置）
3. 创建订单并获取支付链接/二维码，可发送给客户进行支付。
#### 管理待付款订单
- 展示所有已创建的待支付订单，区分客户下单/服务商创建两个列表。
- 服务商可点击查看订单详情并可获取支付二维码/链接发送给客户进行支付。
- 针对客户下单的订单，服务商可进行改价，最低折扣为原价的7折。
- ![image.png](https://wework.qpic.cn/wwpic/800107_suC5mLegQB2MhRn_1669799699/0)
### 2、订单管理
#### 订单列表
- 展示服务商的所有客户订单（测试授权订单除外），可进行搜索和筛选。
- 导出订单：导出当前筛选条件下的所有订单。
#### 测试授权订单
- 展示所有服务商的测试授权订单
- ![image (1).png](https://wework.qpic.cn/wwpic/163639_upsfpk5eQh-x7iT_1669799746/0)
#### 订单状态
1. 待支付状态
- 待支付状态下，用户支付完成，订单状态变为：已支付。
- 待支付状态订单支付有效期为10天，订单创建时间10天后未支付，自动取消。取消后订单状态变为：交易关闭-支付过期。
- 待支付状态下，用户可主动操作取消订单，取消后订单状态变为：交易关闭-订单取消。
- 订单来源为“服务商创建”时：服务商可操作取消订单、获取订单支付链接/二维码。
- ![image (2).png](https://wework.qpic.cn/wwpic/931967_2eTA0zThRLWRDbc_1669799797/0)
- 订单来源为“客户创建”时：服务商可操作改价、获取订单支付链接/二维码
- ![image (3).png](https://wework.qpic.cn/wwpic/437353_yABJwr9mT9GFbso_1669799827/0)
2. 已支付状态
- 用户付款后订单进入已支付状态，此时用户的款项处于冻结期。
- 冻结期默认为10天，冻结期过后款项经过分账，属于服务商收入的部分进入服务商商户号。
- 冻结期内用户可以发起退款申请，只能发起一次。
- ![image (4).png](https://wework.qpic.cn/wwpic/589175_AZPfR3sgQ7OQ_BH_1669799859/0)
3. 交易关闭-支付过期
- 订单创建时间10天后未支付，自动取消。取消后订单状态变为：交易关闭-支付过期。
- 交易关闭-支付过期状态为最终状态，不可操作。
- ![image (5).png](https://wework.qpic.cn/wwpic/366045_byuQ2hVjS6CQSFF_1669799884/0)
4. 交易关闭-订单取消
- 待支付状态下，用户可主动操作取消订单，取消后订单状态变为：交易关闭-订单取消。
- 交易关闭-订单取消状态为最终状态，不可操作。
- ![image (6).png](https://wework.qpic.cn/wwpic/289470_4eMJttgKT_qpWqy_1669799912/0)
5. 退款申请中状态
- 已支付的订单，用户可以在10天内发起退款申请。
- 用户发起退款申请后，服务商需要在5天内处理完成，否则系统自动同意退款。
- 服务商只可操作“同意退款”，如该笔订单不需要退款，可与客户协商让客户侧取消退款。
- ![image (7).png](https://wework.qpic.cn/wwpic/303737_6i4VycJ1R2msbPb_1669799984/0)
6. 交易关闭-已退款状态
- 同意退款/超时自动退款后，订单流转为交易关闭-已退款的状态。
- 同意退款/超时自动退款后，用户支付的款项通过原支付渠道返回给用户。
- 详情页面展示退款申请及退款审核信息。
- ![image (8).png](https://wework.qpic.cn/wwpic/444328_e9Q7W9WcSY2lQA-_1669800012/0)
7. 交易完成状态
- 订单款项过了冻结期进入服务商商户号时，该笔订单流转为“交易完成”状态。
- 交易完成状态的订单详情页展示收入到账的明细信息。
- ![image (9).png](https://wework.qpic.cn/wwpic/279463_uQ6qdxURTbeZwkx_1669800046/0)
### 3、退款管理
#### 退款申请列表
- 列表展示所有“退款申请中”状态的客户订单，可进行搜索。
- 可进入具体订单详情页面进行退款处理。
- ![image (10).png](https://wework.qpic.cn/wwpic/445993_6mzj9wUdTR2wwlU_1669800077/0)
#### 退款处理记录
- 展示所有已处理的订单退款申请，可进行退款状态的筛选。
- ![image (11).png](https://wework.qpic.cn/wwpic/333021_mqT53jrgTHinVdF_1669800106/0)
### 4、发票管理
#### 待开票列表
- 交易完成状态的订单，用户可以发起开票申请。发起申请后进入待开票列表。
- 可查看对应订单详情及发票信息。
- 开票完成后需要“标记为已开票”并录入开票信息，标记后进入已开票列表
- ![image (12).png](https://wework.qpic.cn/wwpic/351505_OEMkC5PqTpWY0KW_1669800142/0)
#### 已开票列表
- 可查看已开票订单的订单详情、发票信息。
- 可查看开票信息，包括开票备注、操作人、操作时间。
- ![image (13).png](https://wework.qpic.cn/wwpic/6688_feg6pwU9Q4SphnU_1669800167/0)
### 5、商户号管理
- 可查看交易手续费说明及已开通的商户号信息。
- 可在线修改微信支付商户的银行账户。
- 可联系客服修改商企付商户的银行账户。
- ![image (14).png](https://wework.qpic.cn/wwpic/731320_xyb1t6JSSZaEMH3_1669800189/0)
### 6、收入数据
- 可查看商户号的收入统计数据及收入明细数据。
- 收入统计数据包括服务商商户号昨日、当日、本周、本月的到账金额。同时提供周维度的收入趋势图供查看。
- 收入明细数据呈现了每笔订单的收入到账数据，可筛选及导出。默认展示当月的收入明细。
- ![image (15).png](https://wework.qpic.cn/wwpic/689242_3kODdFkmQVeIp-m_1669800215/0)
### 7、提现管理
- 提现管理包括商户金额总览及提现记录两个模块。
- 商户金额总览模块包括可提现金额（已经解冻并进入服务商商户号内的金额）及冻结中的金额。
- 提现记录模块呈现了从商户号中提现到银行卡的每一笔提现明细。可筛选及导出。
- ![image (16).png](https://wework.qpic.cn/wwpic/590677_ggA2s2rnQ4WoPlT_1669800247/0)
