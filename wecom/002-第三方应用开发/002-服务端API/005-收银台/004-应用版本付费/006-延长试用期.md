最后更新：2020/01/13

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/service/prolong_try?suite_access_token=SUITE_ACCESS_TOKEN

**请求包体：**

```
{
  "buyer_corpid" : "wx7da9abf8ac62baaa",
  "prolong_days" : 7,
  "appid" : 1
}
```

**参数说明：** 

| 参数 | 是否必须 | 说明 |
| ------- | ---- | ---- |
| buyer_corpid | 是 | 购买方corpid |
| prolong_days | 是 | 延长天数 |
| appid | 否 | 仅旧套件需要填此参数 |

**注意:**
1. 一个应用可以多次延长试用，但是试用总天数不能超过60天
2. 仅限时试用或试用过期状态下的应用可以延长试用期

**返回结果：** 

```
{ 
 "errcode" : 0,
 "errmsg" : "ok",
  "try_end_time" : 1565152189
}
```

**参数说明：**

| 参数 | 说明 |
| ------------- | ---------------------------------------------------- |
| errcode | 错误码 |
| errmsg | 错误码说明 |
| try_end_time | 延长后的试用到期时间（UNIX时间戳） |
