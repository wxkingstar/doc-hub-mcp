查询企业高级接口已购信息。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/advanced_api/get_corp_buy_info?provider_access_token=ACCESS_TOKEN

**请求包体：**

```
{
  "advanced_api_type":1,
  "custom_corpid":"xxxxx"
}
```

**参数说明：** 

| 参数 | 是否必须 | 说明 |
| ------- | ---- | ---- |
| provider_access_token | 是 | 应用服务商的接口调用凭证，获取方法参见[服务商的凭证](#15143 ) |
|advanced_api_type|是|购买的高级接口类型，目前仅支持“会话内容数据接口”，固定为1|
|custom_corpid|是|客户企业id|

**返回结果：** 
```
{
 "errcode": 0,
 "errmsg": "ok",
 "chat_archive_api_buy_info": {
 "edition_list": [
 {
 "edition": 2,
 "purchase_count": 100,
 "begin_time": 1726588800,
 "end_time": 1729180800
 },
 {
 "edition": 3,
 "purchase_count": 100,
 "begin_time": 1726588800,
 "end_time": 1729180800
 }
 ]
 }
}
```

**参数说明：**

| 参数 | 说明 |
| ------- | ---- |
|errcode |错误码 |
|errmsg|错误码说明|
|chat_archive_api_buy_info|“会话内容数据接口”的购买信息|
|chat_archive_api_buy_info.edition_list|参考ChatArchiveApi结构说明|
>返回的版本列表不包括试用版，只返回最新待生效、生效中或已到期的版本。如果同一个版本同时存在已到期跟生效中或待生效的版本，则只返回生效中或者待生效的版本；如果同一个版本同时存在生效中或者待生效的版本，则两个都会返回。

**ChatArchiveApi结构说明：**
| 参数 | 说明 |
| ------- | ---- |
|edition_list|已购版本列表|
|edition|订单购买的版本：1-内部会话（历史购买的），2-内外部会话，3-内外部会话及语音通话|
|purchase_count|购买人数|
|begin_time|生效时间|
|end_time|到期时间|
