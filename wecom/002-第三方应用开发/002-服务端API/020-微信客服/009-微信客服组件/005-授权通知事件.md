企业授权微信客服组件时，会跟安装第三方应用一样产生[授权通知事件](#14951)。

除了公共的字段信息外，微信客服组件的[授权成功通知](#14951/授权成功通知)和[变更授权通知](#14951/变更授权通知)会额外携带`ExtraInfo.AuthOpenKfIdList`参数，用于告知服务商用户当前授权是选择了哪些客服账号。

##授权成功通知
企业授权微信客服组件时，企业微信后台会推送授权成功通知，会产生[授权成功通知](#14951/授权成功通知)。除公共的字段信息外，会额外携带`ExtraInfo.AuthOpenKfIdList`参数，用于告知服务商用户当前授权是选择了哪些客服账号。

**请求包体：**
```
<xml>
 <SuiteId><
- ![CDATA[wxxxxxxx]]></SuiteId>
 <AuthCode><
- ![CDATA[AUTHCODE]]></AuthCode>
 <InfoType><
- ![CDATA[create_auth]]></InfoType>
 <TimeStamp>1403610513</TimeStamp>
 <State><
- ![CDATA[STATE]]></State>
 <ExtraInfo>
 <AuthOpenKfIdList>
 <OpenKfId><
- ![CDATA[wk11111]]></OpenKfId>
 <OpenKfId><
- ![CDATA[wk22222]]></OpenKfId>
 <OpenKfId><
- ![CDATA[wk33333]]></OpenKfId>
 </AuthOpenKfIdList>
 </ExtraInfo>
</xml>
```

参数说明：

 | 参数 | 说明 |
 |---- | ----- |
 |SuiteId等公共字段 | 见[授权成功通知](#14951/授权成功通知)|
 |ExtraInfo.AuthOpenKfIdList|用户本次授权的客服账号列表|
 
##变更授权通知
企业修改授权的客服链接时，企业微信后台会推送[变更授权通知](#14951/变更授权通知)。除公共的字段信息外，会额外携带`ExtraInfo.AuthOpenKfIdList`参数，用于告知服务商用户当前授权是选择了哪些客服账号。

**请求包体：**
```
<xml>
 <SuiteId><
- ![CDATA[wxxxxxxx]]></SuiteId>
 <InfoType><
- ![CDATA[change_auth]]></InfoType>
 <TimeStamp>1403610513</TimeStamp>
 <AuthCorpId><
- ![CDATA[AUTHCORPID]]></AuthCorpId>
 <State><
- ![CDATA[STATE]]></State>
 <ExtraInfo>
 <AuthOpenKfIdList>
 <OpenKfId><
- ![CDATA[wk11111]]></OpenKfId>
 <OpenKfId><
- ![CDATA[wk22222]]></OpenKfId>
 <OpenKfId><
- ![CDATA[wk33333]]></OpenKfId>
 </AuthOpenKfIdList>
 </ExtraInfo>
</xml>
```

参数说明：

 | 参数 | 说明 |
 |---- | ----- |
 |SuiteId等公共字段 | 见[变更授权通知](#14951/变更授权通知)|
 |ExtraInfo.AuthOpenKfIdList|用户本次授权的客服账号列表|
 
##取消授权通知
当授权方（即授权企业）取消了对组件的授权，企业微信后台会推送取消授权通知。详见[取消授权通知](#14951/取消授权通知)
