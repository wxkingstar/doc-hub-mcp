### wx.qy.selectExternalChat(Object object)

>**调用前提：**
> 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态
> 当前成员必须在应用的可见范围
> 仅[配置了客户联系功能](#13473/配置可使用客户联系功能的成员)的用户可调用
> 仅支持「营销获客」分类的第三方应用可调用

「营销获客」类应用需要通过群活码接口为企业获客，在创建群活码时，需要传入群 Id
本接口即提供一种合理且安全的方案，让应用可获得群 id

#### 参数

##### Object object

| 参数名 | 类型 | 必填 | 说明 |
| ---------- | -------- | ---- | ---------------------------------------- |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

###### object.success 回调函数

**参数**

**Object res**

| 参数 | 类型 | 说明 |
| ------- | ----- | -------------------- |
| chatIdList | Array | 返回此次选择的客户群ID列表 |

#### 示例代码：

```js

wx.qy.selectExternalChat({
 success: function(res) {
 var chatIdList = res.chatIdList;
 }
});

```
