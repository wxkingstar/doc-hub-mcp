> iBeacon接口仅在企业微信支持，微信端不支持（微信开发者工具也不支持）

企业微信为开发者提供了一套完整的接口，以便更好地利用iBeacon设备。这些接口包括开始和停止搜索附近的iBeacon设备，以及监听iBeacon设备的更新事件和服务状态的变化。

## 功能介绍
1. 开始搜索附近的iBeacon设备（startBeaconDiscovery）：这个功能允许应用程序开始搜索附近的iBeacon设备。调用后，应用程序将开始接收附近iBeacon设备的信号，并可以根据这些信号提供相关的服务或信息。
2. 停止搜索附近的iBeacon设备（stopBeaconDiscovery）：当应用程序不再需要接收iBeacon设备的信号时，可以使用这个功能停止搜索。这可以帮助节省设备的电池寿命，并减少不必要的信号干扰。
3. 监听iBeacon设备的更新事件（onBeaconUpdate）：这个功能允许应用程序监听iBeacon设备的更新事件。当附近的iBeacon设备的信息（如信号强度或距离）发生变化时，应用程序将收到通知。
4. 监听iBeacon服务的状态变化（onBeaconServiceChange）：这个功能允许应用程序监听iBeacon服务的状态变化。例如，当iBeacon服务从未启动状态变为启动状态，或从启动状态变为停止状态时，应用程序将收到通知。
