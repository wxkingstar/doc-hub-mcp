学校的老师可以添加外部联系人进行工作沟通，外部联系人分为企业微信联系人和微信联系人两种类型。

家校应用开发者可以通过API接口设置“学校通知关注模式”，发送“学校通知”和管理“家校通讯录”等功能。[第三方应用](#90001/90143/92283)和[企业](#90001/90143/92283)的旧版家校接口已经迁移，并且不再推荐使用。

## 开始开发

### 配置“家长可使用的应用”
配置“家长可使用的应用”，家长可在微信接收应用消息：管理员进入企业微信管理后台后点击“家校沟通”-“配置”页面进行配置。
- ![](http://p.qpic.cn/pic_wework/2500642931/1711cd0783f2f3c1d5eb644b99967a34c46a05ab0a86ec22/0)

### 配置可调用接口的应用
管理员进入企业微信管理后台后点击“家校沟通”页面，点开“API”小按钮，点击可调用接口的应用旁的设置按钮，会进入家校沟通的可调用接口应用配置页面。
- ![enter image description here](https://wework.qpic.cn/wwpic3az/891065_SQxgdJE8Rd2QsEb_1701265233/0)
在家校沟通的可调用接口应用配置页面，可选择自建应用进行配置
- ![enter image description here](https://wework.qpic.cn/wwpic3az/629027_jF2X6fYcQzmAKwd_1701264734/0)
#### 通过API读取和编辑家校通讯录
可设置同时具备读取以及编辑家校通讯录权限的应用，此类应用仅可设置一个
#### 通过API读取家校通讯录
可设置具备读取家校通讯录权限的应用，此类应用可配置多个

### API接口
1. 学校可以在相关的应用里调用外部联系人选人接口，获取成员选择的外部联系人的userid以及通过该userid获取外部联系人详情。（ [外部联系人选人接口](https://work.weixin.qq.com/api/doc#14928/外部联系人选人接口) | [获取外部联系人详情](https://work.weixin.qq.com/api/doc#16685)）
- ![](http://p.qpic.cn/pic_wework/3499811875/75733179126f060911ba72014fd21c2934cba849dee7ef07/0)

2. 我们提供了jsapi接口，方便员工在学校的应用里能够快捷[打开外部联系人的个人信息页](https://work.weixin.qq.com/api/doc#14928/打开个人信息页接口)以及[打开聊天窗口](https://work.weixin.qq.com/api/doc#14929/创建会话接口)。

## 关于ExternalUserId
ExternalUserId是企业微信外部联系人的标识，但需要开发者(尤其是第三方开发者)注意的是，对于同一个外部联系人，**不同调用方（企业/第三方服务商）获取到的ExternalUserId是不同的**。
