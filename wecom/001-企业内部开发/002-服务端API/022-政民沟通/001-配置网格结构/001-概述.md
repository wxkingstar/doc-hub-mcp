可配置多级网格结构，通过树状网格结构实现基层工作的分级管理。可逐级添加下级网格并配置网格中的负责人和网格员，通过「添加网格」、「编辑网格」、「删除网格」、「获取网格列表」、「获取网格员及负责人列表」接口，可实现对管理范围内的网格以及网格中的人员进行增、删、改配置。

#### **配置网格结构的限制规则**
1. 整个网格结构最多10层
2. 每个网格节点内，最多有20个负责人、100个网格员
3. 每个成员最多只能是10个网格的负责人
4. 同一个目标网格下，不能存在同名的同级子网格
