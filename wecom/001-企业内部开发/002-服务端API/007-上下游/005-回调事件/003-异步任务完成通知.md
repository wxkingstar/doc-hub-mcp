本事件是在使用[异步导入上下游联系人接口](#38752/自建应用上下游回调通知)时，用于接收任务执行完毕的结果通知。回调推送地址为调用接口应用配置的回调地址。

请求示例：

```
<xml>
 <ToUserName><
- ![CDATA[wx28dbb14e3720FAKE]]></ToUserName>
 <FromUserName><
- ![CDATA[FromUser]]></FromUserName>
 <CreateTime>1425284517</CreateTime>
 <MsgType><
- ![CDATA[event]]></MsgType>
 <Event><
- ![CDATA[batch_job_result]]></Event>
 <BatchJob>
 <JobId><
- ![CDATA[jobid_S0MrnndvRG5fadSlLwiBqiDDbM143UqTmKP3152FZk4]]></JobId>
 <JobType><
- ![CDATA[import_chain_contact]]></JobType>
 <ErrCode>0</ErrCode>
 <ErrMsg><
- ![CDATA[ok]]></ErrMsg>
 </BatchJob>
</xml>
```
参数说明：

 | 参数 | 说明 |
 |----- | ----- |
 | ToUserName | 企业微信CorpID |
 | FromUserName | 此时固定为sys|
 | CreateTime | 消息创建时间（整型）|
 | MsgType | 消息类型，此时固定为：event |
 | Event | 事件类型：batch_job_result |
 | JobId | 异步任务id，最大长度为64字符 |
 | JobType | 操作类型，字符串，固定为：import_chain_contact(导入上下游联系人)|
 | ErrCode | 返回码 |
 | ErrMsg | 对返回码的文本描述内容 |
