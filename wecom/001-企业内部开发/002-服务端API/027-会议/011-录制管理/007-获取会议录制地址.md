获取会议录制地址，可获取会议云录制的播放地址和下载地址。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record/get_file_list?access_token=ACCESS_TOKEN
 
**请求包体：**
```
{
  "meeting_record_id": "record",
  "meetingid":"MEETINGID"
}
```

**参数说明：**
| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meeting_record_id |是 |string|会议录制ID |
|meetingid|是|string|会议ID|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "meeting_record_id": "record",
  "meetingid": "MEETINGID",
  "meeting_code": "MEETINGOCDE",
  "title": "meetingtitle",
  "record_files": [{
    "record_file_id": "FILEID",
    "view_address": "https://view.url/124",
    "download_address": "https://download.url/234",
    "download_address_file_type": "mp4",
    "audio_address": "https://audio.download/342",
    "audio_address_file_type": "m4a",
    "meeting_summary": [{
      "download_address": "https://summary.url",
      "file_type": "txt"
    }]
  }]
}
```

**参数说明：**
|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
|meeting_record_id |string|会议录制 ID。 |
|meetingid |string|会议 ID。 |
|meeting_code |string|会议 code。 |
|title |string|会议主题。 |
|record_files |object[]|直播录制文件列表，详见RecordFile|

**RecordFile说明**
|参数|类型|说明 |
|:--|
|record_file_id |string|录制文件 ID。 |
|view_address |string|播放地址。 |
|download_address |string|下载地址，默认6小时过期 |
|download_address_file_type |string|下载视频文件格式，例如：mp4。 |
|audio_address |string|音频下载地址，默认6小时过期|
|audio_address_file_type |string|下载音频文件格式，例如：m4a。|
|meeting_summary |object[]|会议纪要文件列表，详见Summary |

**Summary说明**
|参数 |类型|说明 |
|:--|
|download_address |string|下载文件地址，默认1小时过期 |
|file_type |string|下载文件类型，例如：txt、pdf、docs。 |
