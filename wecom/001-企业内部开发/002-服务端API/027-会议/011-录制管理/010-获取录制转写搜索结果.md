根据指定内容搜索录制转写

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record/transcript/search?access_token=ACCESS_TOKEN
**请求包体：**

```
{
  "record_file_id": "record",
  "meetingid": "MEETINGID",
  "text": "test"
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|record_file_id |是 |string|会议录制文件ID |
|meetingid |是 |string|会议ID |
|text|是|string|搜索的文本|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许获取该应用创建的会议的数据

**返回结果：**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "hits": [{
    "pid": "11",
    "sid": "2",
    "offset": 10,
    "length": 10
  }],
  "timelines": [{
    "pid": "11",
    "sid": "2",
    "start_time": 11111
  }]
}
```

**参数说明：**

|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
|hits|object[]|搜索结果列表，详见Result。|
|timelines|object[]|搜索结果时间戳对象列表，详见TimeLine。可用于时间轴上的预览。|

**Result说明**
|参数 |类型|说明 |
|:--|
|pid |string|段落 ID。 |
|sid |string|句子 ID。 |
|offset |int32|text 相对词的偏移。 |
|length |int32|匹配长度。 |

**Timeline说明**
|参数 |类型|说明 |
|:--|
|pid |string|搜索结果所在的段落 ID。 |
|sid |string|搜索结果所在的句子 ID。 |
|start_time |int64 |录制文件中的词开始时间（毫秒）。 |
