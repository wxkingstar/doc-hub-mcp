最后更新：2023/05/31

对成功预定的会议添加会议背景，支持多个背景图片的添加。一场会议最多添加7个背景，且仅支持不超过10MB大小的 PNG 格式图片，分辨率最小为1920x1080。添加成功返回新增的会议背景信息。
背景图片上传方式为异步上传，您可以通过订阅“[素材上传结果](#47145)”获取上传结果通知。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/add_background?access_token=ACCESS_TOKEN
 
**请求包体：**
```
{
  "meetingid": "MEETINGID",
  "image_list": [{
    "image_md5": "eax879asd3z",
    "image_url": "https://image.qq.com/1jsaz.png"
  }],
  "default_image_order": 1
}
```

**参数说明：**
|参数 |必须 |类型|说明 |
|:--|
|access_token |是 |string|调用接口凭证。获取方法查看“[获取access_token](https://work.weixin.qq.com/api/doc/90001/90143/90605)” |
|meetingid|是|string|会议 ID |
|image_list|是|object[]|图片对象列表，详见Image|
|default_image_order|否|uint32|图片列表中会议需要使用的背景图片序号，从1开始计数。不填默认为1|

**Image 说明**
|参数 |必须 |类型|说明 |
|:--|
|image_md5 |是|string|背景图片 MD5（图片内容 MD5 的十六进制表示）|
|image_url |是|string|背景图片 URL|

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "selected_background_id": "BACKID1",
  "background_list": [{
    "background_id": "BACKID1",
    "image_md5": "eax879asd3z"
  }]
}
```

**参数说明：**
|参数 |类型|说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
|selected_background_id |string|会议应用的背景 ID |
|background_list |object[]|背景对象列表，详见Background |

**Background 说明**
|参数 |类型|说明 |
|:--|
|background_id |string|背景 ID|
|image_md5 |string|背景图片 MD5（图片内容 MD5 的十六进制表示）|
