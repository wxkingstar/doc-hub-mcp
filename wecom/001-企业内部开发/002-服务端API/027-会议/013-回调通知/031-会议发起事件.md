当应用可见范围内的成员发起快速会议时或者作为首位参与者进入预约会议时，触发该事件。

**示例**
xml请求示例：
```
<xml>
 <ToUserName><
- ![CDATA[toUser]]></ToUserName>
 <FromUserName><
- ![CDATA[fromUser]]></FromUserName> 
 <CreateTime>1348831860</CreateTime>
 <MsgType><
- ![CDATA[event]]></MsgType>
 <Event><
- ![CDATA[meeting_statistics]]></Event>
 <ChangeType><
- ![CDATA[start_meeting]]></ChangeType
 <Status>1</Status>
</xml>
```

**说明**

 | 参数 | 说明 |
 |---- | ----- |
 | ToUserName | 企业微信CorpID |
 | FromUserName | 会议发起者的UserID。如果是企业外部成员作为首位参与者进入预约会议，则该字段为此预约会议创建者的UserID|
 | CreateTime | 消息创建时间，unix时间戳|
 | MsgType | 消息类型，此时固定为：```event``` |
 | Event | 事件类型，此时固定为：```meeting_statistics``` |
 | ChangeType | 类型，此时固定为：```start_meeting``` |
 |Status|会议状态</br>1:会议发起成功</br>2:会议发起失败|
