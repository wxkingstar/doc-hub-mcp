使用该接口，可为会议生成多个专属入会链接，不同链接以 `customer_data` 进行区分。通过用户入会、用户进入等候室等事件，或通过获取等候室成员列表的 API 可查询到该参数。
> 说明
> - 该接口不支持网络研讨会（Webinar）。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/create_customer_short_url?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "meetingid": "meetingid11234",
  "customer_data": "CUSTOMERDATA"
}
```

**参数说明：**

| 参数| 必须 | 类型|说明|
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
| meetingid | 是 |string| 会议ID|
|customer_data|是|string|用户专属字段，长度不超过256 字节。|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许修改该应用创建的会议

**返回结果：**
```
{
  "errcode": 0,
  "errmsg": "ok",
  "meeting_short_url_customer_data": {
    "customer_data": "CUSTOMERDATA",
    "meeting_short_url": "URL"
  }
}
```

**参数说明**

| 参数|类型| 说明|
| :--|
| errcode |int32| 返回码|
| errmsg |string| 对返回码的文本描述内容|
|meeting_short_url_customer_data|object[]|用户专属参会链接对象。详见CustomerData。|

**CustomerData说明**
|参数|类型|说明|
|:--|
|customer_data|string|用户专属字段|
|meeting_short_url|string|用户专属参会链接|
