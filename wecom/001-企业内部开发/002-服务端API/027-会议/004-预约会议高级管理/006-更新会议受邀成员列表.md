最后更新：2023/04/24

可根据会议 ID 更新会议受邀成员列表，最多可以传入2000名受邀成员，与企业所购在线会议室最大方数相关。管理员必须在受邀成员列表中。

**请求方式：**POST（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/meeting/set_invitees?access_token=ACCESS_TOKEN
**请求包体：**

```
{
  "meetingid": "meetingid",
  "invitees": [{
    "userid": "zhangsan"
  }, {
    "userid": "lisi"
  }]
}
```

**参数说明：**
|参数 |必须 |类型|说明 |
|:--|
| access_token | 是 | string|调用接口凭证。获取方法查看“[获取access_token](#15074)”|
|meetingid|是|string|会议ID|
|invitees|否|object[]|受邀成员列表。最大支持2000人。与企业所购在线会议室最大方数相关。|
|invitees.userid|否|string|受邀成员userid|

**权限说明**
- 仅配置在“可调用接口的应用”列表中的自建应用可调用
- 仅允许更新该应用创建的会议
- 成员需要在应用的可见范围内

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok"
}
```

**参数说明：**

|参数 |说明 |
|:--|
|errcode |int32|返回码 |
|errmsg |string|对返回码的文本描述内容 |
