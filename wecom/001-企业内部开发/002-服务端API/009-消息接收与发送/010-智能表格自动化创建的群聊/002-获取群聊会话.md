该接口可以查询通过[智能表格自动化创建的群聊](https://open.work.weixin.qq.com/help2/pc/19684?person_id=1&searchData=#4%E3%80%81%E8%87%AA%E5%8A%A8%E5%8C%96%E6%B5%81%E7%A8%8B%E5%88%9B%E5%BB%BA%E7%BE%A4%E8%81%8A)信息。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/groupchat/get?access_token=ACCESS_TOKEN

**请求包体**

```json
{
  "docid":"DOCID",
  "chat_id":"CHATID"
}
```

**参数说明：**

| 参数 | 必须 | 说明 |
| ------------ | ---- |------------------------------------------------------------------------------------------------------------------|
| access_token | 是 | 调用接口凭证|
|docid|是|智能表格ID。需为当前应用创建的智能表格|
|chat_id|是|群聊ID，需为智能表自动规则创建群聊ID|

**权限说明**

- 仅自建应用可调用，需配置到文档“[可调用应用](#43883)”列表中的应用
- 可见范围需包含根部门

**并发限制：20** 

**返回结果：**

```
{
 "errcode": 0,
 "errmsg": "ok",
 "name": "NAME",
 "owner": "userid2",
 "user_list": ["userid1", "userid2", "userid3"]
}
```
**参数说明：**

| 参数 | 说明 |
| ------- | ---------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
|name|群聊名称|
|owner|群主|
|user_list|群成员列表，仅返回当前企业成员列表|
