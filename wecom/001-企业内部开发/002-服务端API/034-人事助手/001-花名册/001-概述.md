最后更新：2023/11/30

本接口适用于使用API对员工花名册信息进行获取和更新。
如需获取和更新员工基础信息字段，请使用[通讯录管理](#10016)

**调用权限说明：**
- 目前本目录下所有接口调用的应用需要满足如下的权限

| 应用类型 | 权限要求 |
| -------- | ---- | 
|自建应用|配置到「!!#ff0000 人事助手 - 可调用接口的应用!!」中 |
|代开发应用|暂不支持| 
|第三方应用|暂不支持|

 *注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)
