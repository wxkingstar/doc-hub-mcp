该接口用于获取公共邮箱详细信息，包含公共邮箱名称、权限信息。

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/get?access_token=ACCESS_TOKEN

**请求包体**
```json
{
 "id_list": [1, 2]
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|--------------|---------|------|----------|
| id_list | uint32 | 是 | 公共邮箱ID列表 |

**权限说明**

- 自建应用需配置到“[可调用应用](#37468)”列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 应用调用接口只能访问自身创建的公共邮箱

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok",
 "list": [{
 "id": 1,
 "email": "apitest3@gzdev.com",
 "name": "测试",
 "userid_list": {
 "list": [
 "member",
 "member2"
 ]
 },
 "department_list": {
 "list": [
 1,
 2
 ]
 },
 "tag_list": {
 "list": [
 1,
 2
 ]
 },
 "alias_list": {
 "list": [
 "lisi@gzdev.com",
 "wangwu@gzdev.com"
 ]
 }
 }]
}
```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| id | uint32 | 公共邮箱ID |
| email | string | 公共邮箱地址 |
| name | string | 公共邮箱名称 |
| userid_list | string | 有权限使用公共邮箱的成员UserID列表 |
| department_list | uint32 | 有权限使用公共邮箱的部门列表 |
| tag_list | uint32 | 有权限使用公共邮箱的标签列表 |
| alias_list | string | 公共邮箱的别名列表 |
