该接口用于删除公共邮箱的客户端专用密码

**请求方式**：POST（**HTTPS**）
**请求地址**: https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/delete_auth_code?access_token=ACCESS_TOKEN

**请求包体**
```json
{
 "id" : 1,
 "auth_code_id": 1
}
```

**参数说明**

| 参数 | 类型 | 是否必须 | 说明 |
|-----------------|--------|------|-------------------------------------------------------------------------------|
| id | uint32 | 是 | 公共邮箱ID |
|auth_code_id| uint32 | 是 | 客户端专用密码ID |

**返回示例**

```json
{
 "errcode": 0,
 "errmsg": "ok"
}
```
**参数说明**

| 参数 | 类型 | 说明 |
|---------|--------|-------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
