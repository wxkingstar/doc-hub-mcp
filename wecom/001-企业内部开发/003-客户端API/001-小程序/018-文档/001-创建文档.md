此接口用于新建一个文档、表格、收集表、幻灯片或智能表格。

**调用前提：**
> 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态。
>当前成员必须在应用的可见范围之中，否则调用时会报“no permission”错误。
>应用需具有文档使用权限，否则调用时会报“no permission”错误。
>基础应用“文档”如果没有启用，新建文档、表格、PPT、智能表格时会报“doc app closed.”。
>基础应用“收集表”如果没有启用，新建收集表时会报“form app closed.”。

### wx.qy.createDoc(Object object)
#### 参数

##### Object object

| 参数名 | 类型 | 必填 | 说明 | 
| -------- | -------- | ---- | -------------------------------- | 
| docType | Uint32 | 是 | 文档类型，3: 文档，4: 表格，5: 收集表，6: 幻灯片，10: 智能表格，其他类型暂不支持 | 
| success | Function | 否 | 接口调用成功的回调函数 | 
| fail | Function | 否 | 接口调用失败的回调函数 | 
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） | 

#### 示例代码：

```js
wx.qy.createDoc({
    docType: 1,
    success: function(res) {
 // 回调
 }
});

```
