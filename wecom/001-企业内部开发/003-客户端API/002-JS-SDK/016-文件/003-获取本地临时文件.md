## 旧版jweixin调用
### 示例代码
```javascript
wx.invoke('getLocalFileData', {
 localId: '', // 文件的localID
 }, function(res) {
 // 这里是回调函数 
 if (res.err_msg == "getLocalFileData:ok"){
 var localData = res.localData; // localData是文件的base64数据
 }
});
```
