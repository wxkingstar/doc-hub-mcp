最后更新：2021/07/29

具有客户联系权限的企业成员，可通过该接口设置个人的朋友圈封面和签名。当前暂仅支持在企业微信内调用。
>接口支持“仅设置封面”，“仅设置签名”和“同时设置封面和签名”三种情况；

```javascript
wx.qy.updateMomentsSetting( {
 signature:"个性签名", // 个性签名
      imgUrl:"http://www.xxx/" // 封面url

 },function(res) {
 if (res.err_msg == "updateMomentsSetting:ok") {
 }
  }
);
```

#### 参数

##### Object object

| 参数名 | 类型 | 必填 | 说明 |
| ---------- | -------- | ---- | ---------------------------------------- |
|signature|string |否|个性签名， 可不填（imgUrl跟signature必须二填1，否则报错"missing arguments"）。若不填该字段，表示不更新个性签名。若填了该字段且为空字符串，表示清空个性签名；最多支持输入30个中文或字母，若超过报错"invalid signature size"|
|imgUrl|string |否|朋友圈封面，可不填（imgUrl跟signature必须二填1，否则报错"missing arguments"）。封面图片建议为正方形，尺寸建议为1125*1125像素（图片若非正方形，会唤起用户调整封面图片界面）；图片大小建议不超过10M；若填了该字段，该字段不能为空，若为空或者图片地址无效，报错"invalid imgUrl" |

> * 企业微信3.1.12及以上版本支持，仅支持ios跟安卓。微信客户端不支持（微信开发者工具也不支持）。
> * 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态
> * 应用需有[配置客户联系功能权限](#13473/配置可使用客户联系接口的应用)
> * 当前成员必须[配置了客户联系功能](#13473/配置可使用客户联系功能的成员)
> * 当前成员必须在客户朋友圈使用范围
> * 当前成员必须具备外部沟通管理成员使用权限
> * 若同时设置了签名跟封面url，客户端更新顺序为先更新签名，再更新封面图url（封面图若不符合要求会让用户重新调整）。
