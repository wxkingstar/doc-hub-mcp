最后更新：2024/12/02

在微信环境里，当开发者检测到用户不在家校通讯录指定班级中，可以在h5页面中通过重定向到企业微信的组件地址让用户完成学生资料的填写，地址如下：

```
`https://work.weixin.qq.com/openSchoolFollow/${strcorpid}?strpartyid=${strpartyid}&redirect_url=${redirect_url}`
```

**请求参数**

| 参数 | 必须 | 说明 |
| ------------ | ---- | ---------------------------------------------------------------------------------------------------- |
| strcorpid | 是 |学校corpid |
| strpartyid | 否 | 班级的部门id，当传入班级部门id后，用户不能再修改校区和班级|
| redirect_url | 是 | 重定向url，当用户填写完学生资料后，企业微信会重定向到该地址|
