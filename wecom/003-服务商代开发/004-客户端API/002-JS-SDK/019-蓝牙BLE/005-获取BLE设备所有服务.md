## 旧版jweixin调用
### 示例代码
```javascript
wx.getBLEDeviceServices({
 // 这里的 deviceId 需要已经通过 createBLEConnection 与对应设备建立链接 
 deviceId: deviceId,
 success: function (res) {
 console.log('device services:', res.services)
 }
})
```
