会话内容存档第三方接口为增值接口，面向服务商单独定价收费，服务商可前往「桌面端服务商助手-工具-通用配置-购买增值接口」或「服务商后台-应用管理-购买增值接口」进行购买和使用。

[[toc]]
### 1. 接口定价

按照接口能力分为办公版、服务版、企业版，各版本定价统一如下，不设账号阶梯：

| 版本 | 定价 | 说明 |
| :---: | :---: | :---: |
| **办公版** | **100**元/账号/年 | 用于企业内部工作沟通合规管理 |
| **服务版** | **150**元/账号/年 | 用于企业内外部服务沟通质量管理 |
| **企业版** | **300**元/账号/年 | 用于企业内外部服务沟通质量管理（含语音通话） |
- 服务商购买第三方接口时需指定企业，购买的接口仅可在指定的企业下使用
- 一个接口账号可用于调用一个员工的会话内容存档第三方接口，并使用展示组件展示对应的会话内容

[[toc]]
### 2. 接口试用
获得企业授权会话内容存档第三方接口权限后，自授权日起，服务商可在该企业下免费使用接口 30 天，免费到期后需购买接口使用。

[[toc]]
### 3. 会话内容存档时间
企业开通会话内容存档，并授权服务商会话内容存档第三方接口权限后，企业微信将对授权范围内成员的会话内容进行存档，服务商可使用展示组件展示会话内容。
- 在会话内容存档第三方接口付费有效期内，存储企业自授权日起最近 90 天的会话内容，服务商可使用展示组件展示最近 90 天的会话内容；过期的会话内容不存储，展示组件不展示。
- 接口付费到期后，超过 30 天未续期，则之前付费期间存储的会话内容将不再存储，展示组件不展示。
- 试用到期后，超过 30 天未付费，则之前试用期间存储的会话内容将不再存储，展示组件不展示。

[[toc]]
### 4. 接口切换过渡期政策
2024年7月1日0:00前购买了会话内容存档企业内部接口的企业，代理销售该订单的服务商可在订单当前有效期及一次续费有效期和有效人员范围内免费调用会话内容存档第三方接口，7月1日之后新购买的订单则不支持免费切换，需分别购买，请合作伙伴尽快在该日期前接入第三方接口。

::: details 说明
目前仅对接口切换过渡期政策进行说明，后续切换功能上线后将再次通知。
:::

### 5. 客户管理
服务商可前往「桌面端服务商助手-工具-通用配置-增值接口管理」或「服务商后台-应用管理-增值接口管理」查看和管理会话内容存档第三方接口的企业客户，企业客户包括：试用中、试用到期、付费中、付费到期的企业客户。

### 6. 其他
- 退款：会话内容存档第三方接口支付后十天内支持申请退款，申请通过后将原路退款，超过十天后不支持申请退款
- 发票：会话内容存档第三方接口按月开取发票，服务商可在每月的十号开始申请上一个月对应订单的发票
