最后更新：2023/12/14

[配置了客户联系功能](#13473/开始开发)的成员添加外部联系人同意进行聊天内容存档时，回调该事件。

请求示例：

```
<xml>
  <SuiteId><
- ![CDATA[xxxx]]></SuiteId>
  <AuthCorpId><
- ![CDATA[yyyy]]></AuthCorpId>
  <InfoType><
- ![CDATA[chat_archive_audit_approved_single]]></InfoType>
  <TimeStamp>1403610513</TimeStamp>
  <OpenUserID><
- ![CDATA[xxxx]]></UserID>
  <ExternalUserID><
- ![CDATA[xxxx]]></ExternalUserID>
  <ChatId><
- ![CDATA[xxxx]]></ChatId>
</xml>
```
参数说明：

 | 参数 | 说明 |
 |----|---- |
 |SuiteId |「会话存档接口授权」的应用ID |
  |AuthCorpId |授权企业的CorpID |
  |InfoType |事件类型<br>当客户在单聊中同意存档时，该字段值为chat_archive_audit_approved_single<br>当客户在群聊中同意存档时，该字段值为chat_archive_audit_approved_room |
  |TimeStamp |时间戳 |
 |OpenUserID|单聊时返回，企业服务人员的OpenUserID|
 |ExternalUserID|外部联系人的userid，注意不是企业成员的账号 |
 |ChatId|群聊时返回，所在群聊的ChatId|
