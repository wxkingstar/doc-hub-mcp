该API用于获取代开发自建应用授权链接，用于生成带参临时二维码。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/service/get_customized_auth_url?provider_access_token=PROVIDER_ACCESS_TOKEN

**请求包体：**
```
{
  "state":"test",
  "templateid_list": ["dk1","dk2"]
} 
```

**参数说明：** 
 
 | 参数 | 是否必须 | 说明 |
 |----|---- | ----- |
 | provider_access_token | 是 | provider_access_token，获取方法参见[服务商的凭证](#11791/%E6%9C%8D%E5%8A%A1%E5%95%86%E7%9A%84token) |
 | state | 是 |state值，可以填写a-zA-Z0-9，长度不可超过32个字节。当扫带参二维码授权代开发模版时，[获取企业永久授权码](#57310)接口会返回该state值 |
 |templateid_list|是|代开发自建应用模版ID列表，数量不能超过9个|

**返回结果：** 
```
{
  "errcode":0 ,
  "errmsg":"ok" ,
  "qrcode_url":"xxxxx",
  "expires_in":864000
}
 ```
**参数说明：** 

 | 参数 | 说明 |
 |----|---- |
 |errcode|错误码 |
 |errmsg|错误描述 |
 |qrcode_url|可用来生成二维码的授权url，需要开发者自行生成为二维码 |
 |expires_in|有效期（秒）。10天过期。 |
