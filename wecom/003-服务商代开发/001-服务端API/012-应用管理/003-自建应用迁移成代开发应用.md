最后更新：2024/01/16

为保障企业数据安全，企业微信禁止服务商以自建应用的方式为企业开发产品。若历史上服务商曾经为企业开发了自建应用，服务商可以调用该接口，将企业的自建应用迁移到代开发应用。服务商也可以在服务商管理端提交自建应用列表进行迁移，但在管理端迁移之后，**若要重置应用secret或者提交上线审核，则需要先调用该接口，以完成应用的归属校验**。

**请求方式：**POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/agent/claim_customized_app?access_token=ACCESS_TOKEN

**请求包体：**
```
{
  "suite_access_token":"SUITE_ACCESS_TOKEN"
} 
```

**参数说明：** 
 
 | 参数 | 是否必须 | 说明 |
 |----|---- | ----- |
 | access_token | 是 | 待迁移或尚未验证归属的自建应用的接口调用凭证，获取方法参见“[获取access_token](#15074/获取access_token)” |
 |suite_access_token|是|代开发应用模版接口调用凭证，获取方法与第三方应用凭证相同，参见“[获取第三方应用凭证](#14939/获取第三方应用凭证)”|

**返回结果：**
```
{
  "errcode":0,
  "errmsg":"ok"
}
```
**参数说明：**
| 参数 | 说明 |
 |----|---- |
 |errcode|返回码 |
 |errmsg|对返回码的文本描述说明 |
