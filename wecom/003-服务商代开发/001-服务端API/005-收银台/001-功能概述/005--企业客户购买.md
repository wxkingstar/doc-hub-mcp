## 概述
企业微信向企业客户提供了线上的第三方应用购买流程，希望为企业提供安全、便捷、有保障的应用购买体验。
企业可通过应用详情页等入口进行应用下单购买，支付后可在订单中心管理对应的订单。
本指引将介绍企业购买、企业支付相关流程
## 1. 企业购买
**1) 企业试用**
企业添加应用后，默认选择基本版本（即免费试用版本）。应用试用到期后，企业需购买付费版本方可继续使用。
**2) 购买付费版本**
企业管理员可以在 「企业管理后台 - 应用管理 - 应用详情页」或「手机端工作台 - 管理企业 - 应用管理 -应用详情页」，配置 **应用版本**、**购买人数** 和 **购买时长** 后下单支付。

- ![image (8).png](https://wework.qpic.cn/wwpic/581542_3f4uySbRSj2Aaps_1669808881/0)
**3) 付费版本变更**
添加并购买应用的付费版本后，管理员可对付费版本进行变更，包括扩容可用人数、续期使用时间、切换应用版本。

说明：扩容可用人数和切换应用版本时，系统将自动对企业当前版本按剩余可用时间等比例进行价值折算，以抵扣部分付款金额。（例如：企业当前版本价格1000元，包含365天有效时间，当前剩余有效时间100天，则最高抵扣金额为：1000元x（100天/365天））
## 2. 企业支付
1）移动端，支持微信支付（暂时仅安卓系统支持）。
2）电脑端-企业管理后台，支持微信支付+商企付（网银对公转账）。
3）企业直接在企业微信应用市场内下单购买应用，目前仅支持企业管理员支付，暂不支持企业普通成员支付。
- ![image (11).png](https://wework.qpic.cn/wwpic/983199_lr2QIjBhShu_rpg_1669809046/0)
## 3. 企业订单管理
企业管理员可前往 「企业管理后台 - 应用管理 - 查看购买订单” 」或 「移动端工作台 - 添加应用 - 右上角订单中心」对订单进行管理。对于已经支付的订单，支持订单服务：
1) 开发票：可向服务商申请开具普通发票或增值税发票，发票由服务商开具。
2) 退款：支付后10天内且未开发票的订单可以申请退款，单个应用仅支持从该应用的最后一个订单开始申请退款。
3) 投诉服务商：若对服务商不满意，可向企业微信投诉服务商

## 4. 应用到期拦截
#### **试用到期**
企业在应用市场中首次添加第三方应用时，可选试用版本无需付费。应用试用到期后，企业需购买付费版本方可继续使用。
1) 试用版本到期前。企业微信将提前7天向企业管理员发送付费购买通知。
2) 试用版本到期后，当企业访问应用时将会被拦截，购买付费版本后可继续使用。
- ![image (12).png](https://wework.qpic.cn/wwpic/939608_Xv44ipMnQxiJbF4_1669809098/0)
#### **服务到期**
企业购买付费版本后，将按照企业所购买的时长生效服务。应用服务到期后，企业需续费后方可继续使用。
1) 服务到期前，企业微信将提前7天向企业管理员发送续费通知。
2) 服务到期后，当企业访问应用时将会被拦截，企业续费后可继续使用。
